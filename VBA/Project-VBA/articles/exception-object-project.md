# Exception Object (Project)

Represents a single calendar exception. The  **Exception** object is a member of the **[Exceptions](http://msdn.microsoft.com/library/exceptions-object-project%28Office.15%29.aspx)** collection.


## Remarks

Use the  **Exceptions** collections **[Item](http://msdn.microsoft.com/library/exceptions-item-property-project%28Office.15%29.aspx)** property to return a single **Exception** object.

Use the  **Exceptions** collections **[Add](http://msdn.microsoft.com/library/exceptions-add-method-project%28Office.15%29.aspx)** method to add an **Exception** object to the **Exceptions** collection.
