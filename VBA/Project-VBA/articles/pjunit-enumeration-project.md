# PjUnit Enumeration (Project)

Contains constants that specify a unit of time.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjDay**|7|Day.|
|**pjHour**|5|Hour.|
|**pjMinute**|3|Minute.|
|**pjMonthUnit**|11|Month.|
|**pjWeek**|9|Week.|
