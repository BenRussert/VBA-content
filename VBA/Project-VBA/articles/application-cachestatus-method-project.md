# Application.CacheStatus Method (Project)

Displays the  **Cache Status** dialog box. Available in Project Professional only.


## Syntax

 _expression_. **CacheStatus**

 _expression_ A variable that represents an **Application** object.


### Return Value

 **Boolean**
