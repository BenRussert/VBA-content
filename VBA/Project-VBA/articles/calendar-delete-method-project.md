# Calendar.Delete Method (Project)

Deletes the  **Calendar** object from a **Calendars** collection.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a **Calendar** object.


## Remarks

The  **Delete** method does not apply to resource calendars.
