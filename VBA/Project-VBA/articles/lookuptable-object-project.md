# LookupTable Object (Project)



Represents a lookup table collection in Project. The  **[LookupTableEntry](http://msdn.microsoft.com/library/lookuptableentry-object-project%28Office.15%29.aspx)** object is a member of the **LookupTable** collection.
 **Using the LookupTable Object**
The following example adds three level entries to a lookup table, where the third level entry is set with three values.
