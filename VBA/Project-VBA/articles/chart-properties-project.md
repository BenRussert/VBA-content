# Chart Properties (Project)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chart-application-property-project.md)|Gets the [Application](application-object-project.md) object. Read-only **Application**.|
|[AutoScaling](chart-autoscaling-property-project.md)|**True** if Project scales a 3-D chart so that it is closer in size to the equivalent 2-D chart. Read/write **Boolean**.|
|[BackWall](chart-backwall-property-project.md)|Gets an  **Office.IMsoWalls** object that enables the user to individually format the back wall of a 3-D chart. Read-only **IMsoWalls**.|
|[BarShape](chart-barshape-property-project.md)|Gets or sets the shape used with the 3-D bar chart or 3-D column chart. Read/write  **Office.XlBarShape**.|
|[ChartArea](chart-chartarea-property-project.md)|Gets an  **Office.IMsoChartArea** object that represents the complete chart area for the chart. Read-only **IMsoChartArea**.|
|[ChartColor](chart-chartcolor-property-project.md)|Gets or sets the index of chart colors for the active chart. Read/write  **Variant**.|
|[ChartData](chart-chartdata-property-project.md)|Gets an  **Office.IMsoChartData** object that enables the user to show, hide, or unlink the Field List pane for the selected chart. Read-only **IMsoChartData**.|
|[ChartGroups](chart-chartgroups-property-project.md)|Gets an object that represents either a single chart group or a collection of chart groups, where a chart group represents one or more series of data points that are plotted with the same format. Read-only  **Object**.|
|[ChartStyle](chart-chartstyle-property-project.md)|Gets or sets the chart style for a chart. Read/write  **Variant**.|
|[ChartTitle](chart-charttitle-property-project.md)|Gets an  **Office.IMsoChartTitle** object that represents the title of the specified chart. Read-only **IMsoChartTitle**.|
|[ChartType](chart-charttype-property-project.md)|Gets or sets the chart type. Read/write  **Office.XlChartType**.|
|[Creator](chart-creator-property-project.md)|Gets a 32-bit integer that indicates the application in which the chart was created. Read-only  **Long**.|
|[DataTable](chart-datatable-property-project.md)|Gets an  **Office.IMsoDataTable** object that represents the chart data table. Read-only **IMsoDataTable**.|
|[DepthPercent](chart-depthpercent-property-project.md)|Gets or sets the depth of a 3-D chart as a percentage of the chart width (between 20 and 2000 percent). Read/write  **Long**.|
|[DisplayBlanksAs](chart-displayblanksas-property-project.md)|Gets or sets the way that blank cells are plotted on a chart. Can be one of the  **Office.XlDisplayBlanksAs** constants. Read/write **Long**.|
|[Elevation](chart-elevation-property-project.md)|Gets or sets the elevation of the 3-D chart view, in degrees. Read/write  **Long**.|
|[Floor](chart-floor-property-project.md)|Gets an  **Office.IMsoFloor** object that represents the floor of a 3-D chart. Read-only **IMsoFloor**.|
|[Format](chart-format-property-project.md)|Gets an  **Office.IMsoChartFormat** object that enables access to format information for a chart. Read-only **IMsoChartFormat**.|
|[GapDepth](chart-gapdepth-property-project.md)|Gets or sets the distance between the data series in a 3-D chart, as a percentage of the marker width. Read/write  **Long**.|
|[HasAxis](chart-hasaxis-property-project.md)|Gets or sets which axes exist on a chart. Read/write  **Variant**.|
|[HasDataTable](chart-hasdatatable-property-project.md)|**True** if the chart has a data table. Read/write **Boolean**.|
|[HasLegend](chart-haslegend-property-project.md)|**True** if the chart has a legend. Read/write **Boolean**.|
|[HasTitle](chart-hastitle-property-project.md)|**True** if the axis or chart has a visible title. Read/write **Boolean**.|
|[HeightPercent](chart-heightpercent-property-project.md)|Gets or sets the height of a 3-D chart as a percentage of the chart width. Read/write  **Long**.|
|[Legend](chart-legend-property-project.md)|Gets an  **Office.IMsoLegend** object that represents the legend for a chart. Read-only **IMsoLegend**.|
|[Parent](chart-parent-property-project.md)|Gets the parent object of the chart. Read-only  **Object**.|
|[Perspective](chart-perspective-property-project.md)|Gets or sets a value that represents the perspective for the 3-D chart view. Read/write  **Long**.|
|[PivotLayout](chart-pivotlayout-property-project.md)|Gets an object that represents the placement of fields in a PivotTable report and the placement of axes in a PivotChart report. Read-only  **Object**.|
|[PlotArea](chart-plotarea-property-project.md)|Gets an  **Office.IMsoPlotArea** object that represents the plot area of a chart. Read-only **IMsoPlotArea**.|
|[PlotBy](chart-plotby-property-project.md)|Gets or sets the way columns or rows are used as data series on the chart. Can be one of the following  **Office.XlRowCol** constants: **xlColumns** or **xlRows**. Read/write  **Long**.|
|[PlotVisibleOnly](chart-plotvisibleonly-property-project.md)|**True** if only visible cells are plotted. **False** if both visible and hidden cells are plotted. Read/write **Boolean**.|
|[RightAngleAxes](chart-rightangleaxes-property-project.md)|**True** if the chart axes are at right angles, independent of chart rotation or elevation. Applies only to 3-D line, 3-D column, and 3-D bar charts. Read/write **Boolean**.|
|[Rotation](chart-rotation-property-project.md)|Gets or sets the rotation of the 3-D chart view (the rotation of the plot area around the z-axis), in degrees. Read/write  **Variant**.|
|[Shapes](chart-shapes-property-project.md)|Gets a  **[Shapes](ff8b735a-d0a3-debf-97f1-649dd40eadaf.md)** collection that represents all the shapes on the chart sheet. Read-only **Shapes**.|
|[ShowAllFieldButtons](chart-showallfieldbuttons-property-project.md)|Gets or sets whether to display all field buttons on a PivotChart. Read/write  **Boolean**.|
|[ShowAxisFieldButtons](chart-showaxisfieldbuttons-property-project.md)|Gets or sets whether to display axis field buttons on a PivotChart. Read/write  **Boolean**.|
|[ShowDataLabelsOverMaximum](chart-showdatalabelsovermaximum-property-project.md)|Gets or sets whether to show data labels on a 2-D chart when the value is greater than the maximum value on the value axis. Read/write  **Boolean**.|
|[ShowLegendFieldButtons](chart-showlegendfieldbuttons-property-project.md)|Gets or sets whether to display legend field buttons on a PivotChart. Read/write  **Boolean**.|
|[ShowReportFilterFieldButtons](chart-showreportfilterfieldbuttons-property-project.md)|Gets or sets whether to display report filter field buttons on a PivotChart. Read/write  **Boolean**.|
|[ShowValueFieldButtons](chart-showvaluefieldbuttons-property-project.md)|Gets or sets whether to display value field buttons on a PivotChart. Read/write  **Boolean**.|
|[SideWall](chart-sidewall-property-project.md)|Gets an  **Office.IMsoWalls** object that allows the user to individually format the side wall of a 3-D chart. Read-only **IMsoWalls**.|
|[Walls](chart-walls-property-project.md)|Gets an  **Office.IMsoWalls** object that represents the walls of a 3-D chart. Read-only **IMsoWalls**.|
