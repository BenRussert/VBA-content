# Application.PageBreaksRemoveAll Method (Project)

Removes all manual page breaks in the active project.


## Syntax

 _expression_. **PageBreaksRemoveAll**

 _expression_ A variable that represents an **Application** object.


### Return Value

 **Boolean**
