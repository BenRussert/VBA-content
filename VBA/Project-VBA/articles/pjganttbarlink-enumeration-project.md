# PjGanttBarLink Enumeration (Project)

Contains constants that specify the type of Gnatt bar links.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**pjNoGanttBarLinks**|0|No Gantt bar links.|
|**pjToEnd**|2|At the end.|
|**pjToTop**|1|At the top.|
