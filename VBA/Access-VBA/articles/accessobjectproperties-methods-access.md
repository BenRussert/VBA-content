# AccessObjectProperties Methods (Access)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](accessobjectproperties-add-method-access.md)|You can use the  **Add** method to add a new property as an **AccessObjectProperty** object to the **[AccessObjectProperties](accessobjectproperties-object-access.md)** collection of an **[AccessObject](accessobject-object-access.md)** object.|
|[Remove](accessobjectproperties-remove-method-access.md)|You can use the  **Remove** method to remove an **AccessObjectProperty** object from the **[AccessObjectProperties](accessobjectproperties-object-access.md)** collection of an **[AccessObject](accessobject-object-access.md)** object.|
