# TabControl Members (Access)


A tab control contains multiple pages on which you can place other controls, such as text boxes or option buttons. When a user clicks the corresponding tab, that page becomes active.


## Events



|**Name**|**Description**|
|:-----|:-----|
|[Change](tabcontrol-change-event-access.md)|The  **Change** event occurs when when you move from one page to another page.|
|[Click](tabcontrol-click-event-access.md)|The  **Click** event occurs when the user presses and then releases a mouse button over an object.|
|[DblClick](tabcontrol-dblclick-event-access.md)|The  **DblClick** event occurs when the user presses and releases the left mouse button twice over an object within the double-click time limit of the system.|
|[KeyDown](tabcontrol-keydown-event-access.md)|The  **KeyDown** event occurs when the user presses a key while a form or control has the focus. This event also occurs if you send a keystroke to a form or control by using the SendKeys action in a macro or the **SendKeys** statement in Visual Basic.|
|[KeyPress](tabcontrol-keypress-event-access.md)|The  **KeyPress** event occurs when the user presses and releases a key or key combination that corresponds to an ANSI code while a form or control has the focus. This event also occurs if you send an ANSI keystroke to a form or control by using the SendKeys action in a macro or the **SendKeys** statement in Visual Basic.|
|[KeyUp](tabcontrol-keyup-event-access.md)|The  **KeyUp** event occurs when the user releases a key while a form or control has the focus. This event also occurs if you send a keystroke to a form or control by using the SendKeys action in a macro or the **SendKeys** statement in Visual Basic.|
|[MouseDown](tabcontrol-mousedown-event-access.md)|The  **MouseDown** event occurs when the user presses a mouse button.|
|[MouseMove](tabcontrol-mousemove-event-access.md)|The  **MouseMove** event occurs when the user moves the mouse.|
|[MouseUp](tabcontrol-mouseup-event-access.md)|The  **MouseUp** event occurs when the user releases a mouse button.|

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Move](tabcontrol-move-method-access.md)|Moves the specified object to the coordinates specified by the argument values.|
|[SizeToFit](tabcontrol-sizetofit-method-access.md)|You can use the  **SizeToFit** method to size a control so it fits the text or image that it contains.|
