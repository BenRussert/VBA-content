# ReturnVars Properties (Access)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](returnvars-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[Count](returnvars-count-property-access.md)|Gets the number of  **[ReturnVar](returnvar-object-access.md)** objects in the **[ReturnVars](returnvars-object-access.md)** collection. Read-only **Long**.|
|[Item](returnvars-item-property-access.md)|The  **Item** property returns a specific member of a collection either by position or by index. Read-only **ReturnVar**.|
|[Parent](returnvars-parent-property-access.md)|Returns the parent object for the specified object. Read-only  **Object**.|
