# SmartTag Properties (Access)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](smarttag-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[IsMissing](smarttag-ismissing-property-access.md)|Returns  **True** if the specified smart tag is not installed or is installed incorrectly. Read-only **Boolean**.|
|[Name](smarttag-name-property-access.md)|You can use the  **Name** property to determine the string expression that identifies the name of an object. Read-only **String**.|
|[Parent](smarttag-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
|[Properties](smarttag-properties-property-access.md)|Returns a  **SmartTagProperties** collection that represents the collection of properties for a particular smart tag. Read-only.|
|[SmartTagActions](smarttag-smarttagactions-property-access.md)|Returns a  **[SmartTagActions](smarttagactions-object-access.md)** collection that represents the actions available for a specific smart tag. Read-only.|
|[XML](smarttag-xml-property-access.md)|Returns a  **String** that represents the related XML code for a smart tag. Read-only.|
