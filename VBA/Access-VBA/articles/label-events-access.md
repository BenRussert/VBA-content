# Label Events (Access)
This object has the following events:

## Events



|**Name**|**Description**|
|:-----|:-----|
|[Click](label-click-event-access.md)|The  **Click** event occurs when the user presses and then releases a mouse button over an object.|
|[DblClick](label-dblclick-event-access.md)|The  **DblClick** event occurs when the user presses and releases the left mouse button twice over an object within the double-click time limit of the system.|
|[MouseDown](label-mousedown-event-access.md)|The  **MouseDown** event occurs when the user presses a mouse button.|
|[MouseMove](label-mousemove-event-access.md)|The  **MouseMove** event occurs when the user moves the mouse.|
|[MouseUp](label-mouseup-event-access.md)|The  **MouseUp** event occurs when the user releases a mouse button.|
