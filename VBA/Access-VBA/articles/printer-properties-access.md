# Printer Properties (Access)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[BottomMargin](printer-bottommargin-property-access.md)|Along with the  **TopMargin**, **RightMargin**, and **LeftMargin** properties, specifies the margins for a printed page. Read/write **Long**.|
|[ColorMode](printer-colormode-property-access.md)|Returns or sets an  **[AcPrintColor](acprintcolor-enumeration-access.md)** constant representing whether the specified printer should print output in color or monochrome. Read/write.|
|[ColumnSpacing](printer-columnspacing-property-access.md)|Returns or sets a  **Long** representing the vertical space between detail sections in twips. Read/write.|
|[Copies](printer-copies-property-access.md)|Returns or sets a  **Long** indicating the number of copies to be printed. Read/write.|
|[DataOnly](printer-dataonly-property-access.md)|**True** if Microsoft Access prints only the data from a table or query in Datasheet View and not the labels, control borders, gridlines, and display graphics. Read/write **Boolean**.|
|[DefaultSize](printer-defaultsize-property-access.md)|**True** if the size of the detail section in Design View is used for printing; otherwise, the values of the **[ItemSizeHeight](printer-itemsizeheight-property-access.md)** and **[ItemSizeWidth](printer-itemsizewidth-property-access.md)** properties are used. Read/write **Boolean**.|
|[DeviceName](printer-devicename-property-access.md)|Returns a  **String** indicating name of the specified printer device. Read-only.|
|[DriverName](printer-drivername-property-access.md)|Returns a  **String** indicating the name of the driver used by the specified printer. Read-only.|
|[Duplex](printer-duplex-property-access.md)|Returns or sets an  **[AcPrintDuplex](acprintduplex-enumeration-access.md)** constant indicating how the specified printer handles duplex printing. Read/write.|
|[ItemLayout](printer-itemlayout-property-access.md)|Returns or sets an  **[AcPrintItemLayout](acprintitemlayout-enumeration-access.md)** constant indicating whether the printer lays columns across, then down, or down, then across. Read/write.|
|[ItemsAcross](printer-itemsacross-property-access.md)|Returns or sets a  **Long** indicating the number of columns to print across a page for multiple-column reports or labels. Read/write.|
|[ItemSizeHeight](printer-itemsizeheight-property-access.md)|Returns or sets a  **Long** indicating the height of the detail section of a form or report in twips. Read/write.|
|[ItemSizeWidth](printer-itemsizewidth-property-access.md)|Returns or sets a  **Long** indicating the height of the detail section of a form or report in twips. Read/write.|
|[LeftMargin](printer-leftmargin-property-access.md)|Along with the  **TopMargin**, **RightMargin**, and **BottomMargin** properties, specifies the margins for a printed page. Read/write **Long**.|
|[Orientation](printer-orientation-property-access.md)|Gets or sets an  **[AcPrintOrientation](acprintorientation-enumeration-access.md)** constant that represents the print orientation. Read/write.|
|[PaperBin](printer-paperbin-property-access.md)|Returns or sets an  **[AcPrintPaperBin](acprintpaperbin-enumeration-access.md)** constant indicating which paper bin the specified printer should use. Read/write.|
|[PaperSize](printer-papersize-property-access.md)|Returns or sets an  **[AcPrintPaperSize](acprintpapersize-enumeration-access.md)** constant indicating the paper size to use when printing. Read/write.|
|[Port](printer-port-property-access.md)|Returns a  **String** indicating the port name of the specified printer. Read-only.|
|[PrintQuality](printer-printquality-property-access.md)|Returns or sets an  **[AcPrintObjQuality](acprintobjquality-enumeration-access.md)** constant indicating the resolution at which the specified printer should print jobs. Read/write.|
|[RightMargin](printer-rightmargin-property-access.md)|Along with the  **TopMargin**, **LeftMargin**, and **BottomMargin** properties, specifies the margins for a printed page. Read/write **Long**.|
|[RowSpacing](printer-rowspacing-property-access.md)|Returns or sets a  **Long** indicating the horizontal space between detail sections in twips. Read/write.|
|[TopMargin](printer-topmargin-property-access.md)|Along with the  **LeftMargin**, **RightMargin**, and **BottomMargin** properties, specifies the margins for a printed page. Read/write **Long**.|
