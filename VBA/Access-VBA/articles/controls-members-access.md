# Controls Members (Access)


The  **Controls** collection contains all of the controls on a form, report, or subform, within another control, or attached to another control. The **Controls** collection is a member of a **[Form](form-object-access.md)**, **[Report](report-object-access.md)**, and **[SubForm](subform-object-access.md)** objects.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](controls-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[Count](controls-count-property-access.md)|You can use the  **Count** property to determine the number of items in a specified collection. Read-only **Long**.|
|[Item](controls-item-property-access.md)|The  **Item** property returns a specific member of a collection either by position or by index. Read-only **Object**.|
|[Parent](controls-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
