# PageBreak Members (Access)


This object corresponds to a page break control. The page break control marks the start of a new screen or printed page on a form or report.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Move](pagebreak-move-method-access.md)|Moves the specified object to the coordinates specified by the argument values.|
|[SizeToFit](pagebreak-sizetofit-method-access.md)|You can use the  **SizeToFit** method to size a control so it fits the text or image that it contains.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](pagebreak-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[ControlType](pagebreak-controltype-property-access.md)|You can use the  **ControlType** property in Visual Basic to determine the type of a control on a form or report. Read/write **Byte**.|
|[EventProcPrefix](pagebreak-eventprocprefix-property-access.md)|Gets or sets the prefix portion of an event procedure name. Read/write  **String**.|
|[InSelection](pagebreak-inselection-property-access.md)|You can use the  **InSelection** property to determine or specify whether a control on a form in Design view is selected. Read/write **Boolean**.|
|[IsVisible](pagebreak-isvisible-property-access.md)|You can use the  **IsVisible** property in to determine whether a control on a report is visible. Read/write **Boolean**.|
|[Left](pagebreak-left-property-access.md)|You can use the  **Left** property to specify an object's location on a form or report. Read/write **Integer**.|
|[Name](pagebreak-name-property-access.md)|You can use the  **Name** property to specify or determine the string expression that identifies the name of an object. Read/write **String**.|
|[Parent](pagebreak-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
|[Properties](pagebreak-properties-property-access.md)|Returns a reference to a control's **[Properties](properties-object-access.md)** collection object. Read-only.|
|[Section](pagebreak-section-property-access.md)|You can identify these controls by the section of a form or report where the control appears. Read/write  **Integer**.|
|[Tag](pagebreak-tag-property-access.md)|Stores extra information about a form, report, section, or control needed by a Microsoft Access application. Read/write  **String**.|
|[Top](pagebreak-top-property-access.md)|You can use the  **Top** property to specify an object's location on a form or report. Read/write **Integer**. .|
|[Visible](pagebreak-visible-property-access.md)|Returns or sets whether the object is visible. Read/write  **Boolean**.|
