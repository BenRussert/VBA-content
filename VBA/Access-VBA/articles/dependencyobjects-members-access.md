# DependencyObjects Members (Access)


Represents the collection of  **[AccessObject](accessobject-object-access.md)** objects that a **[DependencyInfo](dependencyinfo-object-access.md)** object contains.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](dependencyobjects-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[Count](dependencyobjects-count-property-access.md)|You can use the  **Count** property to determine the number of items in a specified collection. Read-only **Long**.|
|[Item](dependencyobjects-item-property-access.md)|The  **Item** property returns a specific member of a collection either by position or by index. Read-only **AccessObject**.|
|[Parent](dependencyobjects-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
