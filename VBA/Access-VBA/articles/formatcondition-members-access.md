# FormatCondition Members (Access)


The  **FormatCondition** object represents a conditional format of a combo box or text box control and is a member of the **FormatConditions** collection.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](formatcondition-delete-method-access.md)|Deletes the specified object.|
|[Modify](formatcondition-modify-method-access.md)|You can use the  **Modify** method to change the format conditions of a **[FormatCondition](formatcondition-object-access.md)** object in the **[FormatConditions](formatconditions-object-access.md)** collection of a combo box or text box control.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[BackColor](formatcondition-backcolor-property-access.md)|Gets or sets the interior color of the specified object. Read/write  **Long**.|
|[Enabled](formatcondition-enabled-property-access.md)|You can use the  **Enabled** property to set or return the status of the conditional format in the **[FormatCondition](formatcondition-object-access.md)** object. Read/write **Boolean**.|
|[Expression1](formatcondition-expression1-property-access.md)|You can use the  **Expression1** property to return the values of a conditional format within a **[FormatCondition](formatcondition-object-access.md)** object. Read-only **String**.|
|[Expression2](formatcondition-expression2-property-access.md)|You can use the  **Expression2** property to return the values of a conditional format within a **[FormatCondition](formatcondition-object-access.md)** object. Read-only **String**.|
|[FontBold](formatcondition-fontbold-property-access.md)|You can use the  **FontBold** property to specify whether a font appears in a bold style in the following situations:|
|[FontItalic](formatcondition-fontitalic-property-access.md)|You can use the  **FontItalic** property to specify whether text is italic in the following situations:|
|[FontUnderline](formatcondition-fontunderline-property-access.md)|You can use the  **FontUnderline** property to specify whether text is underlined in the following situations:|
|[ForeColor](formatcondition-forecolor-property-access.md)|You can use the  **ForeColor** property to specify the color for text in a control. Read/write **Long**.|
|[LongestBarLimit](formatcondition-longestbarlimit-property-access.md)|Gets or sets how the longest data bar is evaluated for the specified  **[FormatCondition](formatcondition-object-access.md)**. Read/write **[AcFormatBarLimits](acformatbarlimits-enumeration-access.md)**.|
|[LongestBarValue](formatcondition-longestbarvalue-property-access.md)|Gets or sets a numeric expression that specifies the value of the longest bar of a  **[FormatCondition](formatcondition-object-access.md)**. Read/write **String**.|
|[Operator](formatcondition-operator-property-access.md)|You can use the  **Operator** property to return the operator value for the conditional format or data validation of a **[FormatCondition](formatcondition-object-access.md)** object. |[ShortestBarLimit](formatcondition-shortestbarlimit-property-access.md)|Gets or sets how the shortest data bar is evaluated for the specified  **[FormatCondition](formatcondition-object-access.md)**. Read/write **[AcFormatBarLimits](acformatbarlimits-enumeration-access.md)**.|
|[ShortestBarValue](formatcondition-shortestbarvalue-property-access.md)|Gets or sets a numeric expression that specifies the value of the shortest bar of a  **[FormatCondition](formatcondition-object-access.md)**. Read/write **String**.|
|[ShowBarOnly](formatcondition-showbaronly-property-access.md)|Gets or sets whether the data bar or the data bar and its value are displayed. Set to  **True** to display only the data bar. Read/write ** Boolean**.|
|[Type](formatcondition-type-property-access.md)|Returns the value of a  **[FormatCondition](formatcondition-object-access.md)** object type. Read-only **[AcFormatConditionType](acformatconditiontype-enumeration-access.md)**.|
