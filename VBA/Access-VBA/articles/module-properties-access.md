# Module Properties (Access)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](module-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[CountOfDeclarationLines](module-countofdeclarationlines-property-access.md)|The  **CountOfDeclarationLines** property returns a **Long** value indicating the number of lines of code in the Declarations section in a standard module or class module. Read-only **Long**.|
|[CountOfLines](module-countoflines-property-access.md)|The  **CountOfLines** property returns a **Long** value indicating the number of lines of code in a standard module or class module. Read-only **Long**.|
|[Lines](module-lines-property-access.md)|The  **Lines** property returns a string containing the contents of a specified line or lines in a standard module or a class module. Read-only **String**.|
|[Name](module-name-property-access.md)|You can use the  **Name** property to specify or determine the string expression that identifies the name of an object. Read/write **String**.|
|[Parent](module-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
|[ProcBodyLine](module-procbodyline-property-access.md)|The  **ProcBodyLine** property returns the number of the line at which the body of a specified procedure begins in a standard module or a class module. Read-only **Long**.|
|[ProcCountLines](module-proccountlines-property-access.md)|The  **ProcCountLines** property returns the number of lines in a specified procedure in a standard module or a class module. Read-only **Long**.|
|[ProcOfLine](module-procofline-property-access.md)|The  **ProcOfLine** property returns the name of the procedure that contains a specified line in a standard module or a class module. Read-only string.|
|[ProcStartLine](module-procstartline-property-access.md)|The  **ProcStartLine** property returns avalue identifying the line at which a specified procedure begins in a standard module or a class module. Read-only **Long**.|
|[Type](module-type-property-access.md)|Indicates whether a module is a standard module or a class module. Read-only  **[AcModuleType](acmoduletype-enumeration-access.md)**.|
