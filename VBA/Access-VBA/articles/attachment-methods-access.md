# Attachment Methods (Access)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Back](attachment-back-method-access.md)|Moves the focus to the previous file in the specified  **Attachment** control.|
|[Forward](attachment-forward-method-access.md)|Moves the focus to the next file in the specified  **Attachment** control.|
|[Move](attachment-move-method-access.md)|Moves the specified object to the coordinates specified by the argument values.|
|[Requery](attachment-requery-method-access.md)|The  **Requery** method updates the data underlying a specified control that's on the active form by requerying the source of data for the control.|
|[SetFocus](attachment-setfocus-method-access.md)|The  **SetFocus** method moves the focus to the specified form, the specified control on the active form, or the specified field on the active datasheet.|
|[SizeToFit](attachment-sizetofit-method-access.md)|You can use the  **SizeToFit** method to size a control so it fits the text or image that it contains.|
