# SubForm Properties (Access)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[AddColon](subform-addcolon-property-access.md)|Specifies whether a colon follows the text in labels for new controls. Read/write  **Boolean**.|
|[Application](subform-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[AutoLabel](subform-autolabel-property-access.md)|Specifies whether labels are automatically created and attached to new controls. Read/write  **Boolean**.|
|[BorderColor](subform-bordercolor-property-access.md)|You can use the  **BorderColor** property to specify the color of a control's border. Read/write **Long**.|
|[BorderShade](subform-bordershade-property-access.md)|Gets or sets the shade that is applied to the theme color in the  **BorderColor** property of the specified object. Read/write **Single**.|
|[BorderStyle](subform-borderstyle-property-access.md)|Specifies how a control's border appears.Read/write  **Byte**.|
|[BorderThemeColorIndex](subform-borderthemecolorindex-property-access.md)|Gets or sets a value that represents a color in the applied color theme associated with the  **BorderColor** property of the specified object. Read/write **Long**.|
|[BorderTint](subform-bordertint-property-access.md)|Gets or sets the tint that is applied to the theme color in the  **BorderColor** property of the specified object. Read/write **Single**.|
|[BorderWidth](subform-borderwidth-property-access.md)|You can use the  **BorderWidth** property to specify the width of a control's border. Read/write **Byte**.|
|[BottomPadding](subform-bottompadding-property-access.md)|Gets or sets the amount of space (in inches) between the subform and its bottom gridline. Read/write  **Integer**.|
|[CanGrow](subform-cangrow-property-access.md)|Gets or sets whether the specified control automatically adjusts vertically to print or preview all the data the control contains. Read/write  **Boolean**.|
|[CanShrink](subform-canshrink-property-access.md)|Gets or sets whether the specified control automatically adjusts vertically to print or preview all the data the section or control contains. Read/write  **Boolean**.|
|[Controls](subform-controls-property-access.md)|Returns the  **Controls** collection of a form, subform, report or section. Read-only **Controls**.|
|[ControlType](subform-controltype-property-access.md)|You can use the  **ControlType** property in Visual Basic to determine the type of a control on a form or report. Read/write **Byte**.|
|[DisplayWhen](subform-displaywhen-property-access.md)|You can use the  **DisplayWhen** property to specify which of a form's controls you want displayed on screen and in print. Read/write **Byte**.|
|[Enabled](subform-enabled-property-access.md)|You can use the  **Enabled** property to set or return the status of the conditional format in the **[FormatCondition](formatcondition-object-access.md)** object. Read/write **Boolean**.|
|[EventProcPrefix](subform-eventprocprefix-property-access.md)|Gets or sets the prefix portion of an event procedure name. Read/write  **String**.|
|[FilterOnEmptyMaster](subform-filteronemptymaster-property-access.md)|Gets or sets whether all records are displayed when the subform's master field is  **Null**. Read/write **Boolean**.|
|[Form](subform-form-property-access.md)|You can use the  **Form** property to refer to a form or to refer to the form associated with a subformcontrol. Read-only **Form**.|
|[GridlineColor](subform-gridlinecolor-property-access.md)|Gets or sets the color of the gridline for the specified subform. Read/write  **Long**.|
|[GridlineShade](subform-gridlineshade-property-access.md)|Gets or sets the shade applied to the theme color in the  **GridlineColor** property of the specified object. Read/write **Single**.|
|[GridlineStyleBottom](subform-gridlinestylebottom-property-access.md)|Gets or sets the bottom gridline style of the specified subform. Read/write  **Byte**.|
|[GridlineStyleLeft](subform-gridlinestyleleft-property-access.md)|Gets or sets the width of the bottom gridline for the specified subform. Read/write  **Byte**.|
|[GridlineStyleRight](subform-gridlinestyleright-property-access.md)|Gets or sets the right gridline style of the specified subform. Read/write  **Byte**.|
|[GridlineStyleTop](subform-gridlinestyletop-property-access.md)|Gets or sets the top gridline style of the specified subform. Read/write  **Byte**.|
|[GridlineThemeColorIndex](subform-gridlinethemecolorindex-property-access.md)|Gets or sets the theme color index that represents a color in the applied color theme associated with the  **GridlineColor** property of the specified object. Read/write **Long**.|
|[GridlineTint](subform-gridlinetint-property-access.md)|Gets or sets the tint applied to the theme color in the  **GridlineColor** property of the specified object. Read/write **Single**.|
|[GridlineWidthBottom](subform-gridlinewidthbottom-property-access.md)|Gets or sets the width of the bottom gridline for the specified subform. Read/write  **Byte**.|
|[GridlineWidthLeft](subform-gridlinewidthleft-property-access.md)|Gets or sets the width of the left gridline for the specified subform. Read/write  **Byte**.|
|[GridlineWidthRight](subform-gridlinewidthright-property-access.md)|Gets or sets the width of the right gridline for the specified subform. Read/write  **Byte**.|
|[GridlineWidthTop](subform-gridlinewidthtop-property-access.md)|Gets or sets the width of the top gridline for the specified subform. Read/write  **Byte**.|
|[Height](subform-height-property-access.md)|Gets or sets the height of the specified object in twips. Read/write  **Integer**.|
|[HorizontalAnchor](subform-horizontalanchor-property-access.md)|Gets or sets an  **[AcHorizontalAnchor](achorizontalanchor-enumeration-access.md)** constant that indicates how the subform is anchored horizontally within its layout. Read/write.|
|[InSelection](subform-inselection-property-access.md)|You can use the  **InSelection** property to determine or specify whether a control on a form in Design view is selected. Read/write **Boolean**.|
|[IsVisible](subform-isvisible-property-access.md)|You can use the  **IsVisible** property in to determine whether a control on a report is visible. Read/write **Boolean**.|
|[LabelAlign](subform-labelalign-property-access.md)|The property specifies the text alignment within attached labels on new controls. Read/write  **Byte**.|
|[LabelX](subform-labelx-property-access.md)|The  **LabelX** property (along with the **LabelY** property) specifies the placement of the label for a new control. Read/write **Integer**.|
|[LabelY](subform-labely-property-access.md)|The  **LabelY** property (along with the **LabelX** property) specifies the placement of the label for a new control. Read/write **Integer**.|
|[Layout](subform-layout-property-access.md)|Returns the type of layout for the specified subform. Read-only  **[AcLayoutType](aclayouttype-enumeration-access.md)**.|
|[LayoutID](subform-layoutid-property-access.md)|Returns the unique identifier for the layout that contains the specified subform. Read-only  **Long**.|
|[Left](subform-left-property-access.md)|You can use the  **Left** property to specify an object's location on a form or report. Read/write **Integer**.|
|[LeftPadding](subform-leftpadding-property-access.md)|Gets or sets the amount of space (in inches) between the subform and its left gridline. Read/write  **Integer**.|
|[LinkChildFields](subform-linkchildfields-property-access.md)|You can use the  **LinkChildFields** property (along with the **LinkMasterFields** property) together to specify how Microsoft Access links records in a form or report to records in a subform, subreport, or embedded object, such as a chart. If these properties are set, Microsoft Access automatically updates the related record in the subform when you change to a new record in a main form. Read/write **String**.|
|[LinkMasterFields](subform-linkmasterfields-property-access.md)|You can use the  **LinkMasterFields** property (along with the **LinkChildFields** property) together to specify how Microsoft Access links records in a form or report to records in a subform, subreport, or embedded object, such as a chart. If these properties are set, Microsoft Access automatically updates the related record in the subform when you change to a new record in a main form. Read/write **String**.|
|[Locked](subform-locked-property-access.md)|The  **Locked** property specifies whether you can edit data in a control in Form view. Read/write **Boolean**.|
|[Name](subform-name-property-access.md)|You can use the  **Name** property to specify or determine the string expression that identifies the name of an object. Read/write **String**.|
|[OldBorderStyle](subform-oldborderstyle-property-access.md)|You can use this property to set or returns the unedited value of the  **BorderStyle** property for a form or control. This property is useful if you need to revert to an unedited or preferred border style. Read/write **Byte**.|
|[OnEnter](subform-onenter-property-access.md)|Sets or returns the value of the  **On Enter** box in the **Properties** window of specified object. Read/write **String**. .|
|[OnExit](subform-onexit-property-access.md)|Sets or returns the value of the  **On Exit** box in the **Properties** window of specified object. Read/write **String**. .|
|[Parent](subform-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
|[Properties](subform-properties-property-access.md)|Returns a reference to a control's **[Properties](properties-object-access.md)** collection object. Read-only.|
|[Report](subform-report-property-access.md)|You can use the  **Report** property to refer to a report or to refer to the report associated with a subreport control. Read-only **Report**.|
|[RightPadding](subform-rightpadding-property-access.md)|Gets or sets the amount of space (in inches) between the subform and its right gridline. Read/write  **Integer**.|
|[Section](subform-section-property-access.md)|You can identify these controls by the section of a form or report where the control appears. Read/write  **Integer**.|
|[SourceObject](subform-sourceobject-property-access.md)|You can use the  **SourceObject** property to identify the form or report that is the source of the subform or subreport on a form or report. Read/write **String**.|
|[SpecialEffect](subform-specialeffect-property-access.md)|You can use the  **SpecialEffect** property to specify whether special formatting will apply to the specified object. Read/write **Byte**.|
|[StatusBarText](subform-statusbartext-property-access.md)|You can use the  **StatusBarText** property to specify the text that is displayed in the status bar when a control is selected. Read/write **String**.|
|[TabIndex](subform-tabindex-property-access.md)|You can use the  **TabIndex** property to specify a control's place in the tab order on a form or report. Read/write **Integer**.|
|[TabStop](subform-tabstop-property-access.md)|You can use the  **TabStop** property to specify whether you can use the TAB key to move the focus to a control. Read/write **Boolean**.|
|[Tag](subform-tag-property-access.md)|Stores extra information about a form, report, section, or control needed by a Microsoft Access application. Read/write  **String**.|
|[Top](subform-top-property-access.md)|You can use the  **Top** property to specify an object's location on a form or report. Read/write **Integer**. .|
|[TopPadding](subform-toppadding-property-access.md)|Gets or sets the amount of space (in inches) between the subform and its top gridline. Read/write  **Integer**.|
|[VerticalAnchor](subform-verticalanchor-property-access.md)|Gets or sets an [AcVerticalAnchor](acverticalanchor-enumeration-access.md) constant that indicates how the specified subform is anchored vertically within its layout. Read/write.|
|[Visible](subform-visible-property-access.md)|Returns or sets whether the object is visible. Read/write  **Boolean**.|
|[Width](subform-width-property-access.md)|Gets or sets the width of the specified object in twips. Read/write  **Integer**.|
