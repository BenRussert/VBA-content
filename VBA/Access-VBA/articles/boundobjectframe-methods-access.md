# BoundObjectFrame Methods (Access)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Move](boundobjectframe-move-method-access.md)|Moves the specified object to the coordinates specified by the argument values.|
|[Requery](boundobjectframe-requery-method-access.md)|The  **Requery** method updates the data underlying a specified control that's on the active form by requerying the source of data for the control.|
|[SetFocus](boundobjectframe-setfocus-method-access.md)|The  **SetFocus** method moves the focus to the specified form, the specified control on the active form, or the specified field on the active datasheet.|
|[SizeToFit](boundobjectframe-sizetofit-method-access.md)|You can use the  **SizeToFit** method to size a control so it fits the text or image that it contains.|
