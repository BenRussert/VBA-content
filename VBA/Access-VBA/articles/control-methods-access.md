# Control Methods (Access)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Dropdown](control-dropdown-method-access.md)|You can use the  **Dropdown** method to force the list in the specified combo box to drop down.|
|[Move](control-move-method-access.md)|Moves the specified object to the coordinates specified by the argument values.|
|[Requery](control-requery-method-access.md)|The  **Requery** method updates the data underlying a specified control that's on the active form by requerying the source of data for the control.|
|[SetFocus](control-setfocus-method-access.md)|The  **SetFocus** method moves the focus to the specified form, the specified control on the active form, or the specified field on the active datasheet.|
|[SizeToFit](control-sizetofit-method-access.md)|You can use the  **SizeToFit** method to size a control so it fits the text or image that it contains.|
|[Undo](control-undo-method-access.md)|You can use the  **Undo** method to reset a control or form when its value has been changed.|
