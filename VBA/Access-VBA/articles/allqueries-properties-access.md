# AllQueries Properties (Access)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](allqueries-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[Count](allqueries-count-property-access.md)|You can use the  **Count** property to determine the number of items in a specified collection. Read-only **Long**.|
|[Item](allqueries-item-property-access.md)|The  **Item** property returns a specific member of a collection either by position or by index. Read-only **AccessObject**,.|
|[Parent](allqueries-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
