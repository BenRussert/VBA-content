# ADO Event Instantiation by Language

 **Last modified:** December 30, 2015

**Applies to:** Access 2013 | Access 2016

Each programming language creates instances of ADO events differently. All of the following examples create a  **ConnectComplete** event handler.


- [Visual Basic](http://msdn.microsoft.com/library/visual-basic%28Office.15%29.aspx)
    
- [Visual C++](http://msdn.microsoft.com/library/visual-c%28Office.15%29.aspx)
    
- [Visual J++](http://msdn.microsoft.com/library/visual-j%28Office.15%29.aspx)
    
- [VBScript](http://msdn.microsoft.com/library/vbscript%28Office.15%29.aspx)
    
- [JScript](http://msdn.microsoft.com/library/jscript%28Office.15%29.aspx)
    
- [ADO/WFC](http://msdn.microsoft.com/library/ado-wfc%28Office.15%29.aspx)
