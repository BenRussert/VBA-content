# Form Methods (Access)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[GoToPage](form-gotopage-method-access.md)|The  **GoToPage** method moves the focus to the first control on a specified page in the active form.|
|[Move](form-move-method-access.md)|Moves the specified object to the coordinates specified by the argument values.|
|[Recalc](form-recalc-method-access.md)|The  **Recalc** method immediately updates all calculated controls on a form.|
|[Refresh](form-refresh-method-access.md)|The  **Refresh** method immediately updates the records in the underlying record source for a specified form or datasheet to reflect changes made to the data by you and other users in a multiuser environment.|
|[Repaint](form-repaint-method-access.md)|The  **Repaint** method completes any pending screen updates for a specified form. When performed on a form, the **Repaint** method also completes any pending recalculations of the form's controls.|
|[Requery](form-requery-method-access.md)|The  **Requery** method updates the data underlying a specified form by requerying the source of data for the form.|
|[SetFocus](form-setfocus-method-access.md)|The  **SetFocus** method moves the focus to the specified form, the specified control on the active form, or the specified field on the active datasheet.|
|[Undo](form-undo-method-access.md)|You can use the  **Undo** method to reset a control or form when its value has been changed.|
