# Properties Members (Access)


The  **Properties** collection contains all of the built-in properties in an instance of an open form, report, or control. These properties uniquely characterize that instance of the object.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](properties-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[Count](properties-count-property-access.md)|Returns the number of objects in the specified collection. Read-only.|
|[Item](properties-item-property-access.md)|The  **Item** property returns a specific member of a collection either by position or by index. Read-only **Object**.|
|[Parent](properties-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
