# Attachment Events (Access)
This object has the following events:

## Events



|**Name**|**Description**|
|:-----|:-----|
|[AfterUpdate](attachment-afterupdate-event-access.md)|Returns or sets which macro, event procedure, or user-defined function runs when the  **AfterUpdate** event occurs. Read/write **String**.|
|[AttachmentCurrent](attachment-attachmentcurrent-event-access.md)|Occurs when the focus moves to or from a file contained in the specified  **Attachment** control, making it the current attachment, or when the form is refreshed or requeried.|
|[BeforeUpdate](attachment-beforeupdate-event-access.md)|The  **BeforeUpdate** event occurs before changed data in a control or record is updated.|
|[Change](attachment-change-event-access.md)|The  **Change** event occurs when the contents of the specified control change.|
|[Click](attachment-click-event-access.md)|The  **Click** event occurs when the user presses and then releases a mouse button over an object.|
|[DblClick](attachment-dblclick-event-access.md)|The  **DblClick** event occurs when the user presses and releases the left mouse button twice over an object within the double-click time limit of the system.|
|[Dirty](attachment-dirty-event-access.md)|The  **Dirty** event occurs when the content of the specified control changes.|
|[Enter](attachment-enter-event-access.md)|The  **Enter** event occurs before a control actually receives the focus from a control on the same form.|
|[Exit](attachment-exit-event-access.md)|The  **Exit** event occurs just before a control loses the focus to another control on the same form.|
|[GotFocus](attachment-gotfocus-event-access.md)|The  **GotFocus** event occurs when the specified object receives the focus.|
|[KeyDown](attachment-keydown-event-access.md)|The  **KeyDown** event occurs when the user presses a key while a form or control has the focus. This event also occurs if you send a keystroke to a form or control by using the SendKeys action in a macro or the **SendKeys** statement in Visual Basic.|
|[KeyPress](attachment-keypress-event-access.md)|The  **KeyPress** event occurs when the user presses and releases a key or key combination that corresponds to an ANSI code while a form or control has the focus. This event also occurs if you send an ANSI keystroke to a form or control by using the SendKeys action in a macro or the **SendKeys** statement in Visual Basic.|
|[KeyUp](attachment-keyup-event-access.md)|The  **KeyUp** event occurs when the user releases a key while a form or control has the focus. This event also occurs if you send a keystroke to a form or control by using the SendKeys action in a macro or the **SendKeys** statement in Visual Basic.|
|[LostFocus](attachment-lostfocus-event-access.md)|The  **LostFocus** event occurs when the specified object receives the focus.|
|[MouseDown](attachment-mousedown-event-access.md)|The  **MouseDown** event occurs when the user presses a mouse button.|
|[MouseMove](attachment-mousemove-event-access.md)|The  **MouseMove** event occurs when the user moves the mouse.|
|[MouseUp](attachment-mouseup-event-access.md)|The  **MouseUp** event occurs when the user releases a mouse button.|
