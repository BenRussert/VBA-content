# AllDatabaseDiagrams Members (Access)


The  **AllDatabaseDiagrams** collection contains an **[AccessObject](accessobject-object-access.md)** for each database diagram in the **[CurrentData](currentdata-object-access.md)** or **[CodeData](codedata-object-access.md)** object.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](alldatabasediagrams-application-property-access.md)|You can use the  **Application** property to access the active Microsoft Access **[Application](application-object-access.md)** object and its related properties. Read-only **Application** object.|
|[Count](alldatabasediagrams-count-property-access.md)|You can use the  **Count** property to determine the number of items in a specified collection. Read-only **Long**.|
|[Item](alldatabasediagrams-item-property-access.md)|The  **Item** property returns a specific member of a collection either by position or by index. Read-only **AccessObject**.|
|[Parent](alldatabasediagrams-parent-property-access.md)|Returns the parent object for the specified object. Read-only.|
