# Hyperlinks Members (Visio)
Includes  **Hyperlink** objects. A **Hyperlinks** collection enables you to access and manipulate a shape's Hyperlinks section.

Includes  **Hyperlink** objects. A **Hyperlinks** collection enables you to access and manipulate a shape's Hyperlinks section.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](hyperlinks-add-method-visio.md)|Adds a new object to a collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](hyperlinks-application-property-visio.md)|Returns the instance of Microsoft Visio that is associated with an object. Read-only.|
|[Count](hyperlinks-count-property-visio.md)|Returns the number of objects in a collection. Read-only.|
|[Item](hyperlinks-item-property-visio.md)|Returns an item from a collection. The  **Item** property is the default property for all collections. Read-only.|
|[ItemU](hyperlinks-itemu-property-visio.md)|Returns an object from a collection. Read-only.|
|[ObjectType](hyperlinks-objecttype-property-visio.md)|Returns an object's type. Read-only.|
|[Shape](hyperlinks-shape-property-visio.md)|Returns the  **Shape** object that owns a **Cell** , **Characters** , **Row** , or **Section** object or that is associated with a **Hyperlink** or **OLEObject** object or with the **Hyperlinks** collection. Read-only.|
|[Stat](hyperlinks-stat-property-visio.md)|Returns status information for an object. Read-only.|
