# MenuSet Properties (Visio)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[BuiltIn](menuset-builtin-property-visio.md)|Determines whether an object is a default Microsoft Visio user interface object or a custom object. Read-only.|
|[Caption](menuset-caption-property-visio.md)|Gets or sets the caption for a menu set. Read/write.|
|[Enabled](menuset-enabled-property-visio.md)|Determines whether or not an object is currently enabled. Read/write.|
|[Height](menuset-height-property-visio.md)|Gets or sets the height of a menu set. Read/write.|
|[Left](menuset-left-property-visio.md)|Gets the distance in pixels between the left edge of the object and the left side of the docking area. Sets the distance in pixels between the left edge of an object and the left edge of the screen. Read/write.|
|[Menus](menuset-menus-property-visio.md)|Returns the  **Menus** collection of a **MenuSet** object. Read-only.|
|[Parent](menuset-parent-property-visio.md)|Determines the parent of an object. Read-only.|
|[Position](menuset-position-property-visio.md)|Gets or sets the position of an object. Read/write.|
|[Protection](menuset-protection-property-visio.md)|Determines how a  **MenuSet** object is protected from user customization. Read/write.|
|[RowIndex](menuset-rowindex-property-visio.md)|Gets or sets the docking order of a  **MenuSet** object in relation to other items in the same docking area. Read/write.|
|[SetID](menuset-setid-property-visio.md)|Returns the set ID of a  **MenuSet** object in its collection. Read-only.|
|[Top](menuset-top-property-visio.md)|Gets the distance between the top of an object and the top of the docking area or the top of the screen if the object isn't docked; it sets the distance between the top of a  **Menu** object and the top of the screen. Read/write.|
|[Visible](menuset-visible-property-visio.md)|Determines whether an object is visible. Read/write.|
|[Width](menuset-width-property-visio.md)|Gets the width of an object in pixels. Read/write.|
