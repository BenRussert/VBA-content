# Hyperlink Properties (Visio)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Address](hyperlink-address-property-visio.md)|Gets or sets the address for a shape's  **Hyperlink** object, the address to which the hyperlink navigates. Read/write.|
|[Application](hyperlink-application-property-visio.md)|Returns the instance of Microsoft Visio that is associated with an object. Read-only.|
|[Description](hyperlink-description-property-visio.md)|Gets or sets the description of a hyperlink. Read/write.|
|[ExtraInfo](hyperlink-extrainfo-property-visio.md)|Returns or sets extra URL request information used to resolve the hyperlink's URL. Read/write.|
|[Frame](hyperlink-frame-property-visio.md)|Gets or sets the name of an HTML frame in the shape's  **Hyperlink** object. Read/write.|
|[IsDefaultLink](hyperlink-isdefaultlink-property-visio.md)|Determines the default  **Hyperlink** object for a shape. Read/write.|
|[Name](hyperlink-name-property-visio.md)|Specifies the name of an object. Read-only.|
|[NameU](hyperlink-nameu-property-visio.md)|Specifies the universal name of a  **Hyperlink** object. Read/write.|
|[NewWindow](hyperlink-newwindow-property-visio.md)|Determines whether Microsoft Visio opens a window in a new location when it follows a hyperlink to open a Web page or another Visio document. Read/write.|
|[ObjectType](hyperlink-objecttype-property-visio.md)|Returns an object's type. Read-only.|
|[Row](hyperlink-row-property-visio.md)|Returns the row index of a hyperlink. Read-only.|
|[Shape](hyperlink-shape-property-visio.md)|Returns the  **Shape** object that owns a **Cell** , **Characters** , **Row** , or **Section** object or that is associated with a **Hyperlink** or **OLEObject** object or with the **Hyperlinks** collection. Read-only.|
|[Stat](hyperlink-stat-property-visio.md)|Returns status information for an object. Read-only.|
|[SubAddress](hyperlink-subaddress-property-visio.md)|Gets or sets the subaddress in a shape's  **Hyperlink** object. Read/write.|
