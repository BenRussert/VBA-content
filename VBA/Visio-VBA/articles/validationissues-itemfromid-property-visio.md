# ValidationIssues.ItemFromID Property (Visio)

Returns the  **[ValidationIssue](validationissue-object-visio.md)** object that matches the specified ID. Read-only.


## Syntax

 _expression_ . **ItemFromID**( **_IssueID_** )

 _expression_ A variable that represents a **[ValidationIssues](validationissues-object-visio.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _IssueID_|Required| **Long**|The ID of the object to retrieve.|

### Return Value

 **ValidationIssue**
