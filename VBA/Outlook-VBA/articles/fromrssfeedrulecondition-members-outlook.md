# FromRssFeedRuleCondition Members (Outlook)
Represents a rule condition that evaluates whether an item is from a specified RSS subscription.

Represents a rule condition that evaluates whether an item is from a specified RSS subscription.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](fromrssfeedrulecondition-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](fromrssfeedrulecondition-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[ConditionType](fromrssfeedrulecondition-conditiontype-property-outlook.md)|Returns a constant,  **olConditionFromRssFeed** , from the **[OlRuleConditionType](olruleconditiontype-enumeration-outlook.md)** enumeration that indicates the type of rule condition. Read-only.|
|[Enabled](fromrssfeedrulecondition-enabled-property-outlook.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
|[FromRssFeed](fromrssfeedrulecondition-fromrssfeed-property-outlook.md)|Returns or sets an array of  **String** elements that represent the RSS subscriptions that are being evaluated by the rule condition. Read/write.|
|[Parent](fromrssfeedrulecondition-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](fromrssfeedrulecondition-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
