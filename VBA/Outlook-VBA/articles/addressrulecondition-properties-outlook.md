# AddressRuleCondition Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Address](addressrulecondition-address-property-outlook.md)|Returns or sets an array of  **String** elements to evaluate the address rule condition. Read/write.|
|[Application](addressrulecondition-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](addressrulecondition-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[ConditionType](addressrulecondition-conditiontype-property-outlook.md)|Returns a constant from the  **[OlRuleConditionType](olruleconditiontype-enumeration-outlook.md)** enumeration that indicates the type of rule condition. Read-only.|
|[Enabled](addressrulecondition-enabled-property-outlook.md)|Returns or sets a  **Boolean** that determines if the rule condition is enabled. Read/write.|
|[Parent](addressrulecondition-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](addressrulecondition-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
