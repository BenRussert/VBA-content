# Recipient Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Address](recipient-address-property-outlook.md)|Returns a  **String** representing the e-mail address of the **[Recipient](recipient-object-outlook.md)** . Read-only.|
|[AddressEntry](recipient-addressentry-property-outlook.md)|Returns the  **[AddressEntry](addressentry-object-outlook.md)** object corresponding to the resolved recipient. Read/write.|
|[Application](recipient-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[AutoResponse](recipient-autoresponse-property-outlook.md)|Returns or sets a  **String** representing the text of an automatic response for a **[Recipient](recipient-object-outlook.md)** . Read/write.|
|[Class](recipient-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[DisplayType](recipient-displaytype-property-outlook.md)|Returns a constant belonging to the  **[OlDisplayType](oldisplaytype-enumeration-outlook.md)** enumeration that describes the nature of the **[Recipient](recipient-object-outlook.md)** . Read-only.|
|[EntryID](recipient-entryid-property-outlook.md)|Returns a  **String** representing the unique Entry ID of the object. Read-only.|
|[Index](recipient-index-property-outlook.md)|Returns a  **Long** indicating the position of the object within the collection. Read-only.|
|[MeetingResponseStatus](recipient-meetingresponsestatus-property-outlook.md)|Returns an  **[OlResponseStatus](olresponsestatus-enumeration-outlook.md)** constant indicating the overall status of the response to the meeting request for the recipient. Read-only.|
|[Name](recipient-name-property-outlook.md)|Returns a  **String** value that represents the display name for the object. Read-only.|
|[Parent](recipient-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[PropertyAccessor](recipient-propertyaccessor-property-outlook.md)|Returns a  **[PropertyAccessor](propertyaccessor-object-outlook.md)** object that supports creating, getting, setting, and deleting properties of the parent **[Recipient](recipient-object-outlook.md)** object. Read-only.|
|[Resolved](recipient-resolved-property-outlook.md)|Returns a  **Boolean** that indicates **True** if the recipient has been validated against the Address Book. Read-only.|
|[Sendable](recipient-sendable-property-outlook.md)|Returns or sets a  **Boolean** value that indicates whether a meeting request can be sent to the **[Recipient](recipient-object-outlook.md)** . Read/write|
|[Session](recipient-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
|[TrackingStatus](recipient-trackingstatus-property-outlook.md)|Returns or sets an  **[OlTrackingStatus](oltrackingstatus-enumeration-outlook.md)** constant indicating the tracking status for the recipient. Read/write.|
|[TrackingStatusTime](recipient-trackingstatustime-property-outlook.md)|Returns or sets a  **Date** indicating the tracking status date and time for the recipient. Read/write.|
|[Type](recipient-type-property-outlook.md)|Returns or sets a  **Long** representing the type of recipient. Read/write.|
