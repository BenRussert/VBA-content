# NavigationModule Members (Outlook)
Represents a navigation module in the Navigation Pane.

Represents a navigation module in the Navigation Pane.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](navigationmodule-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](navigationmodule-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[Name](navigationmodule-name-property-outlook.md)|Returns a  **String** value that represents the localized display name for the **[NavigationModule](navigationmodule-object-outlook.md)** object. Read-only.|
|[NavigationModuleType](navigationmodule-navigationmoduletype-property-outlook.md)|Returns an  **[olNavigationModuleType](olnavigationmoduletype-enumeration-outlook.md)** constant that represents the navigation module type for the object. Read-only.|
|[Parent](navigationmodule-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Position](navigationmodule-position-property-outlook.md)|Returns or sets a  **Long** value that represents the ordinal position of the **[NavigationModule](navigationmodule-object-outlook.md)** object when displayed in the Navigation Pane. Read/write.|
|[Session](navigationmodule-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
|[Visible](navigationmodule-visible-property-outlook.md)|Returns or sets a  **Boolean** value that determines whether the **[NavigationModule](navigationmodule-object-outlook.md)** object is displayed in the Navigation Pane. Read/write.|
