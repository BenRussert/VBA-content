# View Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](view-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](view-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[Filter](view-filter-property-outlook.md)|Returns or sets a  **String** value that represents the filter for a view. Read/write.|
|[Language](view-language-property-outlook.md)|Returns or sets a  **String** value that represents the language setting for the object that defines the language used in the menu. Read/write.|
|[LockUserChanges](view-lockuserchanges-property-outlook.md)|Returns or sets a  **Boolean** value that indicates whether a user can modify the settings of the current view. Read/write.|
|[Name](view-name-property-outlook.md)|Returns or sets a  **String** value that represents the display name for the object. Read/write.|
|[Parent](view-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[SaveOption](view-saveoption-property-outlook.md)|Returns an  **[OlViewSaveOption](olviewsaveoption-enumeration-outlook.md)** constant that specifies the folders in which the specified view is available and the read permissions attached to the view. Read-only.|
|[Session](view-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
|[Standard](view-standard-property-outlook.md)|Returns a  **Boolean** that specifies whether or not the view is a built-in Outlook view. Read-only|
|[ViewType](view-viewtype-property-outlook.md)|Returns an  **[OlViewType](olviewtype-enumeration-outlook.md)** constant representing the view type of a **[View](view-object-outlook.md)** object. Read-only.|
|[XML](view-xml-property-outlook.md)|Returns or sets a  **String** value that specifies the XML definition of the current view. Read/write.|
