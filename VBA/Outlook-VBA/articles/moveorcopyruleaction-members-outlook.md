# MoveOrCopyRuleAction Members (Outlook)
Represents an action that moves or copies a message.

Represents an action that moves or copies a message.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[ActionType](moveorcopyruleaction-actiontype-property-outlook.md)|Returns a constant,  **olRuleActionMoveToFolder** , from the **[OlRuleActionType](olruleactiontype-enumeration-outlook.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.|
|[Application](moveorcopyruleaction-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](moveorcopyruleaction-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[Enabled](moveorcopyruleaction-enabled-property-outlook.md)|Returns or sets a  **Boolean** that determines if the rule action is enabled. Read/write.|
|[Folder](moveorcopyruleaction-folder-property-outlook.md)|Returns or sets a  **[Folder](folder-object-outlook.md)** object that represents the folder to which the rule moves or copies the message. Read/write.|
|[Parent](moveorcopyruleaction-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](moveorcopyruleaction-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
