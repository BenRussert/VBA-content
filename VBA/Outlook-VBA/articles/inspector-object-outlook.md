# Inspector Object (Outlook)

Represents the window in which an Outlook item is displayed.


## Remarks




- Use the  **[ActiveInspector](http://msdn.microsoft.com/library/application-activeinspector-method-outlook%28Office.15%29.aspx)** method to return the object representing the currently active inspector (if there is one).
    
- Use the  **[GetInspector](http://msdn.microsoft.com/library/mailitem-getinspector-property-outlook%28Office.15%29.aspx)** property to return the **Inspector** object associated with an item.
    
- Use the  **[Display](http://msdn.microsoft.com/library/mailitem-display-method-outlook%28Office.15%29.aspx)** method to display an item in its associated inspector.
