# RuleAction Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[ActionType](ruleaction-actiontype-property-outlook.md)|Returns a constant from the  **[OlRuleActionType](olruleactiontype-enumeration-outlook.md)** enumeration that indicates the type of action that is taken by the **[RuleAction](ruleaction-object-outlook.md)** . Read-only.|
|[Application](ruleaction-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](ruleaction-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[Enabled](ruleaction-enabled-property-outlook.md)|Returns or sets a  **Boolean** that determines if the **[RuleAction](ruleaction-object-outlook.md)** is enabled. Read/write.|
|[Parent](ruleaction-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](ruleaction-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
