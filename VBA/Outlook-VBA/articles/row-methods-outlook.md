# Row Methods (Outlook)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[BinaryToString](row-binarytostring-method-outlook.md)|Obtains a  **String** representing a value that has been converted from a binary value for the parent **[Row](row-object-outlook.md)** at the column specified by _Index_ .|
|[GetValues](row-getvalues-method-outlook.md)|Obtains a one-dimensional array containing the values for all columns at the  **[Row](row-object-outlook.md)** in the parent **[Table](table-object-outlook.md)** .|
|[Item](row-item-method-outlook.md)|Obtains an  **Object** that represents the value for the **[Row](row-object-outlook.md)** object at the column specified by _Index_ .|
|[LocalTimeToUTC](row-localtimetoutc-method-outlook.md)|Obtains a  **Date** value in a **[Table](table-object-outlook.md)** specified by the **[Row](row-object-outlook.md)** object at _Index_ , that has been converted from local time to Coordinated Universal Time (UTC).|
|[UTCToLocalTime](row-utctolocaltime-method-outlook.md)|Obtains a  **Date** value in a **[Table](table-object-outlook.md)** specified by the **[Row](row-object-outlook.md)** object at _Index_ , that has been converted from Coordinated Universal Time (UTC) to local time.|
