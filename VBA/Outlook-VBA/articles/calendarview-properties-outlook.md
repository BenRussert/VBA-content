# CalendarView Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](calendarview-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[AutoFormatRules](calendarview-autoformatrules-property-outlook.md)|Returns an  **[AutoFormatRules](autoformatrules-object-outlook.md)** object that represents the set of formatting rules applicable to the **[CalendarView](calendarview-object-outlook.md)** object. Read-only.|
|[BoldDatesWithItems](calendarview-bolddateswithitems-property-outlook.md)|Returns or sets a  **Boolean** value that indicates whether days that contain Outlook items should be displayed in bold within the Date Navigator for the **[CalendarView](calendarview-object-outlook.md)** object. Read/write.|
|[BoldSubjects](calendarview-boldsubjects-property-outlook.md)|Returns or sets a  **Boolean** value that indicates whether the subject information for Outlook items is displayed in bold within a **[CalendarView](calendarview-object-outlook.md)** object. Read/write|
|[CalendarViewMode](calendarview-calendarviewmode-property-outlook.md)|Returns or sets an  **[OlCalendarViewMode](olcalendarviewmode-enumeration-outlook.md)** that determines the current view mode of the **[CalendarView](calendarview-object-outlook.md)** object. Read/write.|
|[Class](calendarview-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[DaysInMultiDayMode](calendarview-daysinmultidaymode-property-outlook.md)|Returns or sets a  **Long** value that represents the number of consecutive days displayed in the **[CalendarView](calendarview-object-outlook.md)** object. Read/write|
|[DayWeekTimeScale](calendarview-dayweektimescale-property-outlook.md)|Returns or sets an  **[OlDayWeekTimeScale](oldayweektimescale-enumeration-outlook.md)** constant that represents the scale used to represent time periods in a **[CalendarView](calendarview-object-outlook.md)** object. Read/write.|
|[DisplayedDates](calendarview-displayeddates-property-outlook.md)|Returns a  **Variant** array containing strings that represent the days displayed in a **[CalendarView](calendarview-object-outlook.md)** object. Read-only.|
|[EndField](calendarview-endfield-property-outlook.md)|Returns or sets a  **String** value that represents the name of the property that ends the time duration for Outlook items displayed in the **[CalendarView](calendarview-object-outlook.md)** object. Read/write.|
|[Filter](calendarview-filter-property-outlook.md)|Returns or sets a  **String** value that represents the filter for a view. Read/write.|
|[Language](calendarview-language-property-outlook.md)|Returns or sets a  **String** value that represents the language setting for the view. Read/write.|
|[LockUserChanges](calendarview-lockuserchanges-property-outlook.md)|Returns or sets a  **Boolean** value that indicates whether a user can modify the settings of the view. Read/write.|
|[MonthShowEndTime](calendarview-monthshowendtime-property-outlook.md)|Returns or sets a  **Boolean** value that indicates whether end times for Outlook items are displayed in the **[CalendarView](calendarview-object-outlook.md)** object. Read/write.|
|[Name](calendarview-name-property-outlook.md)|Returns or sets a  **String** value that represents the display name for the object. Read/write.|
|[Parent](calendarview-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[SaveOption](calendarview-saveoption-property-outlook.md)|Returns an  **[OlViewSaveOption](olviewsaveoption-enumeration-outlook.md)** constant that specifies the folders in which the specified view is available and the read permissions attached to the view. Read-only.|
|[SelectedEndTime](calendarview-selectedendtime-property-outlook.md)|Returns a  **Date** that represents the end time of a selection in the **[CalendarView](calendarview-object-outlook.md)** object. Read-only.|
|[SelectedStartTime](calendarview-selectedstarttime-property-outlook.md)|Returns a  **Date** that represents the start time of a selection in the **[CalendarView](calendarview-object-outlook.md)** object. Read-only.|
|[Session](calendarview-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
|[Standard](calendarview-standard-property-outlook.md)|Returns a  **Boolean** value that indicates whether the **[CalendarView](calendarview-object-outlook.md)** object is a built-in Outlook view. Read-only.|
|[StartField](calendarview-startfield-property-outlook.md)|Returns or sets a  **String** value that represents the name of the property that starts the time duration for Outlook items displayed in the **[CalendarView](calendarview-object-outlook.md)** object. Read/write.|
|[ViewType](calendarview-viewtype-property-outlook.md)|Returns an  **[OlViewType](olviewtype-enumeration-outlook.md)** constant that indicates the view type of the view. Read-only.|
|[XML](calendarview-xml-property-outlook.md)|Returns or sets a  **String** value that specifies the XML definition of the view. Read/write.|
