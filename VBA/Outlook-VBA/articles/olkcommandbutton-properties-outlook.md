# OlkCommandButton Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Accelerator](olkcommandbutton-accelerator-property-outlook.md)|Returns or sets a  **String** value that represents the accelerator or hot key for the control. Read/write.|
|[AutoSize](olkcommandbutton-autosize-property-outlook.md)|Returns or sets a  **Boolean** that automatically sizes the control to display the entire contents. Read/write.|
|[Caption](olkcommandbutton-caption-property-outlook.md)|Returns or sets a  **String** that appears on an object to identify or describe it. Read/write.|
|[DisplayDropArrow](olkcommandbutton-displaydroparrow-property-outlook.md)|Returns or sets a  **Boolean** that enables the rendering of a downward-pointing arrow on the button. Read/write.|
|[Enabled](olkcommandbutton-enabled-property-outlook.md)|Returns or sets a  **Boolean** that indicates if the control is allowed to function. Read/write.|
|[Font](olkcommandbutton-font-property-outlook.md)|Returns a  **StdFont** that represents the font used to render the text inside the control. Read-only.|
|[MouseIcon](olkcommandbutton-mouseicon-property-outlook.md)|Returns or sets a  **StdPicture** that represents a custom picture to the mouse cursor for this control. Read/write.|
|[MousePointer](olkcommandbutton-mousepointer-property-outlook.md)|Returns or sets an  **[OlMousePointer](olmousepointer-enumeration-outlook.md)** constant that specifies the type of pointer displayed when the user positions the mouse over the control. Read/write.|
|[Picture](olkcommandbutton-picture-property-outlook.md)|Returns or sets a  **StdPicture** value that represents the picture that is displayed on the control. Read/write.|
|[PictureAlignment](olkcommandbutton-picturealignment-property-outlook.md)|Returns or sets an  **[OlPictureAlignment](olpicturealignment-enumeration-outlook.md)** constant that specifies the alignment of the image and the text on the button. Read/write.|
|[TextAlign](olkcommandbutton-textalign-property-outlook.md)|Returns or sets an  **[OlTextAlign](oltextalign-enumeration-outlook.md)** constant that specifies how text is aligned in the control. Read/write.|
|[WordWrap](olkcommandbutton-wordwrap-property-outlook.md)|Returns or sets a  **Boolean** that specifies whether the contents of a control automatically wrap at the end of a line. Read/write.|
