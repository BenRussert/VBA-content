# OlkLabel Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Accelerator](olklabel-accelerator-property-outlook.md)|Returns or sets a  **String** value that represents the accelerator or hot key for the control. Read/write.|
|[AutoSize](olklabel-autosize-property-outlook.md)|Returns or sets a  **Boolean** that automatically sizes the control to display the entire contents. Read/write.|
|[BackColor](olklabel-backcolor-property-outlook.md)|Returns or sets a  **Long** that indicates the background color of the control. Read/write.|
|[BackStyle](olklabel-backstyle-property-outlook.md)|Returns or sets an  **[olBackStyle](olbackstyle-enumeration-outlook.md)** constant that changes the background style of the control between transparent and opaque. Read/write.|
|[BorderStyle](olklabel-borderstyle-property-outlook.md)|Returns or sets an  **[OlBorderStyle](olborderstyle-enumeration-outlook.md)** constant that defines the style of the border around the control. Read/write.|
|[Caption](olklabel-caption-property-outlook.md)|Returns or sets a  **String** that appears on an object to identify or describe it. Read/write.|
|[Enabled](olklabel-enabled-property-outlook.md)|Returns or sets a  **Boolean** that indicates if the control is allowed to function. Read/write.|
|[Font](olklabel-font-property-outlook.md)|Returns a  **StdFont** that represents the font used to render the text inside the control. Read-only.|
|[ForeColor](olklabel-forecolor-property-outlook.md)|Returns or sets a  **Long** that indicates the foreground color of the control. Read/write.|
|[MouseIcon](olklabel-mouseicon-property-outlook.md)|Returns or sets a  **StdPicture** that represents a custom picture to the mouse cursor for this control. Read/write.|
|[MousePointer](olklabel-mousepointer-property-outlook.md)|Returns or sets an  **[OlMousePointer](olmousepointer-enumeration-outlook.md)** constant that specifies the type of pointer displayed when the user positions the mouse over the control. Read/write.|
|[TextAlign](olklabel-textalign-property-outlook.md)|Returns or sets an  **[OlTextAlign](oltextalign-enumeration-outlook.md)** constant that specifies how text is aligned in the control. Read/write.|
|[UseHeaderColor](olklabel-useheadercolor-property-outlook.md)|Returns or sets a  **Boolean** that indicates whether the label control should use the foreground color to match the current Windows XP or Windows Vista theme. Read/write.|
|[WordWrap](olklabel-wordwrap-property-outlook.md)|Returns or sets a  **Boolean** that specifies whether the contents of a control automatically wrap at the end of a line. Read/write.|
