# MarkAsTaskRuleAction Members (Outlook)
Represents an action that marks a message as a task.

Represents an action that marks a message as a task.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[ActionType](markastaskruleaction-actiontype-property-outlook.md)|Returns a constant,  **olRuleActionMarkAsTask** , from the **[OlRuleActionType](olruleactiontype-enumeration-outlook.md)** enumeration that indicates the type of action that is taken by the rule. Read-only.|
|[Application](markastaskruleaction-application-property-outlook.md)|Returns an  **[Application](application-object-outlook.md)** object that represents the parent Outlook application for the object. Read-only.|
|[Class](markastaskruleaction-class-property-outlook.md)|Returns an  **[OlObjectClass](olobjectclass-enumeration-outlook.md)** constant indicating the object's class. Read-only.|
|[Enabled](markastaskruleaction-enabled-property-outlook.md)|Returns or sets a  **Boolean** that determines if the rule action is enabled. Read/write.|
|[FlagTo](markastaskruleaction-flagto-property-outlook.md)|Returns or sets a  **String** that represents the label of the flag for the message. Read/write|
|[MarkInterval](markastaskruleaction-markinterval-property-outlook.md)|Returns or sets a constant in the  **[OlMarkInterval](olmarkinterval-enumeration-outlook.md)** enumeration that represents the interval before the task is due. Read/write.|
|[Parent](markastaskruleaction-parent-property-outlook.md)|Returns the parent  **Object** of the specified object. Read-only.|
|[Session](markastaskruleaction-session-property-outlook.md)|Returns the  **[NameSpace](namespace-object-outlook.md)** object for the current session. Read-only.|
