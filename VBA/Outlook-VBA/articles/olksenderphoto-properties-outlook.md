# OlkSenderPhoto Properties (Outlook)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Enabled](olksenderphoto-enabled-property-outlook.md)|Returns or sets a  **Boolean** that indicates if the control is allowed to function. Read/write.|
|[MouseIcon](olksenderphoto-mouseicon-property-outlook.md)|Returns or sets a  **StdPicture** that represents a custom picture to the mouse cursor for this control. Read/write.|
|[MousePointer](olksenderphoto-mousepointer-property-outlook.md)|Returns or sets an  **[OlMousePointer](olmousepointer-enumeration-outlook.md)** constant that specifies the type of pointer displayed when the user positions the mouse over the control. Read/write.|
|[PreferredHeight](olksenderphoto-preferredheight-property-outlook.md)|Returns a  **Long** that specifies the application-preferred height of the sender picture. Read-only.|
|[PreferredWidth](olksenderphoto-preferredwidth-property-outlook.md)|Returns a  **Long** that specifies the application-preferred width of the sender picture. Read-only.|
