# Adjustments Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](adjustments-application-property-powerpoint.md)|Returns an  **[Application](application-object-powerpoint.md)** object that represents the creator of the specified object.|
|[Count](adjustments-count-property-powerpoint.md)|Returns the number of objects in the specified collection. Read-only.|
|[Creator](adjustments-creator-property-powerpoint.md)|Returns a  **Long** that represents the four-character creator code for the application in which the specified object was created. For example, if the object was created in Microsoft PowerPoint, this property returns the hexadecimal number 50575054. Read-only.|
|[Item](adjustments-item-property-powerpoint.md)|Returns or sets the adjustment value specified by the Index argument. Read/write.|
|[Parent](adjustments-parent-property-powerpoint.md)|Returns the parent object for the specified object.|
