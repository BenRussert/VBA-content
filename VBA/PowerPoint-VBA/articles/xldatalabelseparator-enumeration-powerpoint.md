# XlDataLabelSeparator Enumeration (PowerPoint)

Specifies the separator used with data labels.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**xlDataLabelSeparatorDefault**|1|Word selects the separator.|
