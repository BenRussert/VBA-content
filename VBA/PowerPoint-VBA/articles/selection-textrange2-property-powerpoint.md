# Selection.TextRange2 Property (PowerPoint)

Returns the  **TextRange2** object of the current **Selection** object. Read-only.


## Syntax

 _expression_. **TextRange2**

 _expression_ A variable that represents a **Selection** object.


### Return Value

 **TextRange2**
