# TickLabels Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Alignment](ticklabels-alignment-property-powerpoint.md)|Returns or sets the alignment for the specified phonetic text or tick label. Read/write  **Long**.|
|[Application](ticklabels-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Creator](ticklabels-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Depth](ticklabels-depth-property-powerpoint.md)|Returns the number of levels of category tick labels. Read-only  **Long**.|
|[Font](ticklabels-font-property-powerpoint.md)|Returns the font of the specified object. Read-only  **[ChartFont](chartfont-object-powerpoint.md)**.|
|[Format](ticklabels-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[MultiLevel](ticklabels-multilevel-property-powerpoint.md)|Returns or sets a value that indicates whether an axis is multilevel. Read/write  **Boolean**.|
|[Name](ticklabels-name-property-powerpoint.md)|Returns the name of the object. Read-only  **String**.|
|[NumberFormat](ticklabels-numberformat-property-powerpoint.md)|Returns or sets the format code for the object. Read/write  **String**.|
|[NumberFormatLinked](ticklabels-numberformatlinked-property-powerpoint.md)|**True** if the number format is linked to the cells (so that the number format changes in the labels when it changes in the cells). Read/write **Boolean**.|
|[NumberFormatLocal](ticklabels-numberformatlocal-property-powerpoint.md)|Returns or sets the format code for the object as a string in the language of the user. Read/write  **Variant**.|
|[Offset](ticklabels-offset-property-powerpoint.md)|Returns or sets the distance between the levels of labels, and the distance between the first level and the axis line. Read/write  **Long**.|
|[Orientation](ticklabels-orientation-property-powerpoint.md)|Returns or sets the text orientation. Read/write  **Long**.|
|[Parent](ticklabels-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[ReadingOrder](ticklabels-readingorder-property-powerpoint.md)|Returns or sets an  **[XlReadingOrder](xlreadingorder-enumeration-powerpoint.md)** constant that represents the reading order for the specified object. Read/write **Long**.|
