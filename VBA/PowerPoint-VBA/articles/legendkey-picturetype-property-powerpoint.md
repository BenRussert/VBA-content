# LegendKey.PictureType Property (PowerPoint)

Returns or sets the way pictures are displayed on a legend key. Read/write  **[XlChartPictureType](xlchartpicturetype-enumeration-powerpoint.md)**.


## Syntax

 _expression_. **PictureType**

 _expression_ A variable that represents a **[LegendKey](legendkey-object-powerpoint.md)** object.
