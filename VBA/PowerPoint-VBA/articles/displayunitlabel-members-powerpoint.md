# DisplayUnitLabel Members (PowerPoint)
Represents a unit label on an axis in the specified chart.

Represents a unit label on an axis in the specified chart.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](displayunitlabel-delete-method-powerpoint.md)|Deletes the object.|
|[Select](displayunitlabel-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](displayunitlabel-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Caption](displayunitlabel-caption-property-powerpoint.md)|Returns or sets the display unit label text. Read/write  **String**.|
|[Characters](displayunitlabel-characters-property-powerpoint.md)|Returns a  **[ChartCharacters](chartcharacters-object-powerpoint.md)** object that represents a range of characters within the object text. You can use the **ChartCharacters** object to format characters within a text string.|
|[Creator](displayunitlabel-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](displayunitlabel-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Formula](displayunitlabel-formula-property-powerpoint.md)|Returns or sets the object's formula in A1-style notation. Read/write.|
|[FormulaLocal](displayunitlabel-formulalocal-property-powerpoint.md)|Returns or sets the formula for the object, using A1-style references in the language of the user. Read/write  **String**.|
|[FormulaR1C1](displayunitlabel-formular1c1-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the macro. Read/write  **String**.|
|[FormulaR1C1Local](displayunitlabel-formular1c1local-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the user. Read/write  **String**.|
|[Height](displayunitlabel-height-property-powerpoint.md)|Returns the height, in points, of the object. Read-only.|
|[HorizontalAlignment](displayunitlabel-horizontalalignment-property-powerpoint.md)|Returns or sets the horizontal alignment for the specified object. Read/write  **Variant**.|
|[Left](displayunitlabel-left-property-powerpoint.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double**.|
|[Name](displayunitlabel-name-property-powerpoint.md)|Returns the name of the object. Read-only  **String**.|
|[Orientation](displayunitlabel-orientation-property-powerpoint.md)|Returns or sets the text orientation. Read/write  **Long**.|
|[Parent](displayunitlabel-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Position](displayunitlabel-position-property-powerpoint.md)|Returns or sets the position of the unit label on an axis in the chart. Read/write  **[XlChartElementPosition](xlchartelementposition-enumeration-powerpoint.md)**.|
|[ReadingOrder](displayunitlabel-readingorder-property-powerpoint.md)|Returns or sets an  **[XlReadingOrder](xlreadingorder-enumeration-powerpoint.md)** constant that represents the reading order for the specified object. Read/write **Long**.|
|[Shadow](displayunitlabel-shadow-property-powerpoint.md)|Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.|
|[Text](displayunitlabel-text-property-powerpoint.md)|Returns or sets the text for the specified object. Read/write  **String**.|
|[Top](displayunitlabel-top-property-powerpoint.md)|Returns or sets the distance, in points, from the top edge of the object to the top of the first row (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double**.|
|[VerticalAlignment](displayunitlabel-verticalalignment-property-powerpoint.md)|Returns or sets the vertical alignment of the specified object. Read/write  **Variant**.|
|[Width](displayunitlabel-width-property-powerpoint.md)|Returns the width, in points, of the object. Read-only.|
