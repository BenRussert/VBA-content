# DataLabel Members (PowerPoint)
Represents the data label on a chart point or trendline.

Represents the data label on a chart point or trendline.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](datalabel-delete-method-powerpoint.md)|Deletes the object.|
|[Select](datalabel-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](datalabel-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[AutoText](datalabel-autotext-property-powerpoint.md)|**True** if the object automatically generates appropriate text based on context. Read/write **Boolean**.|
|[Caption](datalabel-caption-property-powerpoint.md)|Returns or sets the data label text. Read/write  **String**.|
|[Characters](datalabel-characters-property-powerpoint.md)|Returns a  **[ChartCharacters](chartcharacters-object-powerpoint.md)** object that represents a range of characters within the object text. You can use the **ChartCharacters** object to format characters within a text string.|
|[Creator](datalabel-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](datalabel-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Formula](datalabel-formula-property-powerpoint.md)|Returns or sets the object's formula in A1-style notation. Read/write.|
|[FormulaLocal](datalabel-formulalocal-property-powerpoint.md)|Returns or sets the formula for the object, using A1-style references in the language of the user. Read/write  **String**.|
|[FormulaR1C1](datalabel-formular1c1-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the macro. Read/write  **String**.|
|[FormulaR1C1Local](datalabel-formular1c1local-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the user. Read/write  **String**.|
|[Height](datalabel-height-property-powerpoint.md)|Returns the height, in points, of the object. Read-only.|
|[HorizontalAlignment](datalabel-horizontalalignment-property-powerpoint.md)|Returns or sets the horizontal alignment for the specified object. Read/write  **Variant**.|
|[Left](datalabel-left-property-powerpoint.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double**.|
|[Name](datalabel-name-property-powerpoint.md)|Returns the name of the object. Read-only  **String**.|
|[NumberFormat](datalabel-numberformat-property-powerpoint.md)|Returns or sets the format code for the object. Read/write  **String**.|
|[NumberFormatLinked](datalabel-numberformatlinked-property-powerpoint.md)|**True** if the number format is linked to the cells (so that the number format changes in the labels when it changes in the cells). Read/write **Boolean**.|
|[NumberFormatLocal](datalabel-numberformatlocal-property-powerpoint.md)|Returns or sets the format code for the object as a string in the language of the user. Read/write  **Variant**.|
|[Orientation](datalabel-orientation-property-powerpoint.md)|Returns or sets the text orientation. Read/write  **Long**.|
|[Parent](datalabel-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Position](datalabel-position-property-powerpoint.md)|Returns or sets the position of the data label. Read/write  **[XlDataLabelPosition](xldatalabelposition-enumeration-powerpoint.md)**.|
|[ReadingOrder](datalabel-readingorder-property-powerpoint.md)|Returns or sets an  **[XlReadingOrder](xlreadingorder-enumeration-powerpoint.md)** constant that represents the reading order for the specified object. Read/write **Long**.|
|[Separator](datalabel-separator-property-powerpoint.md)|Returns or sets the separator used for the data labels on a chart. Read/write  **Variant**.|
|[Shadow](datalabel-shadow-property-powerpoint.md)|Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.|
|[ShowBubbleSize](datalabel-showbubblesize-property-powerpoint.md)|**True** to show the bubble size for the data labels on a chart. **False** to hide the bubble size. Read/write **Boolean**.|
|[ShowCategoryName](datalabel-showcategoryname-property-powerpoint.md)|**True** to display the category name for the data labels on a chart. **False** to hide the category name. Read/write **Boolean**.|
|[ShowLegendKey](datalabel-showlegendkey-property-powerpoint.md)|**True** if the data label legend key is visible. Read/write **Boolean**.|
|[ShowPercentage](datalabel-showpercentage-property-powerpoint.md)|**True** to display the percentage value for the data labels on a chart. **False** to hide the value. Read/write **Boolean**.|
|[ShowRange](datalabel-showrange-property-powerpoint.md)|Set to  **True** to display the **Value From Cells** range field for the specified chart data label. Set to **False** to hide that field. Read/write **Boolean**.|
|[ShowSeriesName](datalabel-showseriesname-property-powerpoint.md)|**True** to show the series name for the data labels on a chart. **False** to hide the series name. Read/write **Boolean**.|
|[ShowValue](datalabel-showvalue-property-powerpoint.md)|**True** to display a specified chart's data label values. **False** to hide the values. Read/write **Boolean**.|
|[Text](datalabel-text-property-powerpoint.md)|Returns or sets the text for the specified object. Read/write  **String**.|
|[Top](datalabel-top-property-powerpoint.md)|Returns or sets the distance, in points, from the top edge of the object to the top of the first row (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double**.|
|[VerticalAlignment](datalabel-verticalalignment-property-powerpoint.md)|Returns or sets the vertical alignment of the specified object. Read/write  **Variant**.|
|[Width](datalabel-width-property-powerpoint.md)|Returns the width, in points, of the object. Read-only.|
