# Floor Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](floor-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Creator](floor-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](floor-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Name](floor-name-property-powerpoint.md)|Returns the name of the object. Read-only  **String**.|
|[Parent](floor-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[PictureType](floor-picturetype-property-powerpoint.md)|Returns or sets a value that specifies how pictures are displayed on the walls and faces of a 3-D chart. Read/write  **Variant**.|
|[Thickness](floor-thickness-property-powerpoint.md)|Returns or sets the thickness of the floor. Read/write  **Long**.|
