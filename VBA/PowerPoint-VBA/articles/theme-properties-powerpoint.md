# Theme Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](theme-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Parent](theme-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[ThemeVariants](theme-themevariants-property-powerpoint.md)|Returns a  **[ThemeVariants](themevariant-object-powerpoint.md)** collection that represents the variations in the theme.|
