# DataLabels.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents a **[DataLabels](datalabels-object-powerpoint.md)** object.
