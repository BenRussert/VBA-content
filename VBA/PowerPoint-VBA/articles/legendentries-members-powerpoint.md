# LegendEntries Members (PowerPoint)
A collection of all the  **[LegendEntry](legendentry-object-powerpoint.md)** objects in the specified chart legend.

A collection of all the  **[LegendEntry](legendentry-object-powerpoint.md)** objects in the specified chart legend.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Item](legendentries-item-method-powerpoint.md)|Returns a single object from a collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](legendentries-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Count](legendentries-count-property-powerpoint.md)|Returns the number of objects in the collection. Read-only  **Long**.|
|[Creator](legendentries-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Parent](legendentries-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
