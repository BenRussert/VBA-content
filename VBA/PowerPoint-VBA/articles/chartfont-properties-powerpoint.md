# ChartFont Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chartfont-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Background](chartfont-background-property-powerpoint.md)|Returns or sets the type of background for text used in charts. Read/write  **Variant** that is set to one of the constants of **[XlBackground](xlbackground-enumeration-powerpoint.md)**.|
|[Bold](chartfont-bold-property-powerpoint.md)|**True** if the font is bold. Read/write **Variant**.|
|[Color](chartfont-color-property-powerpoint.md)|Returns or sets the primary color of the object. Read/write  **Variant**.|
|[ColorIndex](chartfont-colorindex-property-powerpoint.md)|Returns or sets the color of the font. Read/write  **Variant**.|
|[Creator](chartfont-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[FontStyle](chartfont-fontstyle-property-powerpoint.md)|Returns or sets the font style. Read/write  **String**.|
|[Italic](chartfont-italic-property-powerpoint.md)|**True** if the font style is italic. Read/write **Boolean**.|
|[Name](chartfont-name-property-powerpoint.md)|Returns or sets the name of the object. Read/write  **String**.|
|[Parent](chartfont-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Size](chartfont-size-property-powerpoint.md)|Returns or sets the size of the font. Read/write  **Variant**.|
|[Strikethrough](chartfont-strikethrough-property-powerpoint.md)|**True** if the font is struck through with a horizontal line. Read/write **Boolean**.|
|[Subscript](chartfont-subscript-property-powerpoint.md)|**True** if the font is formatted as subscript. The default is **False**. Read/write **Variant**.|
|[Superscript](chartfont-superscript-property-powerpoint.md)|**True** if the font is formatted as superscript. The default is **False**. Read/write **Variant**.|
|[Underline](chartfont-underline-property-powerpoint.md)|Returns or sets the type of underline applied to the font. Can be one of the  **[XlUnderlineStyle](xlunderlinestyle-enumeration-powerpoint.md)** constants. Read/write **Variant**.|
