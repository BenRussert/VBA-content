# ChartGroup Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chartgroup-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[AxisGroup](chartgroup-axisgroup-property-powerpoint.md)|Returns the type of axis group. Read/write  **[XlAxisGroup](xlaxisgroup-enumeration-powerpoint.md)**.|
|[BinsCountValue](chartgroup-binscountvalue-property-powerpoint.md)|Specifies the number of bins in the histogram chart. Read/write  **Long**.|
|[BinsOverflowEnabled](chartgroup-binsoverflowenabled-property-powerpoint.md)|Specifies whether a bin for values above the [ChartGroup.BinsOverflowValue Property (PowerPoint)](chartgroup-binsoverflowvalue-property-powerpoint.md) is enabled. Read/write **Boolean**.|
|[BinsOverflowValue](chartgroup-binsoverflowvalue-property-powerpoint.md)|If an [ChartGroup.BinsOverflowEnabled Property (PowerPoint)](chartgroup-binsoverflowenabled-property-powerpoint.md) is **True**, specifies the value above which an overflow bin is displayed. Read/write  **Double**.|
|[BinsType](chartgroup-binstype-property-powerpoint.md)|Specifies how the horizontal axis of the histogram chart is formatted, by bins type. Read/write [XlBinsType Enumeration (PowerPoint)](xlbinstype-enumeration-powerpoint.md).|
|[BinsUnderflowEnabled](chartgroup-binsunderflowenabled-property-powerpoint.md)|Specifies whether a bin for values below the [ChartGroup.BinsUnderflowValue Property (PowerPoint)](chartgroup-binsunderflowvalue-property-powerpoint.md) is enabled. Read/write **Boolean**.|
|[BinsUnderflowValue](chartgroup-binsunderflowvalue-property-powerpoint.md)|If [ChartGroup.BinsUnderflowEnabled Property (PowerPoint)](chartgroup-binsunderflowenabled-property-powerpoint.md) is **True**, specifies the value below which an underflow bin is displayed. Read/write  **Double**.|
|[BinWidthValue](chartgroup-binwidthvalue-property-powerpoint.md)|Specifies the number of points in each range. Read/write  **Double**.|
|[BubbleScale](chartgroup-bubblescale-property-powerpoint.md)|Returns or sets the scale factor for bubbles in the specified chart group. Read/write  **Long**.|
|[Creator](chartgroup-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[DoughnutHoleSize](chartgroup-doughnutholesize-property-powerpoint.md)|Returns or sets the size of the hole in a doughnut chart group. Read/write  **Long**.|
|[DownBars](chartgroup-downbars-property-powerpoint.md)|Returns the down bars on a line chart. Read-only  **[DownBars](downbars-object-powerpoint.md)**.|
|[DropLines](chartgroup-droplines-property-powerpoint.md)|Returns the drop lines for a series on a line chart or area chart. Read-only  **[DropLines](droplines-object-powerpoint.md)**.|
|[FirstSliceAngle](chartgroup-firstsliceangle-property-powerpoint.md)|Returns or sets the angle, in degrees (clockwise from vertical), of the first pie-chart or doughnut-chart slice. Read/write  **Long**.|
|[GapWidth](chartgroup-gapwidth-property-powerpoint.md)|For bar and column charts, returns or sets the space, as a percentage of the bar or column width, between bar or column clusters. For pie-of-pie and bar-of-pie charts, returns or sets the space between the primary and secondary sections of the chart. Read/write  **Long**.|
|[Has3DShading](chartgroup-has3dshading-property-powerpoint.md)|**True** if a chart group has three-dimensional shading. Read/write **Boolean**.|
|[HasDropLines](chartgroup-hasdroplines-property-powerpoint.md)|**True** if the line chart or area chart has drop lines. Read/write **Boolean**.|
|[HasHiLoLines](chartgroup-hashilolines-property-powerpoint.md)|**True** if the line chart has high-low lines. Read/write **Boolean**.|
|[HasRadarAxisLabels](chartgroup-hasradaraxislabels-property-powerpoint.md)|**True** if a radar chart has axis labels. Read/write **Boolean**.|
|[HasSeriesLines](chartgroup-hasserieslines-property-powerpoint.md)|**True** if a stacked column chart or bar chart has series lines or if a pie-of-pie chart or bar-of-pie chart has connector lines between the two sections. Read/write **Boolean**.|
|[HasUpDownBars](chartgroup-hasupdownbars-property-powerpoint.md)|**True** if a line chart has up and down bars. Read/write **Boolean**.|
|[HiLoLines](chartgroup-hilolines-property-powerpoint.md)|Returns the high-low lines for a series on a line chart. Read-only  **[HiLoLines](hilolines-object-powerpoint.md)**.|
|[Index](chartgroup-index-property-powerpoint.md)|Returns the index number of the object within the collection of similar objects. Read-only  **Long**.|
|[Overlap](chartgroup-overlap-property-powerpoint.md)|Specifies how bars and columns are positioned. Read/write  **Long**.|
|[Parent](chartgroup-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[RadarAxisLabels](chartgroup-radaraxislabels-property-powerpoint.md)|Returns the radar axis labels for the specified chart group. Read-only  **[TickLabels](ticklabels-object-powerpoint.md)**.|
|[SecondPlotSize](chartgroup-secondplotsize-property-powerpoint.md)|Returns or sets the size, as a percentage of the primary pie, of the secondary section of either a pie-of-pie chart or a bar-of-pie chart. Read/write  **Long**.|
|[SeriesLines](chartgroup-serieslines-property-powerpoint.md)|Returns the series lines for a 2-D stacked bar, 2-D stacked column, pie-of-pie, or bar-of-pie chart. Read-only  **[SeriesLines](serieslines-object-powerpoint.md)**.|
|[ShowNegativeBubbles](chartgroup-shownegativebubbles-property-powerpoint.md)|**True** if negative bubbles are shown for the chart group. Read/write **Boolean**.|
|[SizeRepresents](chartgroup-sizerepresents-property-powerpoint.md)|Returns or sets what the bubble size represents on a bubble chart. Read/write  **Long**.|
|[SplitType](chartgroup-splittype-property-powerpoint.md)|Returns or sets the way the two sections of either a pie-of-pie chart or a bar-of-pie chart are split. Read/write  **[XlChartSplitType](xlchartsplittype-enumeration-powerpoint.md)**.|
|[SplitValue](chartgroup-splitvalue-property-powerpoint.md)|Returns or sets the threshold value separating the two sections of either a pie-of-pie chart or a bar-of-pie chart. Read/write  **Variant**.|
|[UpBars](chartgroup-upbars-property-powerpoint.md)|Returns the up bars on a line chart. Read-only  **[UpBars](upbars-object-powerpoint.md)**.|
|[VaryByCategories](chartgroup-varybycategories-property-powerpoint.md)|**True** if Microsoft Word assigns a different color or pattern to each data marker. Read/write **Boolean**.|
