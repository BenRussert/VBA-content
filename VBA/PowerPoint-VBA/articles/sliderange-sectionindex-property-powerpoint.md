# SlideRange.sectionIndex Property (PowerPoint)

Returns the index of the selected section in the  **SlideRange**. Read-only.


## Syntax

 _expression_. **sectionIndex**

 _expression_ A variable that represents a **SlideRange** object.


### Return Value

Single
