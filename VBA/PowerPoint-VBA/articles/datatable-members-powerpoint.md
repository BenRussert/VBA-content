# DataTable Members (PowerPoint)
Represents a chart data table.

Represents a chart data table.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](datatable-delete-method-powerpoint.md)|Deletes the object.|
|[Select](datatable-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](datatable-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Border](datatable-border-property-powerpoint.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-powerpoint.md)**.|
|[Creator](datatable-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Font](datatable-font-property-powerpoint.md)|Returns the font of the specified object. Read-only  **[ChartFont](chartfont-object-powerpoint.md)**.|
|[Format](datatable-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[HasBorderHorizontal](datatable-hasborderhorizontal-property-powerpoint.md)|**True** if the chart data table has horizontal cell borders. Read/write **Boolean**.|
|[HasBorderOutline](datatable-hasborderoutline-property-powerpoint.md)|**True** if the chart data table has outline borders. Read/write **Boolean**.|
|[HasBorderVertical](datatable-hasbordervertical-property-powerpoint.md)|**True** if the chart data table has vertical cell borders. Read/write **Boolean**.|
|[Parent](datatable-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[ShowLegendKey](datatable-showlegendkey-property-powerpoint.md)|**True** if the data label legend key is visible. Read/write **Boolean**.|
