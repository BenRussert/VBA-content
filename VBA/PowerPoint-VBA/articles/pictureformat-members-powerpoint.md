# PictureFormat Members (PowerPoint)
Contains properties and methods that apply to pictures and OLE objects. 

Contains properties and methods that apply to pictures and OLE objects. 


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[IncrementBrightness](pictureformat-incrementbrightness-method-powerpoint.md)|Changes the brightness of the picture by the specified amount. |
|[IncrementContrast](pictureformat-incrementcontrast-method-powerpoint.md)|Changes the contrast of the picture by the specified amount. |

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](pictureformat-application-property-powerpoint.md)|Returns an  **[Application](application-object-powerpoint.md)** object that represents the creator of the specified object.|
|[Brightness](pictureformat-brightness-property-powerpoint.md)|Returns or sets the brightness of the specified picture or OLE object. Read/write.|
|[ColorType](pictureformat-colortype-property-powerpoint.md)|Returns or sets the type of color transformation applied to the specified picture or OLE object. Read/write.|
|[Contrast](pictureformat-contrast-property-powerpoint.md)|Returns or sets the contrast for the specified picture or OLE object.|
|[Creator](pictureformat-creator-property-powerpoint.md)|Returns a  **Long** that represents the four-character creator code for the application in which the specified object was created. For example, if the object was created in Microsoft PowerPoint, this property returns the hexadecimal number 50575054. Read-only.|
|[Crop](pictureformat-crop-property-powerpoint.md)|Returns or sets the number of points that are cropped off the specified picture or OLE object. Read-only.|
|[CropBottom](pictureformat-cropbottom-property-powerpoint.md)|Returns or sets the number of points that are cropped off the bottom of the specified picture or OLE object. Read/write. |
|[CropLeft](pictureformat-cropleft-property-powerpoint.md)|Returns or sets the number of points that are cropped off the left side of the specified picture or OLE object. Read/write.|
|[CropRight](pictureformat-cropright-property-powerpoint.md)|Returns or sets the number of points that are cropped off the right side of the specified picture or OLE object. Read/write. |
|[CropTop](pictureformat-croptop-property-powerpoint.md)|Returns or sets the number of points that are cropped off the top of the specified picture or OLE object. Read/write. |
|[Parent](pictureformat-parent-property-powerpoint.md)|Returns the parent object for the specified object.|
|[TransparencyColor](pictureformat-transparencycolor-property-powerpoint.md)|Returns or sets the transparent color for the specified picture as a red-green-blue (RGB) value. Read/write.|
|[TransparentBackground](pictureformat-transparentbackground-property-powerpoint.md)|Determines whether parts of the picture that are the color defined as the transparent color appear transparent. Applies to bitmaps only. Read/write. |
