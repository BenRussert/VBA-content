# ResampleMediaTasks.Cancel Method (PowerPoint)

Cancels mobilization of the current media asset.


## Syntax

 _expression_. **Cancel**

 _expression_ A variable that represents a **ResampleMediaTasks** object.


### Return Value

Nothing
