# ThemeVariants Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](themevariants-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Count](themevariants-count-property-powerpoint.md)|Returns the number of objects in the specified collection. Read-only.|
|[Parent](themevariants-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
