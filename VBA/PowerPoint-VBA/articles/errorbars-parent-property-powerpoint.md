# ErrorBars.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **[ErrorBars](errorbars-object-powerpoint.md)** object.
