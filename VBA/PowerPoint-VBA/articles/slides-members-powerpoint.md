# Slides Members (PowerPoint)
A collection of all the  **[Slide](slide-object-powerpoint.md)** objects in the specified presentation.

A collection of all the  **[Slide](slide-object-powerpoint.md)** objects in the specified presentation.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[AddSlide](slides-addslide-method-powerpoint.md)|Creates a new slide, adds it to the  **[Slides](slides-object-powerpoint.md)** collection, and returns the slide.|
|[FindBySlideID](slides-findbyslideid-method-powerpoint.md)|Returns a  **[Slide](slide-object-powerpoint.md)** object that represents the slide with the specified slide ID number. Each slide is automatically assigned a unique slide ID number when it is created. Use the **SlideID** property to return a slide's ID number.|
|[InsertFromFile](slides-insertfromfile-method-powerpoint.md)|Inserts slides from a file into a presentation, at the specified location. Returns an  **Integer** that represents the number of slides inserted.|
|[Item](slides-item-method-powerpoint.md)|Returns a single  **Slide** object from the specified **Slides** collection.|
|[Paste](slides-paste-method-powerpoint.md)|Pastes the slides on the Clipboard into the  **Slides** collection for the presentation. Specify where you want to insert the slides with the **Index** argument. Returns a **[SlideRange](sliderange-object-powerpoint.md)** object that represents the pasted objects. Each pasted slide becomes a member of the specified **Slides** collection.|
|[Range](slides-range-method-powerpoint.md)|Returns a  **[SlideRange](sliderange-object-powerpoint.md)** object that represents a subset of the slides in a **[Slides](slides-object-powerpoint.md)** collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](slides-application-property-powerpoint.md)|Returns an  **[Application](application-object-powerpoint.md)** object that represents the creator of the specified object.|
|[Count](slides-count-property-powerpoint.md)|Returns the number of objects in the specified collection. Read-only.|
|[Parent](slides-parent-property-powerpoint.md)|Returns the parent object for the specified object.|
