# ShapeRange.ShapeStyle Property (PowerPoint)

Sets or returns the shape style index for the specified object.


## Syntax

 _expression_. **ShapeStyle**

 _expression_ An expression that returns a **ShapeRange** object.


## Remarks

The value of the  **ShapeStyle** property can be one of the **[MsoShapeStyleIndex](http://msdn.microsoft.com/library/msoshapestyleindex-enumeration-office%28Office.15%29.aspx)** constants.
