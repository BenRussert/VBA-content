# SeriesLines Members (PowerPoint)
Represents series lines in a chart group.

Represents series lines in a chart group.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](serieslines-delete-method-powerpoint.md)|Deletes the object.|
|[Select](serieslines-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](serieslines-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Border](serieslines-border-property-powerpoint.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-powerpoint.md)**.|
|[Creator](serieslines-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](serieslines-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Name](serieslines-name-property-powerpoint.md)|Returns the name of the object. Read-only  **String**.|
|[Parent](serieslines-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
