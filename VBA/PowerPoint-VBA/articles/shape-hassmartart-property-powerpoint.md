# Shape.HasSmartArt Property (PowerPoint)

Returns  **True** if the current **Shape** object contains a SmartArt diagram. Read-only.


## Syntax

 _expression_. **HasSmartArt**

 _expression_ A variable that represents a **Shape** object.


### Return Value

Boolean
