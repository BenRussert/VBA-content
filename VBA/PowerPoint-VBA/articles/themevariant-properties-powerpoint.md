# ThemeVariant Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](themevariant-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Height](themevariant-height-property-powerpoint.md)|Returns the height of the theme variation. Read-only.|
|[Id](themevariant-id-property-powerpoint.md)|Returns a string that represents the ID of the theme variation. Read-only.|
|[Name](themevariant-name-property-powerpoint.md)|Returns the name of the theme variation. Read-only.|
|[Parent](themevariant-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Width](themevariant-width-property-powerpoint.md)|Returns the height of the theme variation. Read-only.|
