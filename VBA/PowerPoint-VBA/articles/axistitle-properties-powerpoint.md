# AxisTitle Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](axistitle-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Caption](axistitle-caption-property-powerpoint.md)|Returns or sets a  **String** value that represents the axis title text.|
|[Characters](axistitle-characters-property-powerpoint.md)|Returns a  **[ChartCharacters](chartcharacters-object-powerpoint.md)** object that represents a range of characters within the object text. You can use the **ChartCharacters** object to format characters within a text string.|
|[Creator](axistitle-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](axistitle-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Formula](axistitle-formula-property-powerpoint.md)|Returns or sets the object's formula in A1-style notation. Read-only.|
|[FormulaLocal](axistitle-formulalocal-property-powerpoint.md)|Returns or sets the formula for the object, using A1-style references in the language of the user. Read/write.|
|[FormulaR1C1](axistitle-formular1c1-property-powerpoint.md)|Returns or sets the formula for the object by using R1C1-style notation. Read/write.|
|[FormulaR1C1Local](axistitle-formular1c1local-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the user. Read/write.|
|[Height](axistitle-height-property-powerpoint.md)|Returns the height, in points, of the object. Read-only|
|[HorizontalAlignment](axistitle-horizontalalignment-property-powerpoint.md)|Returns or sets the horizontal alignment for the specified object. Read/write  **Variant**.|
|[IncludeInLayout](axistitle-includeinlayout-property-powerpoint.md)|**True** if an axis title will occupy the chart layout space when a chart layout is being determined. The default is **True**. Read/write **Boolean**.|
|[Left](axistitle-left-property-powerpoint.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double**.|
|[Name](axistitle-name-property-powerpoint.md)|Returns a  **String** value that represents the name of the object.|
|[Orientation](axistitle-orientation-property-powerpoint.md)|Returns or sets the text orientation. Read/write  **Long**.|
|[Parent](axistitle-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Position](axistitle-position-property-powerpoint.md)|Returns or sets the position of the axis title on the chart. Read/write  **[XlChartElementPosition](xlchartelementposition-enumeration-powerpoint.md)**.|
|[ReadingOrder](axistitle-readingorder-property-powerpoint.md)|Returns or sets an  **[XlReadingOrder](xlreadingorder-enumeration-powerpoint.md)** constant that represents the reading order for the specified object. Read/write **Long**.|
|[Shadow](axistitle-shadow-property-powerpoint.md)|Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean**.|
|[Text](axistitle-text-property-powerpoint.md)|Returns or sets the text for the specified object. Read/write  **String**.|
|[Top](axistitle-top-property-powerpoint.md)|Returns or sets the distance, in points, from the top edge of the object to the top of row 1 (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double**.|
|[VerticalAlignment](axistitle-verticalalignment-property-powerpoint.md)|Returns or sets the vertical alignment of the specified object. Read/write  **Variant**.|
|[Width](axistitle-width-property-powerpoint.md)|Returns a  **Double** value that represents the width, in points, of the object. Read-only.|
