# Trendline Members (PowerPoint)
Represents a trendline in a chart.

Represents a trendline in a chart.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[ClearFormats](trendline-clearformats-method-powerpoint.md)|Clears the formatting of the object.|
|[Delete](trendline-delete-method-powerpoint.md)|Deletes the object.|
|[Select](trendline-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](trendline-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Backward2](trendline-backward2-property-powerpoint.md)|Returns or sets the number of periods (or units on a scatter chart) that the trendline extends backward. Read/write  **Double**.|
|[Border](trendline-border-property-powerpoint.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-powerpoint.md)**.|
|[Creator](trendline-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[DataLabel](trendline-datalabel-property-powerpoint.md)|Returns the data label that is associated with the trendline. Read-only  **[DataLabel](datalabel-object-powerpoint.md)**.|
|[DisplayEquation](trendline-displayequation-property-powerpoint.md)|**True** if the equation for the trendline is displayed on the chart (in the same data label as the R-squared value). Read/write **Boolean**.|
|[DisplayRSquared](trendline-displayrsquared-property-powerpoint.md)|**True** if the R-squared value of the trendline is displayed on the chart (in the same data label as the equation). Read/write **Boolean**.|
|[Format](trendline-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Forward2](trendline-forward2-property-powerpoint.md)|Returns or sets the number of periods (or units on a scatter chart) that the trendline extends forward. Read/write  **Double**.|
|[Index](trendline-index-property-powerpoint.md)|Returns the index number of the object within the collection of similar objects. Read-only  **Long**.|
|[Intercept](trendline-intercept-property-powerpoint.md)|Returns or sets the point where the trendline crosses the value axis. Read/write  **Double**.|
|[InterceptIsAuto](trendline-interceptisauto-property-powerpoint.md)|**True** if the point where the trendline crosses the value axis is automatically determined by the regression. Read/write **Boolean**.|
|[Name](trendline-name-property-powerpoint.md)|Returns or sets name of the object. Read/write  **String**.|
|[NameIsAuto](trendline-nameisauto-property-powerpoint.md)|**True** if Microsoft Word automatically determines the name of the trendline. Read/write **Boolean**.|
|[Order](trendline-order-property-powerpoint.md)|Returns or sets the trendline order (an integer greater than 1) when the trendline type is  **xlPolynomial**. Read/write **Long**.|
|[Parent](trendline-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Period](trendline-period-property-powerpoint.md)|Returns or sets the period for the moving-average trendline. Read/write  **Long**.|
|[Type](trendline-type-property-powerpoint.md)|Returns or sets the trendline type. Read/write  **[XlTrendlineType](xltrendlinetype-enumeration-powerpoint.md)**.|
