# LegendEntry Members (PowerPoint)
Represents a legend entry in a chart legend.

Represents a legend entry in a chart legend.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](legendentry-delete-method-powerpoint.md)|Deletes the object.|
|[Select](legendentry-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](legendentry-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Creator](legendentry-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Font](legendentry-font-property-powerpoint.md)|Returns the font of the specified object. Read-only  **[ChartFont](chartfont-object-powerpoint.md)**.|
|[Format](legendentry-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Height](legendentry-height-property-powerpoint.md)|Returns the height, in points, of the object. Read-only  **Double**.|
|[Index](legendentry-index-property-powerpoint.md)|Returns the index number of the object within the collection of similar objects. Read-only  **Long**.|
|[Left](legendentry-left-property-powerpoint.md)|Returns the distance, in points, from the left edge of the object to the left edge of the chart area. Read-only  **Double**.|
|[LegendKey](legendentry-legendkey-property-powerpoint.md)|Returns the legend key that is associated with the entry. Read-only  **[LegendKey](legendkey-object-powerpoint.md)**.|
|[Parent](legendentry-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Top](legendentry-top-property-powerpoint.md)|Returns the distance, in points, from the top edge of the object to the top of the first row (on a worksheet) or the top of the chart area (on a chart). Read-only  **Double**.|
|[Width](legendentry-width-property-powerpoint.md)|Returns the width, in points, of the object. Read-only  **Double**.|
