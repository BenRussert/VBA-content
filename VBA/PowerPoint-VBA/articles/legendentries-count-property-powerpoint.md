# LegendEntries.Count Property (PowerPoint)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **[LegendEntries](legendentries-object-powerpoint.md)** object.
