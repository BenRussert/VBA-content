# SlideNavigation Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](slidenavigation-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Parent](slidenavigation-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Visible](slidenavigation-visible-property-powerpoint.md)|Specifies whether the slide navigation screen is displayed during a slide show. Read/write.|
