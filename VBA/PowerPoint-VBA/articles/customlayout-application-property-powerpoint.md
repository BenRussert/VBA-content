# CustomLayout.Application Property (PowerPoint)

Returns an  **[Application](application-object-powerpoint.md)** object that represents the creator of the specified object. Read-only.


## Syntax

 _expression_. **Application**

 _expression_ A variable that represents a **CustomLayout** object.


### Return Value

Application
