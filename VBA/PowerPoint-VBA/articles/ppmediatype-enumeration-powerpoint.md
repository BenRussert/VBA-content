# PpMediaType Enumeration (PowerPoint)

Indicates the OLE media type.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
|**ppMediaTypeMixed**|-2|Mixed|
|**ppMediaTypeMovie**|3|Movie|
|**ppMediaTypeOther**|1|Others|
|**ppMediaTypeSound**|2|Sound|
