# ChartTitle Members (PowerPoint)
Represents the chart title.

Represents the chart title.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](charttitle-delete-method-powerpoint.md)|Deletes the object.|
|[Select](charttitle-select-method-powerpoint.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](charttitle-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Caption](charttitle-caption-property-powerpoint.md)|Returns or sets the chart title text. Read/write  **String**.|
|[Characters](charttitle-characters-property-powerpoint.md)|Returns a  **[ChartCharacters](chartcharacters-object-powerpoint.md)** object that represents a range of characters within the object text. You can use the **ChartCharacters** object to format characters within a text string.|
|[Creator](charttitle-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Format](charttitle-format-property-powerpoint.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-powerpoint.md)**.|
|[Formula](charttitle-formula-property-powerpoint.md)|Returns or sets the object's formula in A1-style notation. Read/write.|
|[FormulaLocal](charttitle-formulalocal-property-powerpoint.md)|Returns or sets the formula for the object, using A1-style references in the language of the user. Read/write  **String**.|
|[FormulaR1C1](charttitle-formular1c1-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the macro. Read/write  **String**.|
|[FormulaR1C1Local](charttitle-formular1c1local-property-powerpoint.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the user. Read/write  **String**.|
|[Height](charttitle-height-property-powerpoint.md)|Returns or sets a  **Double** value that represents the height, in points, of the object. Read-only.|
|[HorizontalAlignment](charttitle-horizontalalignment-property-powerpoint.md)|Returns or sets the horizontal alignment for the specified object. Read/write  **Variant**.|
|[IncludeInLayout](charttitle-includeinlayout-property-powerpoint.md)|**True** if a chart title will occupy the chart layout space when a chart layout is being determined. The default is **True**. Read/write **Boolean**.|
|[Left](charttitle-left-property-powerpoint.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double**.|
|[Name](charttitle-name-property-powerpoint.md)|Returns the name of the object. Read-only  **String**.|
|[Orientation](charttitle-orientation-property-powerpoint.md)|Returns or sets the text orientation. Read/write  **Long**.|
|[Parent](charttitle-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Position](charttitle-position-property-powerpoint.md)|Returns or sets the position of the chart title on the chart. Read/write  **[XlChartElementPosition](xlchartelementposition-enumeration-powerpoint.md)**.|
|[ReadingOrder](charttitle-readingorder-property-powerpoint.md)|Returns or sets an  **[XlReadingOrder](xlreadingorder-enumeration-powerpoint.md)** constant that represents the reading order for the specified object. Read/write **Long**.|
|[Shadow](charttitle-shadow-property-powerpoint.md)|Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean**.|
|[Text](charttitle-text-property-powerpoint.md)|Returns or sets the text for the specified object. Read/write  **String**.|
|[Top](charttitle-top-property-powerpoint.md)|Returns or sets the distance, in points, from the top edge of the object to the top of the first row (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double**.|
|[VerticalAlignment](charttitle-verticalalignment-property-powerpoint.md)|Returns or sets the vertical alignment of the specified object. Read/write  **Variant**.|
|[Width](charttitle-width-property-powerpoint.md)|Returns or sets a  **Double** value that represents the width, in points, of the object. Read-only.|
