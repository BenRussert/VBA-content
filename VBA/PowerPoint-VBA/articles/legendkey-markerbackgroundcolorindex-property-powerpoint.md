# LegendKey.MarkerBackgroundColorIndex Property (PowerPoint)

Returns or sets the marker background color as an index into the current color palette, or as one of the following  **[XlColorIndex](xlcolorindex-enumeration-powerpoint.md)** constants: **xlColorIndexAutomatic** or **xlColorIndexNone**. Read/write **Long**.


## Syntax

 _expression_. **MarkerBackgroundColorIndex**

 _expression_ A variable that represents a **[LegendKey](legendkey-object-powerpoint.md)** object.


## Remarks

This property applies only to line, scatter, and radar charts.
