# ChartFormat Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Adjustments](chartformat-adjustments-property-powerpoint.md)|Returns an [Adjustments](adjustments-object-powerpoint.md) object that contains a collection of adjustment values for the specified **Chart Format** object.|
|[Application](chartformat-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[AutoShapeType](chartformat-autoshapetype-property-powerpoint.md)|Returns or sets the auto shape type of the specified object. Read-write [MSOAutoShapeType](msoautoshapetype-enumeration-office.md).|
|[Creator](chartformat-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[Fill](chartformat-fill-property-powerpoint.md)|Returns a  **[FillFormat](fillformat-object-powerpoint.md)** object for the parent chart element that contains fill formatting properties for the chart element. Read-only.|
|[Glow](chartformat-glow-property-powerpoint.md)|Returns the glow formatting properties for the chart element. Read-only  **[GlowFormat](glowformat-object-office.md)**.|
|[Line](chartformat-line-property-powerpoint.md)|Returns the line formatting properties for the specified chart element. Read-only  **[LineFormat](lineformat-object-powerpoint.md)**.|
|[Parent](chartformat-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[PictureFormat](chartformat-pictureformat-property-powerpoint.md)|Returns a  **[PictureFormat](pictureformat-object-powerpoint.md)** object for a specified chart that contains pictures. Read-only.|
|[Shadow](chartformat-shadow-property-powerpoint.md)|Returns shadow formatting properties for the chart element. Read-only  **[ShadowFormat](shadowformat-object-powerpoint.md)**.|
|[SoftEdge](chartformat-softedge-property-powerpoint.md)|Returns the soft edge formatting for a shape. Read-only  **[SoftEdgeFormat](softedgeformat-object-office.md)**.|
|[TextFrame2](chartformat-textframe2-property-powerpoint.md)|Returns the text formatting for the specified chart element. Read-only  **[TextFrame2](textframe2-object-powerpoint.md)**.|
|[ThreeD](chartformat-threed-property-powerpoint.md)|Returns the 3-D-effect formatting properties for the specified chart. Read-only  **[ThreeDFormat](threedformat-object-powerpoint.md)**.|
