# ChartBorder Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chartborder-application-property-powerpoint.md)|When used without an object qualifier, returns an  **[Application](application-object-powerpoint.md)** object that represents the Microsoft PowerPoint application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Color](chartborder-color-property-powerpoint.md)|Returns or sets the primary color of the object. Read/write  **Variant**.|
|[ColorIndex](chartborder-colorindex-property-powerpoint.md)|Returns or sets the color of the border. Read/write  **Variant**.|
|[Creator](chartborder-creator-property-powerpoint.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long**.|
|[LineStyle](chartborder-linestyle-property-powerpoint.md)|Returns or sets the line style for the border. Read/write  **[XlLineStyle](xllinestyle-enumeration-powerpoint.md)**, **xlGray25**, **xlGray50**, **xlGray75**, or **xlAutomatic**.|
|[Parent](chartborder-parent-property-powerpoint.md)|Returns the parent for the specified object. Read-only  **Object**.|
|[Weight](chartborder-weight-property-powerpoint.md)|Returns or sets the weight of the border. Read/write  **[XlBorderWeight](xlborderweight-enumeration-powerpoint.md)**.|
