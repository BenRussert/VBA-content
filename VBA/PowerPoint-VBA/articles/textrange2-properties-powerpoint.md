# TextRange2 Properties (PowerPoint)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](http://msdn.microsoft.com/library/textrange2-application-property-office%28Office.15%29.aspx)|Used without an object qualifier, this property returns an  **Application** object that represents the current instance of the Microsoft Office application. Used with an object qualifier, this property returns an **Application** object that represents the creator of the **TextRange2** object. When used with an OLE Automation object, it returns the object's application. Read-only.|
|[BoundHeight](http://msdn.microsoft.com/library/textrange2-boundheight-property-office%28Office.15%29.aspx)|Gets the height, in points, of the text bounding box for the specified text. Read-only.|
|[BoundLeft](http://msdn.microsoft.com/library/textrange2-boundleft-property-office%28Office.15%29.aspx)|Gets the left coordinate, in points, of the text bounding box for the specified text. Read-only.|
|[BoundTop](http://msdn.microsoft.com/library/textrange2-boundtop-property-office%28Office.15%29.aspx)|Gets the top coordinate, in points, of the text bounding box for the specified text. Read-only.|
|[BoundWidth](http://msdn.microsoft.com/library/textrange2-boundwidth-property-office%28Office.15%29.aspx)|Gets the width, in points, of the text bounding box for the specified text. Read-only.|
|[Characters](http://msdn.microsoft.com/library/textrange2-characters-property-office%28Office.15%29.aspx)|Read-only.|
|[Count](http://msdn.microsoft.com/library/textrange2-count-property-office%28Office.15%29.aspx)|Gets a  **Long** indicating the number of items in the **TextRange2** collection. Read-only.|
|[Creator](http://msdn.microsoft.com/library/textrange2-creator-property-office%28Office.15%29.aspx)|Gets a 32-bit integer that indicates the application in which the **TextRange2** object was created. Read-only.|
|[Font](http://msdn.microsoft.com/library/textrange2-font-property-office%28Office.15%29.aspx)|Returns a  **Font** object that represents character formatting for the **TextRange2** object. Read-only.|
|[LanguageID](http://msdn.microsoft.com/library/textrange2-languageid-property-office%28Office.15%29.aspx)|Gets or sets the  **MsoLanguageID** value of the **TextRange2** object. Read/write.|
|[Length](http://msdn.microsoft.com/library/textrange2-length-property-office%28Office.15%29.aspx)|Get a Long that represents the length of a text range. Read-only.|
|[Lines](http://msdn.microsoft.com/library/textrange2-lines-property-office%28Office.15%29.aspx)|Returns a TextRange2 object that represents the specified subset of text lines. Read-only.|
|[MathZones](http://msdn.microsoft.com/library/textrange2-mathzones-property-office%28Office.15%29.aspx)|Sets the starting point and length of a math zone within a text range. Read-only|
|[ParagraphFormat](http://msdn.microsoft.com/library/textrange2-paragraphformat-property-office%28Office.15%29.aspx)|Returns a  **ParagraphFormat** object that represents paragraph formatting for the specified text. Read-only.|
|[Paragraphs](http://msdn.microsoft.com/library/textrange2-paragraphs-property-office%28Office.15%29.aspx)|Gets a  **TextRange2** object that represents the specified subset of text paragraphs. Read-only.|
|[Parent](http://msdn.microsoft.com/library/textrange2-parent-property-office%28Office.15%29.aspx)|Gets the  **Parent** object for the **TextRange2** object. Read-only.|
|[Runs](http://msdn.microsoft.com/library/textrange2-runs-property-office%28Office.15%29.aspx)|Gets a  **TextRange2** object that represents the specified subset of text runs. A text run consists of a range of characters that share the same font attributes. Read-only.|
|[Sentences](http://msdn.microsoft.com/library/textrange2-sentences-property-office%28Office.15%29.aspx)|Returns a  **TextRange2** object that represents the specified subset of text sentences. Read-only.|
|[Start](http://msdn.microsoft.com/library/textrange2-start-property-office%28Office.15%29.aspx)|Gets a  **Long** value indicating the starting point of the specified text range. Read-only.|
|[Text](http://msdn.microsoft.com/library/textrange2-text-property-office%28Office.15%29.aspx)|Gets or sets a  **String** value that represents the text in a text range. Read/write.|
|[Words](http://msdn.microsoft.com/library/textrange2-words-property-office%28Office.15%29.aspx)|Gets a  **TextRange2** object that represents the specified subset of text words. Read-only.|
