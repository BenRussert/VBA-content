# Effect.Shape Property (PowerPoint)

Returns a  **[Shape](shape-object-powerpoint.md)** object that represents an animated shape.


## Syntax

 _expression_. **Shape**

 _expression_ A variable that represents an **Effect** object.


### Return Value

Shape
