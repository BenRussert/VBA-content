# Calendar Constants

The following [constants](vbe-glossary.md) can be used anywhere in your code in place of the actual values:



|**Constant**|**Value**|**Description**|
|:-----|:-----|:-----|
|**vbCalGreg**|0|Indicates that the Gregorian calendar is used.|
|**vbCalHijri**|1|Indicates that the Hijri calendar is used.|
