# Cell Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](cell-application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [BorderBottom](cell-borderbottom-property-publisher.md)|Returns a  [CellBorder](cellborder-object-publisher.md)object that represents the bottom border for a specified table cell.|
| [BorderDiagonal](cell-borderdiagonal-property-publisher.md)|Returns a  [CellBorder](cellborder-object-publisher.md)object that represents the diagonal border for a specified table cell.|
| [BorderLeft](cell-borderleft-property-publisher.md)|Returns a  [CellBorder](cellborder-object-publisher.md)object that represents the left border for a specified table cell.|
| [BorderRight](cell-borderright-property-publisher.md)|Returns a  [CellBorder](cellborder-object-publisher.md)object that represents the right border for a specified table cell.|
| [BorderTop](cell-bordertop-property-publisher.md)|Returns a  [CellBorder](cellborder-object-publisher.md)object that represents the top border for a specified table cell.|
| [CellTextOrientation](cell-celltextorientation-property-publisher.md)|Returns or sets a  **PbTextOrientation** that represents the flow of text in a specified table cell. Read/write.|
| [Column](cell-column-property-publisher.md)|Returns a  **Long** that represents the table column containing the specified cell. Read-only.|
| [Diagonal](cell-diagonal-property-publisher.md)|Sets or returns a  **PbCellDiagonalType** constant that represents a cell that is diagonally split. Read/write.|
| [Fill](cell-fill-property-publisher.md)| Returns a **[FillFormat](fillformat-object-publisher.md)** object representing the fill for the specified shape or table cell.|
| [HasText](cell-hastext-property-publisher.md)|Returns a  **Boolean** value indicating whether the specified cell contains any text. Returns **True** if the specified cell contains text. Read-only.|
| [Height](cell-height-property-publisher.md)|Returns a  **Long** that represent the height (in cells) of a table, range of cells, or page. Read-only.|
| [MarginBottom](cell-marginbottom-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the bottom edge of a cell, text frame, or page. Read/write.|
| [MarginLeft](cell-marginleft-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the left edge of a cell, text frame, or page. Read/write.|
| [MarginRight](cell-marginright-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the right edge of a cell, text frame, or page. Read/write.|
| [MarginTop](cell-margintop-property-publisher.md)|Returns or sets a  **Variant** that represents the amount of space (in points) between the text and the top edge of a cell, text frame, or page. Read/write.|
| [Parent](cell-parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [Row](cell-row-property-publisher.md)|Returns a  **Long** that represents the row number containing the specified cell. Read-only.|
| [Selected](cell-selected-property-publisher.md)|Returns  **True** if a cell is selected. Read-only **Boolean**.|
| [TextRange](cell-textrange-property-publisher.md)|Returns a  **[TextRange](textrange-object-publisher.md)** object that represents the text that is attached to a shape and properties and methods for manipulating the text.|
| [VerticalTextAlignment](cell-verticaltextalignment-property-publisher.md)|Returns or sets a  **PbVerticalTextAlignmentType**constant that represents the vertical alignment of text in a text box. Read/write.|
| [Width](cell-width-property-publisher.md)|Returns a  **Long** that represent the width (in cells) of a table, range of cells, or page. Read-only.|
|Name|Description|
