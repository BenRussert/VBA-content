# ColorsInUse Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](colorsinuse.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Count](colorsinuse.count-property-publisher.md)|Returns a  **Long** that represents the number of items in the specified collection.|
| [Item](colorsinuse.item-property-publisher.md)|Returns an individual object from a specified collection. Read-only.|
| [Parent](colorsinuse.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
|Name|Description|
