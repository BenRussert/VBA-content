# PrintableRect Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](printablerect.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Height](printablerect.height-property-publisher.md)|Returns a  **Single** that represents the height, in points, of the printable rectangle. Read-only.|
| [Left](printablerect.left-property-publisher.md)|Returns a  **Single** indicating the distance (in points) from the left edge of the printer sheet to the left edge of the printable rectangle. Read-only.|
| [Parent](printablerect.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. Read-only.|
| [Top](printablerect.top-property-publisher.md)|Returns the a  **Single** that represents the distance (in points) from the top edge of the printer sheet to the top edge of the printable rectangle. Read-only.|
| [Width](printablerect.width-property-publisher.md)|Returns a  **Single** that represents the printable rectangle. Read-only.|
|Name|Description|
