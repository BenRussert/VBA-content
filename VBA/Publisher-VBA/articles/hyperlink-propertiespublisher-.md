# Hyperlink Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Address](hyperlink.address-property-publisher.md)|Returns or sets a  **String** that represents the URL address for a hyperlink. Read/write.|
| [Application](hyperlink.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [EmailSubject](hyperlink.emailsubject-property-publisher.md)|Sets or returns a  **String** that represents the subject for e-mail messages generated to process Web form data. Read/write.|
| [PageID](hyperlink.pageid-property-publisher.md)|Returns or sets a  **Long** indicating the page in the publication that is the destination for the specified hyperlink. Read/write.|
| [Parent](hyperlink.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [Range](hyperlink.range-property-publisher.md)|Returns a  **[TextRange](textrange-object-publisher.md)** object representing the base text to which the specified hyperlink has been applied.|
| [Shape](hyperlink.shape-property-publisher.md)|Returns a  **[Shape](shape-object-publisher.md)** object that represents the shape associated with a hyperlink.|
| [TargetType](hyperlink.targettype-property-publisher.md)|Returns a  **PbHlinkTargetType** constant that represents the type of hyperlink. Read-only.|
| [TextToDisplay](hyperlink.texttodisplay-property-publisher.md)|Returns or sets a  **String** that represents the text displayed for a hyperlink. Read/write.|
| [Type](hyperlink.type-property-publisher.md)|Returns an  **MsoHyperlinkType** constant that represents the hyperlink type. Read-only.|
|Name|Description|
