# TextEffectFormat Properties (Publisher)

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Alignment](texteffectformat.alignment-property-publisher.md)|Returns or sets a  **MsoTextEffectAlignment** constant that represents the alignment for the specified text effect. Read/write.|
| [Application](texteffectformat.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [FontBold](texteffectformat.fontbold-property-publisher.md)|Sets or returns an  **MsoTriState**constant that represents whether the font for a dropped capital letter or WordArt text effect is bold. Read/write.|
| [FontItalic](texteffectformat.fontitalic-property-publisher.md)|Sets or returns an  **MsoTriState**constant that represents whether the font for a dropped capital letter or WordArt text effect is italic. Read/write.|
| [FontName](texteffectformat.fontname-property-publisher.md)|Sets or returns a  **String** that represents the name of the font applied to a dropped capital letter or WordArt text effect. Read/write.|
| [FontSize](texteffectformat.fontsize-property-publisher.md)|Returns or sets a  **Variant** that represents the font size for the specified WordArt, in points. Read/write.|
| [KernedPairs](texteffectformat.kernedpairs-property-publisher.md)|Sets or returns an  **MsoTriState** constant that indicates whether character pairs in a WordArt object have been kerned. Read/write.|
| [NormalizedHeight](texteffectformat.normalizedheight-property-publisher.md)|Specifies whether all characters (both uppercase and lowercase) in the specified WordArt are the same height. Read/write.|
| [Parent](texteffectformat.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. For example, for a  **[TextFrame](textframe-object-publisher.md)** object, returns a **[Shape](shape-object-publisher.md)** object representing the parent shape of the text frame. Read-only.|
| [PresetShape](texteffectformat.presetshape-property-publisher.md)|Returns or sets an  **MsoPresetTextEffectShape** constant that represents the shape of the specified WordArt. Read/write.|
| [PresetTextEffect](texteffectformat.presettexteffect-property-publisher.md)|Returns or sets an  **MsoPresetTextEffect** constant that represents the style of the specified WordArt. The values for this property correspond to the formats in the **WordArt Gallery** dialog box, numbered from left to right, top to bottom. Read/write.|
| [PresetWordArt](texteffectformat.presetwordart-property-publisher.md)|A  **[pbPresetWordArt](pbpresetwordart-enumeration-publisher.md)** constant that indicates the type of preset WordArt associated with the specified object. Read/write.|
| [RotatedChars](texteffectformat.rotatedchars-property-publisher.md)| **msoTrue** if characters in the specified WordArt are rotated 90 degrees relative to the WordArt's bounding shape. **msoFalse** if characters in the specified WordArt retain their original orientation relative to the bounding shape. Read/write.|
| [Text](texteffectformat.text-property-publisher.md)|Returns or sets a  **String** that represents the text in a text range or WordArt shape. Read/write.|
| [Tracking](texteffectformat.tracking-property-publisher.md)|Returns or sets a  **Variant** indicating the tracking value used to display space between the characters in the specified text range. Read/write.|
|Name|Description|
