# Table Members (Publisher)
Represents a single table.

## Methods



|**Name**|**Description**|
|:-----|:-----|
| [ApplyAutoFormat](table.applyautoformat-method-publisher.md)|Applies automatic built-in table formatting to a specified table.|
|Name|Description|

## Properties



|**Name**|**Description**|
|:-----|:-----|
| [Application](table.application-property-publisher.md)|Used without an object qualifier, this property returns an  **[Application](application-object-publisher.md)** object that represents the current instance of Publisher. Used with an object qualifier, this property returns an  **Application** object that represents the creator of the specified object. When used with an OLE Automation object, it returns the object's application.|
| [Cells](table.cells-property-publisher.md)|Returns a  **CellRange** object that represents a range of cells in a table.|
| [Columns](table.columns-property-publisher.md)|Returns a  **[Columns](columns-object-publisher.md)** collection that represents all the columns of the specified table.|
| [GrowToFitText](table.growtofittext-property-publisher.md)| **True** for cells in a table to increase vertically to fit text. Read/write.|
| [Parent](table.parent-property-publisher.md)|Returns an object that represents the parent object of the specified object. Read-only.|
| [Rows](table.rows-property-publisher.md)|Returns a  **[Rows](rows-object-publisher.md)** collection that represents all the table rows in a range, selection, or table.|
| [TableDirection](table.tabledirection-property-publisher.md)|Returns or sets a  **PbTableDirectionType** constant that represents whether text in a table is read from left to right or from right to left. Read/write.|
