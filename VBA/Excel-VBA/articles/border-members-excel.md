# Border Members (Excel)
Represents the border of an object.

Represents the border of an object.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](border-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Color](border-color-property-excel.md)|Returns or sets the primary color of the object, as shown in the table in the remarks section. Use the  **RGB** function to create a color value. Read/write **Variant** .|
|[ColorIndex](border-colorindex-property-excel.md)|Returns or sets a  **Variant** value that represents the color of the border.|
|[Creator](border-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[LineStyle](border-linestyle-property-excel.md)|Returns or sets the line style for the border. Read/write  **[XlLineStyle](xllinestyle-enumeration-excel.md)** , **xlGray25** , **xlGray50** , **xlGray75** , or **xlAutomatic** .|
|[Parent](border-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[ThemeColor](border-themecolor-property-excel.md)|Returns or sets the theme color in the applied color scheme that is associated with the specified object. Read/write  **Variant** .|
|[TintAndShade](border-tintandshade-property-excel.md)|Returns or sets a  **Single** that lightens or darkens a color.|
|[Weight](border-weight-property-excel.md)|Returns or sets a  **[XlBorderWeight](xlborderweight-enumeration-excel.md)** value that represents the weight of the border.|
