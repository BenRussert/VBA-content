# XlSlicerCacheType Enumeration (Excel)

Designates the type of slicer or slicer cache.


## Members



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlSlicer**|1|Slicer cache represents a Slicer.|
| **xlTimeline**|2|Slicer cache represents a Timeline.|
