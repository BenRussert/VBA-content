# ColorStops Members (Excel)
A collection of all the [ColorStop](colorstop-object-excel.md) objects for the specified series.

A collection of all the [ColorStop](colorstop-object-excel.md) objects for the specified series.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](colorstops-add-method-excel.md)|Adds a  **[ColorStop](colorstop-object-excel.md)** object to the specified collection.|
|[Clear](colorstops-clear-method-excel.md)|Clears the represented object.|
|[Item](colorstops-item-method-excel.md)|Returns a single object from the represented collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](colorstops-application-property-excel.md)|When used without an object qualifier, this property returns an  **Application** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Count](colorstops-count-property-excel.md)|Returns or sets the count of the represented object. Read-only|
|[Creator](colorstops-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long**|
|[Parent](colorstops-parent-property-excel.md)|Returns the parent object for the specified object. Read-only|
