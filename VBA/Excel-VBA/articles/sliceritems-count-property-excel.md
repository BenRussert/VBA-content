# SlicerItems.Count Property (Excel)

Returns the number of objects in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[SlicerItems](sliceritems-object-excel.md)** object.


### Return Value

 **Long**
