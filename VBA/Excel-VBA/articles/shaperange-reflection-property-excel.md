# ShapeRange.Reflection Property (Excel)

Returns a  **[ReflectionFormat](http://msdn.microsoft.com/library/reflectionformat-object-office%28Office.15%29.aspx)** object for a specified shape range that contains reflection formatting properties for the shape range. Read-only.


## Syntax

 _expression_ . **Reflection**

 _expression_ A variable that represents a **ShapeRange** object.
