# PivotAxis Members (Excel)
The  **PivotAxis** object is used for asymmetric drilling in a PivotTable.

The  **PivotAxis** object is used for asymmetric drilling in a PivotTable.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](pivotaxis-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Creator](pivotaxis-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Parent](pivotaxis-parent-property-excel.md)|Returns the parent object for the specified  **PivotAxis** object. Read-only.|
|[PivotLines](pivotaxis-pivotlines-property-excel.md)|Returns the PivotLines attached to the specified  **PivotAxis** object. Read-only.|
