# Model.ModelFormatGeneral Property (Excel)

Returns a [ModelFormatGeneral](modelformatgeneral-object-excel.md) object that represents formatting of type general in the data model. Read-only.


## Syntax

 _expression_ . **ModelFormatGeneral**

 _expression_ A variable that represents a **Model** object.
