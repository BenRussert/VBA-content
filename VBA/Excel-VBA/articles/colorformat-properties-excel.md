# ColorFormat Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](colorformat-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Brightness](colorformat-brightness-property-excel.md)|Returns or sets the luminosity of the specified object. Read/write|
|[Creator](colorformat-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[ObjectThemeColor](colorformat-objectthemecolor-property-excel.md)|Returns or sets a color that is mapped to the theme color scheme. Read/write  **[MsoThemeColorIndex](msothemecolorindex-enumeration-office.md)** .|
|[Parent](colorformat-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[RGB](colorformat-rgb-property-excel.md)|Returns or sets a  **Long** value that represents the red-green-blue value of the specified color.|
|[SchemeColor](colorformat-schemecolor-property-excel.md)|Returns or sets an  **Integer** value that represents the color of a **Color** object, as an index in the current color.|
|[TintAndShade](colorformat-tintandshade-property-excel.md)|Returns or sets a  **Single** that lightens or darkens a color.|
|[Type](colorformat-type-property-excel.md)|Returns a  **[MsoColorType](msocolortype-enumeration-office.md)** value that represents the color format type.|
