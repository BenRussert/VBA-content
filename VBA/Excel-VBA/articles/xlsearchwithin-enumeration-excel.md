# XlSearchWithin Enumeration (Excel)

Specifies the extent of the search for the range.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlWithinSheet**|1|Limit search to current sheet.|
| **xlWithinWorkbook**|2|Search whole workbook.|
