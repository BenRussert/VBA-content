# Worksheet.Type Property (Excel)

Returns an  **[XlSheetType](xlsheettype-enumeration-excel.md)** value that represents the worksheet type.


## Syntax

 _expression_ . **Type**

 _expression_ A variable that represents a **Worksheet** object.
