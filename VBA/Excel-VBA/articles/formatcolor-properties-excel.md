# FormatColor Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](formatcolor-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Color](formatcolor-color-property-excel.md)|Returns or sets the fill color associated with a threshold for a data bar or color scale conditional formatting rule.|
|[ColorIndex](formatcolor-colorindex-property-excel.md)|Returns or sets one of the constants of the  **[XlColorIndex](xlcolorindex-enumeration-excel.md)** enumeration, specifying if the fill color is expressed as an index value into the current color palette.|
|[Creator](formatcolor-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Parent](formatcolor-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[ThemeColor](formatcolor-themecolor-property-excel.md)|Returns or sets one of the constants of the  **[XlThemeColor](xlthemecolor-enumeration-excel.md)** enumeration, specifying the theme color used in a threshold of a data bar or color scale conditional format.|
|[TintAndShade](formatcolor-tintandshade-property-excel.md)|Returns or sets a  **Single** that lightens or darkens the fill color of a cell for a threshold of a data bar or color scale conditional formatting rule.|
