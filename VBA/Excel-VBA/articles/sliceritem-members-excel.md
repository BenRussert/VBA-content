# SlicerItem Members (Excel)
Represents an item in a slicer.

Represents an item in a slicer.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](sliceritem-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Caption](sliceritem-caption-property-excel.md)|Returns the label text for the slicer item. Read-only.|
|[Creator](sliceritem-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[HasData](sliceritem-hasdata-property-excel.md)|Returns whether the slicer item contains data that matches the current manual filter state. Read-only.|
|[Name](sliceritem-name-property-excel.md)|Returns the name of the slicer item. Read-only.|
|[Parent](sliceritem-parent-property-excel.md)|Returns the parent  **[SlicerCache](slicercache-object-excel.md)** object for the slicer item. Read-only.|
|[Selected](sliceritem-selected-property-excel.md)|Returns or sets whether the slicer item is selected. Read/write for slicers connected to non-OLAP data sources. Read-only for slicers connected to OLAP data sources.|
|[SourceName](sliceritem-sourcename-property-excel.md)|Returns the source name of the slicer item. Read-only.|
|[SourceNameStandard](sliceritem-sourcenamestandard-property-excel.md)|Returns the slicer item's source name in standard English (United States) format settings. Read-only|
|[Value](sliceritem-value-property-excel.md)|Returns the value of the slicer item. Read-only.|
