# SortField Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](sortfield-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Creator](sortfield-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[CustomOrder](sortfield-customorder-property-excel.md)|Specifies a custom order to sort the fields. Read/write  **Variant** .|
|[DataOption](sortfield-dataoption-property-excel.md)|Specifies how to sort text in the range specified in  **SortField** object. Read/write **[XlSortDataOption](xlsortdataoption-enumeration-excel.md)** .|
|[Key](sortfield-key-property-excel.md)|Specifies the range that is currently being sorted on. Read-only.|
|[Order](sortfield-order-property-excel.md)|Determines the sort order for the values specified in the key. Read/write.|
|[Parent](sortfield-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[Priority](sortfield-priority-property-excel.md)|Specifies the priority for the sort field. Read/write.|
|[SortOn](sortfield-sorton-property-excel.md)|Returns or sets what attribute of the cell to sort on . Read/write  **[XlSortOn](xlsorton-enumeration-excel.md)** .|
|[SortOnValue](sortfield-sortonvalue-property-excel.md)|Retun the value on which the sort is performed for the specified  **SortField** object. Read-only.|
