# NegativeBarFormat Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](negativebarformat-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[BorderColor](negativebarformat-bordercolor-property-excel.md)| Returns a **[FormatColor](formatcolor-object-excel.md)** object that you can use to specify the border color for negative data bars. Read-only|
|[BorderColorType](negativebarformat-bordercolortype-property-excel.md)| Specifies whether to use the same border color as positive data bars. Read/write|
|[Color](negativebarformat-color-property-excel.md)| Returns a **[FormatColor](formatcolor-object-excel.md)** object that you can use to specify the fill color for negative data bars. Read-only|
|[ColorType](negativebarformat-colortype-property-excel.md)| Specifies whether to use the same fill color as positive data bars. Read/write|
|[Creator](negativebarformat-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Parent](negativebarformat-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
