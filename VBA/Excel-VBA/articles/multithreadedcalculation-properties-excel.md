# MultiThreadedCalculation Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](multithreadedcalculation-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Creator](multithreadedcalculation-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Enabled](multithreadedcalculation-enabled-property-excel.md)|The  **Enabled** property allows **MultiThreadedCalculation** objects to be enabled or disabled at run time. Read/write.|
|[Parent](multithreadedcalculation-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[ThreadCount](multithreadedcalculation-threadcount-property-excel.md)|Gets the total count of the process threads that are a part of the specified  **MultiThreadedCalculation** object.|
|[ThreadMode](multithreadedcalculation-threadmode-property-excel.md)|Returns or sets the thread mode for the specified  **MultiThreadedCalculation** object. Read/write **XlThreadMode** .|
