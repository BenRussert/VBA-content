# PivotLine Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](pivotline-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Creator](pivotline-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[LineType](pivotline-linetype-property-excel.md)|Returns an  **[XlPivotLineType](xlpivotlinetype-enumeration-excel.md)** constant that indicates the type of PivotLine. Read-only.|
|[Parent](pivotline-parent-property-excel.md)|Returns the parent object for the specified  **PivotLine** object. Read-only.|
|[PivotLineCells](pivotline-pivotlinecells-property-excel.md)|Returns a collection of  **PivotCell** objects in a PivotLine. Read-only.|
|[PivotLineCellsFull](pivotline-pivotlinecellsfull-property-excel.md)|Retrieves all of the  **PivotLine** cells including those that are hidden in compact form. Read-only|
|[Position](pivotline-position-property-excel.md)|Returns or sets the position of the  **PivotLine** object. Read-only.|
