# SlicerCacheLevel Members (Excel)
Represents a level of a hierarchy in an OLAP data source being filtered by a slicer.

Represents a level of a hierarchy in an OLAP data source being filtered by a slicer.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](slicercachelevel-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Count](slicercachelevel-count-property-excel.md)|Returns the number of  **[SlicerItem](sliceritem-object-excel.md)** objects in the specified **[SlicerCacheLevel](slicercachelevel-object-excel.md)** object. Read-only.|
|[Creator](slicercachelevel-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[CrossFilterType](slicercachelevel-crossfiltertype-property-excel.md)|Returns or sets whether a slicer is participating in cross filtering with other slicers that share the same slicer cache, and how cross filtering is displayed. Read/write|
|[Name](slicercachelevel-name-property-excel.md)|Returns the MDX unique name of OLAP hierarchy level represented by the object.|
|[Ordinal](slicercachelevel-ordinal-property-excel.md)|Returns the one-based ordinal value of the hierarchy level represented by the specified  **[SlicerCacheLevel](slicercachelevel-object-excel.md)** object. Read-only|
|[Parent](slicercachelevel-parent-property-excel.md)|Returns the  **[SlicerCache](slicercache-object-excel.md)** object for the specified **[SlicerCacheLevel](slicercachelevel-object-excel.md)** object. Read-only.|
|[SlicerItems](slicercachelevel-sliceritems-property-excel.md)|Returns a  **[SlicerItems](sliceritems-object-excel.md)** collection that contains the collection of all slicer items for the specified level. Read-only|
|[SortItems](slicercachelevel-sortitems-property-excel.md)|Returns or sets the sort order of the items in the slicer. Read/write.|
|[VisibleSlicerItemsList](slicercachelevel-visiblesliceritemslist-property-excel.md)|Returns the list of slicer items that are currently included in the slicer filter. Read-only|
