# OLEDBErrors Members (Excel)
A collection of  **[OLEDBError](oledberror-object-excel.md)** objects.

A collection of  **[OLEDBError](oledberror-object-excel.md)** objects.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Item](oledberrors-item-method-excel.md)|Returns a single object from a collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](oledberrors-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Count](oledberrors-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](oledberrors-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Parent](oledberrors-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
