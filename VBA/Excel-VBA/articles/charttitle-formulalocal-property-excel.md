# ChartTitle.FormulaLocal Property (Excel)

Gets or sets a  **String** value that represents the formula of the object using A1-style notation, in the language of the user. Read/write.


## Syntax

 _expression_ . **FormulaLocal**

 _expression_ A variable that represents a **ChartTitle** object.


### Return Value

String


## Remarks

For more information about A1-style notation, see [How to: Refer to Cells and Ranges by Using A1 Notation](http://msdn.microsoft.com/library/refer-to-cells-and-ranges-by-using-a1-notation%28Office.15%29.aspx).
