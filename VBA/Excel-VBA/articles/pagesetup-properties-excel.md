# PageSetup Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[AlignMarginsHeaderFooter](pagesetup-alignmarginsheaderfooter-property-excel.md)|Returns  **True** for Excel to align the header and the footer with the margins set in the page setup options. Read/write **Boolean** .|
|[Application](pagesetup-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[BlackAndWhite](pagesetup-blackandwhite-property-excel.md)| **True** if elements of the document will be printed in black and white. Read/write **Boolean** .|
|[BottomMargin](pagesetup-bottommargin-property-excel.md)|Returns or sets the size of the bottom margin, in points. Read/write  **Double** .|
|[CenterFooter](pagesetup-centerfooter-property-excel.md)|Center aligns the footer information in the  **PageSetup** object. Read/write **String** .|
|[CenterFooterPicture](pagesetup-centerfooterpicture-property-excel.md)|Returns a  **[Graphic](graphic-object-excel.md)** object that represents the picture for the center section of the footer. Used to set attributes about the picture.|
|[CenterHeader](pagesetup-centerheader-property-excel.md)|Center aligns the header information in the  **PageSetup** object. Read/write **String** .|
|[CenterHeaderPicture](pagesetup-centerheaderpicture-property-excel.md)|Returns a  **[Graphic](graphic-object-excel.md)** object that represents the picture for the center section of the header. Used to set attributes about the picture.|
|[CenterHorizontally](pagesetup-centerhorizontally-property-excel.md)| **True** if the sheet is centered horizontally on the page when it's printed. Read/write **Boolean** .|
|[CenterVertically](pagesetup-centervertically-property-excel.md)| **True** if the sheet is centered vertically on the page when it's printed. Read/write **Boolean** .|
|[Creator](pagesetup-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[DifferentFirstPageHeaderFooter](pagesetup-differentfirstpageheaderfooter-property-excel.md)| **True** if a different header or footer is used on the first page. Read/write **Boolean** .|
|[Draft](pagesetup-draft-property-excel.md)| **True** if the sheet will be printed without graphics. Read/write **Boolean** .|
|[EvenPage](pagesetup-evenpage-property-excel.md)|Returns or sets the alignment of text on the even page of a workbook or section.|
|[FirstPage](pagesetup-firstpage-property-excel.md)|Returns or sets the alignment of text on the first page of a workbook or section.|
|[FirstPageNumber](pagesetup-firstpagenumber-property-excel.md)|Returns or sets the first page number that will be used when this sheet is printed. If  **xlAutomatic** , Microsoft Excel chooses the first page number. The default is **xlAutomatic** . Read/write **Long** .|
|[FitToPagesTall](pagesetup-fittopagestall-property-excel.md)|Returns or sets the number of pages tall the worksheet will be scaled to when it's printed. Applies only to worksheets. Read/write  **Variant** .|
|[FitToPagesWide](pagesetup-fittopageswide-property-excel.md)|Returns or sets the number of pages wide the worksheet will be scaled to when it's printed. Applies only to worksheets. Read/write  **Variant** .|
|[FooterMargin](pagesetup-footermargin-property-excel.md)|Returns or sets the distance from the bottom of the page to the footer, in points. Read/write  **Double** .|
|[HeaderMargin](pagesetup-headermargin-property-excel.md)|Returns or sets the distance from the top of the page to the header, in points. Read/write  **Double** .|
|[LeftFooter](pagesetup-leftfooter-property-excel.md)|Returns or sets the alignment of text on the left footer of a workbook or section.|
|[LeftFooterPicture](pagesetup-leftfooterpicture-property-excel.md)|Returns a  **[Graphic](graphic-object-excel.md)** object that represents the picture for the left section of the footer. Used to set attributes about the picture.|
|[LeftHeader](pagesetup-leftheader-property-excel.md)|Returns or sets the alignment of text on the left header of a workbook or section.|
|[LeftHeaderPicture](pagesetup-leftheaderpicture-property-excel.md)|Returns a  **[Graphic](graphic-object-excel.md)** object that represents the picture for the left section of the header. Used to set attributes about the picture.|
|[LeftMargin](pagesetup-leftmargin-property-excel.md)|Returns or sets the size of the left margin, in points. Read/write  **Double** .|
|[OddAndEvenPagesHeaderFooter](pagesetup-oddandevenpagesheaderfooter-property-excel.md)| **True** if the specified **PageSetup** object has different headers and footers for odd-numbered and even-numbered pages. Read/write **Boolean** .|
|[Order](pagesetup-order-property-excel.md)|Returns or sets a  **[XlOrder](xlorder-enumeration-excel.md)** value that represents the order that Microsoft Excel uses to number pages when printing a large worksheet.|
|[Orientation](pagesetup-orientation-property-excel.md)|Returns or sets a  **[XlPageOrientation](xlpageorientation-enumeration-excel.md)** value that represents the portrait or landscape printing mode.|
|[Pages](pagesetup-pages-property-excel.md)|Returns or sets the count or item number of the pages in  **Pages** collection.|
|[PaperSize](pagesetup-papersize-property-excel.md)|Returns or sets the size of the paper. Read/write [XlPaperSize](xlpapersize-enumeration-excel.md).|
|[Parent](pagesetup-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[PrintArea](pagesetup-printarea-property-excel.md)|Returns or sets the range to be printed, as a string using A1-style references in the language of the macro. Read/write  **String** .|
|[PrintComments](pagesetup-printcomments-property-excel.md)|Returns or sets the way comments are printed with the sheet. Read/write  **[XlPrintLocation](xlprintlocation-enumeration-excel.md)** .|
|[PrintErrors](pagesetup-printerrors-property-excel.md)|Sets or returns an  **[XlPrintErrors](xlprinterrors-enumeration-excel.md)** contstant specifying the type of print error displayed. This feature allows users to suppress the display of error values when printing a worksheet. Read/write .|
|[PrintGridlines](pagesetup-printgridlines-property-excel.md)| **True** if cell gridlines are printed on the page. Applies only to worksheets. Read/write **Boolean** .|
|[PrintHeadings](pagesetup-printheadings-property-excel.md)| **True** if row and column headings are printed with this page. Applies only to worksheets. Read/write **Boolean** .|
|[PrintNotes](pagesetup-printnotes-property-excel.md)| **True** if cell notes are printed as end notes with the sheet. Applies only to worksheets. Read/write **Boolean** .|
|[PrintQuality](pagesetup-printquality-property-excel.md)|Returns or sets the print quality. Read/write  **Variant** .|
|[PrintTitleColumns](pagesetup-printtitlecolumns-property-excel.md)|Returns or sets the columns that contain the cells to be repeated on the left side of each page, as a string in A1-style notation in the language of the macro. Read/write  **String** .|
|[PrintTitleRows](pagesetup-printtitlerows-property-excel.md)|Returns or sets the rows that contain the cells to be repeated at the top of each page, as a string in A1-style notation in the language of the macro. Read/write  **String** .|
|[RightFooter](pagesetup-rightfooter-property-excel.md)|Returns or sets the distance (in points) between the right edge of the page and the right boundary of the footer. Read/write  **String** .|
|[RightFooterPicture](pagesetup-rightfooterpicture-property-excel.md)|Returns a  **[Graphic](graphic-object-excel.md)** object that represents the picture for the right section of the footer. Used to set attributes of the picture.|
|[RightHeader](pagesetup-rightheader-property-excel.md)|Returns or sets the right part of the header. Read/write  **String** .|
|[RightHeaderPicture](pagesetup-rightheaderpicture-property-excel.md)|Specifies the graphic image that should be displayed in the right header. Read-only.|
|[RightMargin](pagesetup-rightmargin-property-excel.md)|Returns or sets the size of the right margin, in points. Read/write  **Double** .|
|[ScaleWithDocHeaderFooter](pagesetup-scalewithdocheaderfooter-property-excel.md)|Returns or sets if the header and footer should be scaled with the document when the size of the document changes. Read/write  **Boolean** .|
|[TopMargin](pagesetup-topmargin-property-excel.md)|Returns or sets the size of the top margin, in points. Read/write  **Double** .|
|[Zoom](pagesetup-zoom-property-excel.md)|Returns or sets a  **Variant** value that represents a percentage (between 10 and 400 percent) by which Microsoft Excel will scale the worksheet for printing.|
