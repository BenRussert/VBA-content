# Top10 Methods (Excel)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](top10-delete-method-excel.md)|Deletes the specified conditional formatting rule object.|
|[ModifyAppliesToRange](top10-modifyappliestorange-method-excel.md)|Sets the cell range to which this formatting rule applies.|
|[SetFirstPriority](top10-setfirstpriority-method-excel.md)|Sets the priority value for this conditional formatting rule to "1" so that it will be evaluated before all other rules on the worksheet.|
|[SetLastPriority](top10-setlastpriority-method-excel.md)|Sets the evaluation order for this conditional formatting rule so it is evaluated after all other rules on the worksheet.|
