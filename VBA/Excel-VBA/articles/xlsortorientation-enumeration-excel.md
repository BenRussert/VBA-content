# XlSortOrientation Enumeration (Excel)

Specifies the sort orientation.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlSortColumns**|1|Sorts by column.|
| **xlSortRows**|2|Sorts by row. This is the default value.|
