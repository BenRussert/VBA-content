# Style Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[AddIndent](style-addindent-property-excel.md)|Returns or sets a  **Boolean** value that indicates if text is automatically indented when the text alignment in a cell is set to equal distribution (either horizontally or vertically.)|
|[Application](style-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Borders](style-borders-property-excel.md)|Returns a  **[Borders](borders-object-excel.md)** collection that represents the borders of a style or a range of cells (including a range defined as part of a conditional format).|
|[BuiltIn](style-builtin-property-excel.md)| **True** if the style is a built-in style. Read-only **Boolean** .|
|[Creator](style-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Font](style-font-property-excel.md)|Returns a  **[Font](font-object-excel.md)** object that represents the font of the specified object.|
|[FormulaHidden](style-formulahidden-property-excel.md)|Returns or sets a  **Boolean** value that indicates if the formula will be hidden when the worksheet is protected.|
|[HorizontalAlignment](style-horizontalalignment-property-excel.md)|Returns or sets a  **[XlHAlign](xlhalign-enumeration-excel.md)** value that represents the horizontal alignment for the specified object.|
|[IncludeAlignment](style-includealignment-property-excel.md)| **True** if the style includes the **AddIndent** , **HorizontalAlignment** , **VerticalAlignment** , **WrapText** , **IndentLevel** , and **Orientation** properties. Read/write **Boolean** .|
|[IncludeBorder](style-includeborder-property-excel.md)| **True** if the style includes the **Color** , **ColorIndex** , **LineStyle** , and **Weight** border properties. Read/write **Boolean** .|
|[IncludeFont](style-includefont-property-excel.md)| **True** if the style includes the **[Background](font-background-property-excel.md)** , **[Bold](texteffectformat-fontbold-property-excel.md)** , **[Color](font-color-property-excel.md)** , **[ColorIndex](font-colorindex-property-excel.md)** , **[FontStyle](font-fontstyle-property-excel.md)** , **[Italic](texteffectformat-fontitalic-property-excel.md)** , **[Name](texteffectformat-fontname-property-excel.md)** , **[Size](texteffectformat-fontsize-property-excel.md)** , **[Strikethrough](font-strikethrough-property-excel.md)** , **[Subscript](font-subscript-property-excel.md)** , **[Superscript](font-superscript-property-excel.md)** , and **[Underline](font-underline-property-excel.md)** font properties. Read/write **Boolean** .|
|[IncludeNumber](style-includenumber-property-excel.md)| **True** if the style includes the **NumberFormat** property. Read/write **Boolean**|
|[IncludePatterns](style-includepatterns-property-excel.md)| **True** if the style includes the **Color** , **ColorIndex** , **InvertIfNegative** , **Pattern** , **PatternColor** , and **PatternColorIndex** interior properties. Read/write **Boolean** .|
|[IncludeProtection](style-includeprotection-property-excel.md)| **True** if the style includes the **FormulaHidden** and **Locked** protection properties. Read/write **Boolean** .|
|[IndentLevel](style-indentlevel-property-excel.md)|Returns or sets a  **Long** value that represents the indent level for the style.|
|[Interior](style-interior-property-excel.md)|Returns an  **[Interior](interior-object-excel.md)** object that represents the interior of the specified object.|
|[Locked](style-locked-property-excel.md)|Returns or sets a  **Boolean** value that indicates if the object is locked.|
|[MergeCells](style-mergecells-property-excel.md)| **True** if the style contains merged cells. Read/write **Variant** .|
|[Name](style-name-property-excel.md)|Returns a  **String** value that represents the name of the object.|
|[NameLocal](style-namelocal-property-excel.md)|Returns or sets the name of the object, in the language of the user. Read-only  **String** .|
|[NumberFormat](style-numberformat-property-excel.md)|Returns or sets a  **String** value that represents the format code for the object.|
|[NumberFormatLocal](style-numberformatlocal-property-excel.md)|Returns or sets a  **String** value that represents the format code for the object as a string in the language of the user.|
|[Orientation](style-orientation-property-excel.md)|Returns or sets a  **[XlOrientation](xlorientation-enumeration-excel.md)** value that represents the text orientation.|
|[Parent](style-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[ReadingOrder](style-readingorder-property-excel.md)|Returns or sets the reading order for the specified object. Can be one of the following constants:  **xlRTL** (right-to-left), **xlLTR** (left-to-right), or **xlContext** . Read/write **Long** .|
|[ShrinkToFit](style-shrinktofit-property-excel.md)|Returns or sets a  **Boolean** value that indicates if text automatically shrinks to fit in the available column width.|
|[Value](style-value-property-excel.md)|Returns a  **String** value that represents the name of the specified style.|
|[VerticalAlignment](style-verticalalignment-property-excel.md)|Returns or sets a  **[XlVAlign](xlvalign-enumeration-excel.md)** value that represents the vertical alignment of the specified object.|
|[WrapText](style-wraptext-property-excel.md)|Returns or sets a  **Boolean** value that indicates if Microsoft Excel wraps the text in the object.|
