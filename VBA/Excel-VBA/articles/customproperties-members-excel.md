# CustomProperties Members (Excel)
A collection of  **CustomProperty** objects that represent additional information. The information can be used as metadata for XML.

A collection of  **CustomProperty** objects that represent additional information. The information can be used as metadata for XML.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](customproperties-add-method-excel.md)|Adds custom property information.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](customproperties-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Count](customproperties-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](customproperties-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](customproperties-item-property-excel.md)|Returns a single object from a collection.|
|[Parent](customproperties-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
