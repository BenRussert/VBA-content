# DataLabel Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](datalabel-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[AutoText](datalabel-autotext-property-excel.md)| **True** if the object automatically generates appropriate text based on context. Read/write **Boolean** .|
|[Caption](datalabel-caption-property-excel.md)|Returns or sets a  **String** value that represents the data label text.|
|[Characters](datalabel-characters-property-excel.md)|Returns a  **[Characters](characters-object-excel.md)** object that represents a range of characters within the object text. You can use the **Characters** object to format characters within a text string.|
|[Creator](datalabel-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Format](datalabel-format-property-excel.md)|Returns the  **[ChartFormat](chartformat-object-excel.md)** object. Read-only.|
|[Formula](datalabel-formula-property-excel.md)|Gets or sets a  **String** value that represents the formula of the object using A1-style notation, in English. Read/write.|
|[FormulaLocal](datalabel-formulalocal-property-excel.md)|Gets or sets a  **String** value that represents the formula of the object using A1-style notation, in the language of the user. Read/write.|
|[FormulaR1C1](datalabel-formular1c1-property-excel.md)|Gets or sets a  **String** value that represents the formula of the object using R1C1-style notation, in English. Read/write.|
|[FormulaR1C1Local](datalabel-formular1c1local-property-excel.md)|Gets or sets a  **String** value that represents the formula of the object using R1C1-style notation, in the language of the user. Read/write.|
|[Height](datalabel-height-property-excel.md)|Returns the height, in points, of the object. Read/Write.|
|[HorizontalAlignment](datalabel-horizontalalignment-property-excel.md)|Returns or sets a  **Variant** value that represents the horizontal alignment for the specified object.|
|[Left](datalabel-left-property-excel.md)|Returns or sets a  **Double** value that represents the distance, in points, from the left edge of the object to the left edge of column A (on a worksheet) or the left edge of the chart area (on a chart).|
|[Name](datalabel-name-property-excel.md)|Returns a  **String** value that represents the name of the object.|
|[NumberFormat](datalabel-numberformat-property-excel.md)|Returns or sets a  **String** value that represents the format code for the object.|
|[NumberFormatLinked](datalabel-numberformatlinked-property-excel.md)| **True** if the number format is linked to the cells (so that the number format changes in the labels when it changes in the cells). Read/write **Boolean** .|
|[NumberFormatLocal](datalabel-numberformatlocal-property-excel.md)|Returns or sets a  **Variant** value that represents the format code for the object as a string in the language of the user.|
|[Orientation](datalabel-orientation-property-excel.md)|Returns or sets a  **Variant** value that represents the text orientation.|
|[Parent](datalabel-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[Position](datalabel-position-property-excel.md)|Returns or sets a  **[XlDataLabelPosition](xldatalabelposition-enumeration-excel.md)** value that represents the position of the data label.|
|[ReadingOrder](datalabel-readingorder-property-excel.md)|Returns or sets the reading order for the specified object. Can be one of the following constants:  **xlRTL** (right-to-left), **xlLTR** (left-to-right), or **xlContext** . Read/write **Long** .|
|[Separator](datalabel-separator-property-excel.md)|Sets or returns a  **Variant** representing the separator used for the data labels on a chart. Read/write.|
|[Shadow](datalabel-shadow-property-excel.md)|Returns or sets a  **Boolean** value that determines if the object has a shadow.|
|[ShowBubbleSize](datalabel-showbubblesize-property-excel.md)| **True** to show the bubble size for the data labels on a chart. **False** to hide. Read/write **Boolean** .|
|[ShowCategoryName](datalabel-showcategoryname-property-excel.md)| **True** to display the category name for the data labels on a chart. **False** to hide. Read/write **Boolean** .|
|[ShowLegendKey](datalabel-showlegendkey-property-excel.md)| **True** if the data label legend key is visible. Read/write **Boolean** .|
|[ShowPercentage](datalabel-showpercentage-property-excel.md)| **True** to display the percentage value for the data labels on a chart. **False** to hide. Read/write **Boolean** .|
|[ShowRange](datalabel-showrange-property-excel.md)|Used to toggle the  **Range** field if it exists on the data label range. **Boolean** Read/Write|
|[ShowSeriesName](datalabel-showseriesname-property-excel.md)|Returns or sets a  **Boolean** to indicate the series name display behavior for the data labels on a chart. **True** to show the series name. **False** to hide. Read/write.|
|[ShowValue](datalabel-showvalue-property-excel.md)|Returns or sets a  **Boolean** corresponding to a specified chart's data label values display behavior. **True** displays the values. **False** to hide. Read/write.|
|[Text](datalabel-text-property-excel.md)|Returns or sets the text for the specified object. Read/write  **String** .|
|[Top](datalabel-top-property-excel.md)|Returns or sets a  **Double** value that represents the distance, in points, from the top edge of the object to the top of row 1 (on a worksheet) or the top of the chart area (on a chart).|
|[VerticalAlignment](datalabel-verticalalignment-property-excel.md)|Returns or sets a  **Variant** value that represents the vertical alignment of the specified object.|
|[Width](datalabel-width-property-excel.md)|Returns the width, in points, of the object. Read-only.|
