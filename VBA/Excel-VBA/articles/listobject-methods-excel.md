# ListObject Methods (Excel)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](listobject-delete-method-excel.md)|Deletes the  **[ListObject](listobject-object-excel.md)** object and clears the cell data from the worksheet.|
|[ExportToVisio](listobject-exporttovisio-method-excel.md)|Exports a  **ListObject** object to Visio.|
|[Publish](listobject-publish-method-excel.md)|Publishes the  **[ListObject](listobject-object-excel.md)** object to a server that is running Microsoft SharePoint Foundation.|
|[Refresh](listobject-refresh-method-excel.md)|Retrieves the current data and schema for the list from the server that is running Microsoft SharePoint Foundation. This method can be used only with lists that are linked to a SharePoint site. If the SharePoint site is not available, calling this method will return an error.|
|[Resize](listobject-resize-method-excel.md)|The  **Resize** method allows a **[ListObject](listobject-object-excel.md)** object to be resized over a new range. No cells are inserted or moved.|
|[Unlink](listobject-unlink-method-excel.md)|Removes the link to a Microsoft SharePoint Foundation site from a list. Returns  **Nothing** .|
|[Unlist](listobject-unlist-method-excel.md)|Removes the list functionality from a  **[ListObject](listobject-object-excel.md)** object. After you use this method, the range of cells that made up the the list will be a regular range of data.|
