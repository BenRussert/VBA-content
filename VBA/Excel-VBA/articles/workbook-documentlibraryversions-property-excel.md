# Workbook.DocumentLibraryVersions Property (Excel)

Returns a  **[DocumentLibraryVersions](http://msdn.microsoft.com/library/documentlibraryversions-object-office%28Office.15%29.aspx)** collection that represents the collection of versions of a shared workbook that has versioning enabled and that is stored in a document library on a server.


## Syntax

 _expression_ . **DocumentLibraryVersions**

 _expression_ A variable that represents a **Workbook** object.
