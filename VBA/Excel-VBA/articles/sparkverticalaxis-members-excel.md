# SparkVerticalAxis Members (Excel)
Represents the settings for the vertical axes of a group of sparklines.

Represents the settings for the vertical axes of a group of sparklines.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](sparkverticalaxis-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Creator](sparkverticalaxis-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[CustomMaxScaleValue](sparkverticalaxis-custommaxscalevalue-property-excel.md)|Returns or sets the custom maximum value for the vertical axis of a sparkline. Read/write|
|[CustomMinScaleValue](sparkverticalaxis-customminscalevalue-property-excel.md)|Returns or sets the custom minimum value for the vertical axis of a sparkline. Read/write|
|[MaxScaleType](sparkverticalaxis-maxscaletype-property-excel.md)|Returns or sets how the maximum value of the vertical axis of the sparkline is scaled relative to other sparklines in the group. Read/write|
|[MinScaleType](sparkverticalaxis-minscaletype-property-excel.md)|Returns or sets how the minimum value of the vertical axis of the sparkline is scaled relative to other sparklines in the group. Read/write|
|[Parent](sparkverticalaxis-parent-property-excel.md)|Returns the parent  **[SparklineGroup](sparklinegroup-object-excel.md)** object for the specified object. Read-only.|
