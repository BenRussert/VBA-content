# Phonetics Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Alignment](phonetics-alignment-property-excel.md)|Returns or sets a  **Long** value that represents the alignment for the specified phonetic text or tick label.|
|[Application](phonetics-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[CharacterType](phonetics-charactertype-property-excel.md)|Returns or sets the type of phonetic text in the specified cell. Read/write  **[XlPhoneticCharacterType](xlphoneticcharactertype-enumeration-excel.md)** .|
|[Count](phonetics-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](phonetics-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Font](phonetics-font-property-excel.md)|Returns a  **[Font](font-object-excel.md)** object that represents the font of the specified object.|
|[Item](phonetics-item-property-excel.md)|Returns a single object from a collection.|
|[Length](phonetics-length-property-excel.md)|Returns a  **Long** value that represents the number of characters of phonetic text from the position you've specified with the **[Start](phonetics-start-property-excel.md)** property.|
|[Parent](phonetics-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[Start](phonetics-start-property-excel.md)|Returns the position that represents the first character of a phonetic text string in the specified cell. Read-only  **Long** .|
|[Text](phonetics-text-property-excel.md)|Returns or sets the text for the specified object. Read/write  **String** .|
|[Visible](phonetics-visible-property-excel.md)|Returns or sets a  **Boolean** value that determines whether the object is visible. Read/write.|
