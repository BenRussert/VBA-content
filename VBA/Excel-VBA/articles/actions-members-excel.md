# Actions Members (Excel)
A collection of all [Action](action-object-excel.md) objects for the specified series.

A collection of all [Action](action-object-excel.md) objects for the specified series.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](actions-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Count](actions-count-property-excel.md)|Returns the number of objects in the collection. Read-only  **Long** .|
|[Creator](actions-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](actions-item-property-excel.md)|Returns a  **Actions** object that represents a collection of actions in a workbook. Read-only.|
|[Parent](actions-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
