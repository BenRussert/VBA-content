# PivotTableChangeList Members (Excel)
Represents the list of changes a user has made to value cells in a PivotTable report based on an OLAP data source.

Represents the list of changes a user has made to value cells in a PivotTable report based on an OLAP data source.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](pivottablechangelist-add-method-excel.md)|Adds a  **[ValueChange](valuechange-object-excel.md)** object to the specified **[PivotTableChangeList](pivottablechangelist-object-excel.md)** collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](pivottablechangelist-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Count](pivottablechangelist-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](pivottablechangelist-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](pivottablechangelist-item-property-excel.md)|Returns a single  **[ValueChange](valuechange-object-excel.md)** object from the specified **[PivotTableChangeList](pivottablechangelist-object-excel.md)** collection.|
|[Parent](pivottablechangelist-parent-property-excel.md)|Returns the parent  **[PivotTable](pivottable-object-excel.md)** object for the specified **[PivotTableChangeList](pivottablechangelist-object-excel.md)** collection. Read-only.|
