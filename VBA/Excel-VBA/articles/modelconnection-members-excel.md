# ModelConnection Members (Excel)
Contain information for the new Model Connection Type introduced in Excel 2013 to interact with the integrated data model.

Contain information for the new Model Connection Type introduced in Excel 2013 to interact with the integrated data model.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[ADOConnection](modelconnection-adoconnection-property-excel.md)|The ADO Connection Object is used to create an open connection to a data source. Enables addins, such as Powerview, to create a direct connection to the engine and hence the data model.  **ADOConnection** object Read-only.|
|[Application](modelconnection-application-property-excel.md)|Returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. Read-only.|
|[CalculatedMembers](modelconnection-calculatedmembers-property-excel.md)|Returns a [CalculatedMembers](calculatedmembers-object-excel.md) object that represents the calculated members in the model connection.|
|[CommandText](modelconnection-commandtext-property-excel.md)|Returns or sets the command string for the specified data source.  **Variant** Read/Write|
|[CommandType](modelconnection-commandtype-property-excel.md)|Returns or sets one of the [XlCmdType Enumeration (Excel)](xlcmdtype-enumeration-excel.md) constants. Read/Write|
|[Creator](modelconnection-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Parent](modelconnection-parent-property-excel.md)|Returns an  **Object** that represents the parent object of the specified[ModelConnection](modelconnection-object-excel.md) object. Read-only.|
