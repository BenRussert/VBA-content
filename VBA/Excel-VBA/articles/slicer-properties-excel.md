# Slicer Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[ActiveItem](slicer-activeitem-property-excel.md)|Returns a  **[SlicerItem](sliceritem-object-excel.md)** object that represents the slicer button that is currently in focus for the specified slicer. Read-only.|
|[Application](slicer-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Caption](slicer-caption-property-excel.md)|Returns or sets the caption of the specified slicer. Read/write.|
|[ColumnWidth](slicer-columnwidth-property-excel.md)|Returns or sets the width, in points, of each column in the slicer. Read/write|
|[Creator](slicer-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[DisableMoveResizeUI](slicer-disablemoveresizeui-property-excel.md)|Returns or sets whether the specified slicer can be moved or resized by using the user interface. Read/write.|
|[DisplayHeader](slicer-displayheader-property-excel.md)|Returns or sets whether the header that displays the slicer  **[Caption](slicer-caption-property-excel.md)** is visible. Read/write.|
|[Height](slicer-height-property-excel.md)|Returns or sets the height of the specified slicer, in points. Read/write.|
|[Left](slicer-left-property-excel.md)|Returns or sets the horizontal position of the specified slicer, in points, relative to the upper-left corner of cell A1 on a worksheet. Read/write.|
|[Locked](slicer-locked-property-excel.md)|Returns or sets whether the specified slicer can be modified when the sheet that contains it is protected. Read/write.|
|[Name](slicer-name-property-excel.md)|Returns or sets the name of the specified slicer. Read/write.|
|[NumberOfColumns](slicer-numberofcolumns-property-excel.md)|Returns or sets the number of columns in the specified slicer. Read/write.|
|[Parent](slicer-parent-property-excel.md)|Returns the  **[Worksheet](worksheet-object-excel.md)** object that represents the sheet that contains the slicer. Read-only.|
|[RowHeight](slicer-rowheight-property-excel.md)|Returns or sets the height, in points, of each row in the specified slicer. Read/write.|
|[Shape](slicer-shape-property-excel.md)|Returns the  **[Shape](shape-object-excel.md)** object associated with the specified slicer. Read-only.|
|[SlicerCache](slicer-slicercache-property-excel.md)|Returns the  **[SlicerCache](slicercache-object-excel.md)** object associated with the slicer. Read-only.|
|[SlicerCacheLevel](slicer-slicercachelevel-property-excel.md)|Returns the  **[SlicerCacheLevel](slicercachelevel-object-excel.md)** object associated with the slicer. Read-only.|
|[SlicerCacheType](slicer-slicercachetype-property-excel.md)|Returns the type of the slicer cache - slicer or timeline.  **XlSlicerCacheType** Read-only|
|[Style](slicer-style-property-excel.md)|Returns or sets the style currently applied to the specified slicer. Read/write.|
|[TimelineViewState](slicer-timelineviewstate-property-excel.md)|The timeline-specific state of the Slicer.  **TimelineViewState** Read-only|
|[Top](slicer-top-property-excel.md)|Returns or sets the vertical position of the specified slicer, in points, relative to the upper-left corner of cell A1 on a worksheet. Read/write.|
|[Width](slicer-width-property-excel.md)|Returns or sets the width of the specified slicer, in points. Read/write.|
