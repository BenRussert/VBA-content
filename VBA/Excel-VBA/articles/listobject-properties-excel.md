# ListObject Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Active](listobject-active-property-excel.md)| Returns a **Boolean** value indicating whether a **[ListObject](listobject-object-excel.md)** object in a worksheet is activeâ€”that is, whether the active cell is inside the range of the **ListObject** object. Read-only **Boolean** .|
|[AlternativeText](listobject-alternativetext-property-excel.md)|Returns or sets the descriptive (alternative) text string for the specified table. Read/write|
|[Application](listobject-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[AutoFilter](listobject-autofilter-property-excel.md)|Filters a  **table** using the AutoFilter feature. Read-only.|
|[Comment](listobject-comment-property-excel.md)|Returns or sets the comment associated with the list object. Read/write  **String** .|
|[Creator](listobject-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[DataBodyRange](listobject-databodyrange-property-excel.md)|Returns a  **[Range](range-object-excel.md)** object that represents the range of values, excluding the header row, in a table. Read-only.|
|[DisplayName](listobject-displayname-property-excel.md)|Returns or sets the display name for the specified  **ListObject** object. Read/write **String** .|
|[DisplayRightToLeft](listobject-displayrighttoleft-property-excel.md)| **True** if the specified **[ListObject](listobject-object-excel.md)** is displayed from right to left instead of from left to right. **False** if the object is displayed from left to right. Read-only **Boolean** .|
|[HeaderRowRange](listobject-headerrowrange-property-excel.md)|Returns a  **[Range](range-object-excel.md)** object that represents the range of the header row for a list. Read-only **Range** .|
|[InsertRowRange](listobject-insertrowrange-property-excel.md)| Returns a **Range** object representing the Insert row, if any, of a specified **[ListObject](listobject-object-excel.md)** object. Read-only **Range** .|
|[ListColumns](listobject-listcolumns-property-excel.md)|Returns a  **[ListColumns](listcolumns-object-excel.md)** collection that represents all the columns in a **[ListObject](listobject-object-excel.md)** object. Read-only.|
|[ListRows](listobject-listrows-property-excel.md)|Returns a  **[ListRows](listrows-object-excel.md)** object that represents all the rows of data in the **[ListObject](listobject-object-excel.md)** object. Read-only.|
|[Name](listobject-name-property-excel.md)|Returns or sets a  **String** value that represents the name of the **[ListObject](listobject-object-excel.md)** object.|
|[Parent](listobject-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[QueryTable](listobject-querytable-property-excel.md)|Returns the  **[QueryTable](querytable-object-excel.md)** object that provides a link for the **[ListObject](listobject-object-excel.md)** object to the list server. Read-only.|
|[Range](listobject-range-property-excel.md)|Returns a  **[Range](range-object-excel.md)** object that represents the range to which the specified list object in the above list applies.|
|[SharePointURL](listobject-sharepointurl-property-excel.md)| Returns a **String** representing the URL of the SharePoint list for a given **[ListObject](listobject-object-excel.md)** object. Read-only **String** .|
|[ShowAutoFilter](listobject-showautofilter-property-excel.md)| Returns **Boolean** to indicate whether the AutoFilter will be displayed. Read/write **Boolean** .|
|[ShowAutoFilterDropDown](listobject-showautofilterdropdown-property-excel.md)| **True** when the AutoFilter drop down for the **ListObject** object is displayed. **Boolean** Read/Write|
|[ShowHeaders](listobject-showheaders-property-excel.md)|Returns or sets if the header information should be displayed for the specified  **ListObject** object. Read/write **Boolean** .|
|[ShowTableStyleColumnStripes](listobject-showtablestylecolumnstripes-property-excel.md)|Returns or sets if the Column Stripes table style is used for the specified  **ListObject** object. Read/write **Boolean** .|
|[ShowTableStyleFirstColumn](listobject-showtablestylefirstcolumn-property-excel.md)|Returns or sets if the first column is formatted for the specified  **ListObject** object. Read/write **Boolean** .|
|[ShowTableStyleLastColumn](listobject-showtablestylelastcolumn-property-excel.md)|Returns or sets if the last column is displayed for the specified  **ListObject** object. Read/write **Boolean** .|
|[ShowTableStyleRowStripes](listobject-showtablestylerowstripes-property-excel.md)|Returns or sets if the Row Stripes table style is used for the specified  **ListObject** object. Read/write **Boolean** .|
|[ShowTotals](listobject-showtotals-property-excel.md)|Gets or sets a  **Boolean** to indicate whether the Total row is visible. Read/write **Boolean** .|
|[Slicers](listobject-slicers-property-excel.md)|Returns a list of the table slicers associated with a  **ListObject** . Read-only|
|[Sort](listobject-sort-property-excel.md)|Gets or sets the sort column or columns, and sort order for the  **ListObject** collection.|
|[SourceType](listobject-sourcetype-property-excel.md)|Returns a  **[XlListObjectSourceType](xllistobjectsourcetype-enumeration-excel.md)** value that represents the current source of the list.|
|[Summary](listobject-summary-property-excel.md)|Returns or sets the description associated with the alternative text string for the specified table. Read/write|
|[TableObject](listobject-tableobject-property-excel.md)|Returns a [TableObject Object (Excel)](tableobject-object-excel.md) object. Read-only|
|[TableStyle](listobject-tablestyle-property-excel.md)|Gets or sets the table style for the specified  **ListObject** object. Read/write **Variant** .|
|[TotalsRowRange](listobject-totalsrowrange-property-excel.md)| Returns a **[Range](range-object-excel.md)** representing the Total row, if any, from a specified **ListObject** object. Read-only.|
|[XmlMap](listobject-xmlmap-property-excel.md)|Returns an  **XmlMap** object that represents the schema map used for the specified table. Read-only.|
