# ModelFormatCurrency Properties (Excel)

## Properties



|[Application](modelformatcurrency-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Creator](modelformatcurrency-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[DecimalPlaces](modelformatcurrency-decimalplaces-property-excel.md)|Specifies the number of decimal places after the dot. Read/write  **Long**.|
|[Parent](modelformatcurrency-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[Symbol](modelformatcurrency-symbol-property-excel.md)|Specifies the symbol to use to represent the currency. Read/write  **String**.|
