# RecentFiles Members (Excel)
Represents the list of recently used files.

Represents the list of recently used files.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](recentfiles-add-method-excel.md)|Adds a file to the list of recently used files.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](recentfiles-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Count](recentfiles-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](recentfiles-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](recentfiles-item-property-excel.md)|Returns a single object from a collection.|
|[Maximum](recentfiles-maximum-property-excel.md)|Returns or sets the maximum number of files in the list of recently used files. Can be a value from 0 (zero) through 50. Read/write  **Long** .|
|[Parent](recentfiles-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
