# LinearGradient Members (Excel)
The  **LinearGradient** object transitions through a series of colors in a linear manner along a specific angle.

The  **LinearGradient** object transitions through a series of colors in a linear manner along a specific angle.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](lineargradient-application-property-excel.md)|When used without an object qualifier, this property returns an  **Application** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[ColorStops](lineargradient-colorstops-property-excel.md)|Returns the  **[ColorStops](colorstops-object-excel.md)** for the **LinearGradient** object. Read-only|
|[Creator](lineargradient-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long**|
|[Degree](lineargradient-degree-property-excel.md)|The angle of the linear gradient fill within a selection. Read/write|
|[Parent](lineargradient-parent-property-excel.md)|Returns the parent object for the specified object. Read-only|
