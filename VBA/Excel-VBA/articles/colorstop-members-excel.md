# ColorStop Members (Excel)
Represents the color stop point for a gradient fill in a range or selection.

Represents the color stop point for a gradient fill in a range or selection.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](colorstop-delete-method-excel.md)|Deletes the represented object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](colorstop-application-property-excel.md)|When used without an object qualifier, this property returns an  **Application** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Color](colorstop-color-property-excel.md)|Returns or sets the color of the represented object. Read/write|
|[Creator](colorstop-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Parent](colorstop-parent-property-excel.md)|Returns the parent object for the specified object. Read-only|
|[Position](colorstop-position-property-excel.md)|Returns or sets the position of the  **ColorStop** . Read/write|
|[ThemeColor](colorstop-themecolor-property-excel.md)|Returns or sets the theme color of the represented object. Read/write|
|[TintAndShade](colorstop-tintandshade-property-excel.md)|Returns or sets the tint and shade of the represented object. Read/write|
