# LegendKey Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](legendkey-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Creator](legendkey-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Format](legendkey-format-property-excel.md)|Returns the  **[ChartFormat](chartformat-object-excel.md)** object. Read-only.|
|[Height](legendkey-height-property-excel.md)|Returns a  **Double** value that represents the height, in points, of the object.|
|[InvertIfNegative](legendkey-invertifnegative-property-excel.md)| **True** if Microsoft Excel inverts the pattern in the item when it corresponds to a negative number. Read/write **Boolean** .|
|[Left](legendkey-left-property-excel.md)|Returns a  **Double** value that represents the distance, in points, from the left edge of the object to the left edge of the chart area.|
|[MarkerBackgroundColor](legendkey-markerbackgroundcolor-property-excel.md)|Sets the marker background color as an RGB value or returns the corresponding color index value. Applies only to line, scatter, and radar charts. Read/write  **Long** .|
|[MarkerBackgroundColorIndex](legendkey-markerbackgroundcolorindex-property-excel.md)|Returns or sets the marker background color as an index into the current color palette, or as one of the following  **[XlColorIndex](xlcolorindex-enumeration-excel.md)** constants: **xlColorIndexAutomatic** or **xlColorIndexNone** . Applies only to line, scatter, and radar charts. Read/write **Long** .|
|[MarkerForegroundColor](legendkey-markerforegroundcolor-property-excel.md)|Sets the marker foreground color as an RGB value or returns the corresponding color index value. Applies only to line, scatter, and radar charts. Read/write  **Long** .|
|[MarkerForegroundColorIndex](legendkey-markerforegroundcolorindex-property-excel.md)|Returns or sets the marker foreground color as an index into the current color palette, or as one of the following  **[XlColorIndex](xlcolorindex-enumeration-excel.md)** constants: **xlColorIndexAutomatic** or **xlColorIndexNone** . Applies only to line, scatter, and radar charts. Read/write **Long** .|
|[MarkerSize](legendkey-markersize-property-excel.md)|Returns or sets the data-marker size, in points. Can be a value from 2 through 72. Read/write  **Long** .|
|[MarkerStyle](legendkey-markerstyle-property-excel.md)|Returns or sets the marker style for a point or series in a line chart, scatter chart, or radar chart. Read/write  **[XlMarkerStyle](xlmarkerstyle-enumeration-excel.md)** .|
|[Parent](legendkey-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[PictureType](legendkey-picturetype-property-excel.md)|Returns or sets a  **[XlChartPictureType](xlchartpicturetype-enumeration-excel.md)** value that represents the way pictures are displayed on a legend key.|
|[PictureUnit2](legendkey-pictureunit2-property-excel.md)|Returns or sets the unit for each picture on the chart if the  **[PictureType ](legendkey-picturetype-property-excel.md)** property is set to **xlStackScale** (if not, this property is ignored). Read/write **Double** .|
|[Shadow](legendkey-shadow-property-excel.md)|Returns or sets a  **Boolean** value that determines if the object has a shadow.|
|[Smooth](legendkey-smooth-property-excel.md)| **True** if curve smoothing is turned on for the legend key. Read/write.|
|[Top](legendkey-top-property-excel.md)|Returns a  **Double** value that represents the distance, in points, from the top edge of the object to the top of row 1 (on a worksheet) or the top of the chart area (on a chart).|
|[Width](legendkey-width-property-excel.md)|Returns a  **Double** value that represents the width, in points, of the object.|
