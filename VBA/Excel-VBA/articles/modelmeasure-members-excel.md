# ModelMeasure Members (Excel)

Represents a single  **ModelMeasure** object in the[ModelMeasures](modelmeasures-object-excel.md) collection.


## Methods
<a name="methods"> </a>



|[Delete](modelmeasure-delete-method-excel.md)|Deletes the model measure from the data model.|

## Properties
<a name="properties"> </a>



|**Name**|**Description**|
|:-----|:-----|
|[Application](modelmeasure-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[AssociatedTable](modelmeasure-associatedtable-property-excel.md)|Specifies the table that contains the model measure, as displayed in the  **Field List** task pane. Read/write[ModelTable](modeltable-object-excel.md).|
|[Creator](modelmeasure-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Description](modelmeasure-description-property-excel.md)|The description of the model measure. Read/write  **String**.|
|[FormatInformation](modelmeasure-formatinformation-property-excel.md)|The format of the model measure. Read/write  **Variant**.|
|[Formula](modelmeasure-formula-property-excel.md)|The Data Analysis Expressions (DAX) formula of the model measure. Read/write  **String**.|
|[Name](modelmeasure-name-property-excel.md)|The name of the model measure. Read/write  **String**.|
|[Parent](modelmeasure-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
