# PivotFormula Members (Excel)
Represents a formula used to calculate results in a PivotTable report.

Represents a formula used to calculate results in a PivotTable report.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](pivotformula-delete-method-excel.md)|Deletes the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](pivotformula-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Creator](pivotformula-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Formula](pivotformula-formula-property-excel.md)|Returns or sets a  **String** value that represents the object's formula in A1-style notation and in the language of the macro.|
|[Index](pivotformula-index-property-excel.md)|Returns or sets a  **Long** value that represents the index number of the **[PivotFormula](pivotformula-object-excel.md)** object within the **[PivotFormulas](pivotformulas-object-excel.md)** collection.|
|[Parent](pivotformula-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[StandardFormula](pivotformula-standardformula-property-excel.md)|Returns or sets a  **String** specifying formulas with standard English (United States) formatting. Read/write.|
|[Value](pivotformula-value-property-excel.md)|Returns or sets a  **String** value that represents the name of the specified formula in the PivotTable formula.|
