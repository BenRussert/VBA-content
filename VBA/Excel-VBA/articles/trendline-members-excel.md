# Trendline Members (Excel)
Represents a trendline in a chart.

Represents a trendline in a chart.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[ClearFormats](trendline-clearformats-method-excel.md)|Clears the formatting of the object.|
|[Delete](trendline-delete-method-excel.md)|Deletes the object.|
|[Select](trendline-select-method-excel.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](trendline-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Backward2](trendline-backward2-property-excel.md)|Returns or sets the number of periods (or units on a scatter chart) that the trendline extends backward. Read/write  **Double** .|
|[Border](trendline-border-property-excel.md)|Returns a  **[Border](border-object-excel.md)** object that represents the border of the object.|
|[Creator](trendline-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[DataLabel](trendline-datalabel-property-excel.md)|Returns a  **[DataLabel](datalabel-object-excel.md)** object that represents the data label associated with the trendline. Read-only.|
|[DisplayEquation](trendline-displayequation-property-excel.md)| **True** if the equation for the trendline is displayed on the chart (in the same data label as the R-squared value). Setting this property to **True** automatically turns on data labels. Read/write **Boolean** .|
|[DisplayRSquared](trendline-displayrsquared-property-excel.md)| **True** if the R-squared value of the trendline is displayed on the chart (in the same data label as the equation). Setting this property to **True** automatically turns on data labels. Read/write **Boolean** .|
|[Format](trendline-format-property-excel.md)|Returns the  **[ChartFormat](chartformat-object-excel.md)** object. Read-only.|
|[Forward2](trendline-forward2-property-excel.md)|Returns or sets the number of periods (or units on a scatter chart) that the trendline extends forward. Read/write  **Double** .|
|[Index](trendline-index-property-excel.md)|Returns a  **Long** value that represents the index number of the object within the collection of similar objects.|
|[Intercept](trendline-intercept-property-excel.md)|Returns or sets the point where the trendline crosses the value axis. Read/write  **Double** .|
|[InterceptIsAuto](trendline-interceptisauto-property-excel.md)| **True** if the point where the trendline crosses the value axis is automatically determined by the regression. Read/write **Boolean** .|
|[Name](trendline-name-property-excel.md)|Returns or sets a  **String** value representing the name of the object.|
|[NameIsAuto](trendline-nameisauto-property-excel.md)| **True** if Microsoft Excel automatically determines the name of the trendline. Read/write **Boolean** .|
|[Order](trendline-order-property-excel.md)|Returns or sets a  **Long** value that represents the trendline order (an integer greater than 1) when the trendline type is **xlPolynomial** .|
|[Parent](trendline-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[Period](trendline-period-property-excel.md)|Returns or sets the period for the moving-average trendline. Can be a value from 2 through 255. Read/write  **Long** .|
|[Type](trendline-type-property-excel.md)|Returns or sets a  **[XlTrendlineType](xltrendlinetype-enumeration-excel.md)** value that represents the trendline type.|
