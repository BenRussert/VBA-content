# TableStyleElement Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](tablestyleelement-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object. Read-only.|
|[Borders](tablestyleelement-borders-property-excel.md)|Returns a  **[Borders](borders-object-excel.md)** collection that represents the borders of a table style element. Read-only.|
|[Creator](tablestyleelement-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Font](tablestyleelement-font-property-excel.md)|Returns a  **[Font](font-object-excel.md)** object that represents the font of the specified object. Read-only.|
|[HasFormat](tablestyleelement-hasformat-property-excel.md)|Returns whether a table style element has formatting applied to the specified element. Read-only  **Boolean** .|
|[Interior](tablestyleelement-interior-property-excel.md)|Returns an  **[Interior](interior-object-excel.md)** object that represents the interior of the specified object. Read-only.|
|[Parent](tablestyleelement-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[StripeSize](tablestyleelement-stripesize-property-excel.md)|Returns or sets the size of banding. Read/write  **Long** .|
