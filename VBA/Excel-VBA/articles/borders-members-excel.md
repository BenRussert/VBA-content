# Borders Members (Excel)
A collection of four  **[Border](border-object-excel.md)** objects that represent the four borders of a **[Range](range-object-excel.md)** or **[Style](style-object-excel.md)** object.

A collection of four  **[Border](border-object-excel.md)** objects that represent the four borders of a **[Range](range-object-excel.md)** or **[Style](style-object-excel.md)** object.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](borders-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Color](borders-color-property-excel.md)|Returns or sets the primary color of the object, as shown in the table in the remarks section. Use the  **RGB** function to create a color value. Read/write **Variant** .|
|[ColorIndex](borders-colorindex-property-excel.md)|Returns or sets a  **Variant** value that represents the color of all four borders.|
|[Count](borders-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](borders-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](borders-item-property-excel.md)|Returns a  **[Border](border-object-excel.md)** object that represents one of the borders of either a range of cells or a style.|
|[LineStyle](borders-linestyle-property-excel.md)|Returns or sets the line style for the border. Read/write  **[XlLineStyle](xllinestyle-enumeration-excel.md)** , **xlGray25** , **xlGray50** , **xlGray75** , or **xlAutomatic** .|
|[Parent](borders-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[ThemeColor](borders-themecolor-property-excel.md)|Returns or sets the theme color in the applied color scheme that is associated with the specified object. Read/write  **Variant** .|
|[TintAndShade](borders-tintandshade-property-excel.md)|Returns or sets a  **Single** that lightens or darkens a color.|
|[Value](borders-value-property-excel.md)|Synonym for  **[Borders.LineStyle](borders-linestyle-property-excel.md)** .|
|[Weight](borders-weight-property-excel.md)|Returns or sets a  **[XlBorderWeight](xlborderweight-enumeration-excel.md)** value that represents the weight of the border.|
