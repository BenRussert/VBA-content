# DataTable Members (Excel)
Represents a chart data table.

Represents a chart data table.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](datatable-delete-method-excel.md)|Deletes the object.|
|[Select](datatable-select-method-excel.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](datatable-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Border](datatable-border-property-excel.md)|Returns a  **[Border](border-object-excel.md)** object that represents the border of the object.|
|[Creator](datatable-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Font](datatable-font-property-excel.md)|Returns a  **[Font](font-object-excel.md)** object that represents the font of the specified object.|
|[Format](datatable-format-property-excel.md)|Returns the  **[ChartFormat](chartformat-object-excel.md)** object. Read-only.|
|[HasBorderHorizontal](datatable-hasborderhorizontal-property-excel.md)| **True** if the chart data table has horizontal cell borders. Read/write **Boolean** .|
|[HasBorderOutline](datatable-hasborderoutline-property-excel.md)| **True** if the chart data table has outline borders. Read/write **Boolean** .|
|[HasBorderVertical](datatable-hasbordervertical-property-excel.md)| **True** if the chart data table has vertical cell borders. Read/write **Boolean** .|
|[Parent](datatable-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[ShowLegendKey](datatable-showlegendkey-property-excel.md)| **True** if the data label legend key is visible. Read/write **Boolean** .|
