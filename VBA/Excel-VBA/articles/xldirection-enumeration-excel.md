# XlDirection Enumeration (Excel)

Specifies the direction in which to move.



|**Name**|**Value**|**Description**|
|:-----|:-----|:-----|
| **xlDown**|-4121|Down.|
| **xlToLeft**|-4159|To left.|
| **xlToRight**|-4161|To right.|
| **xlUp**|-4162|Up.|
