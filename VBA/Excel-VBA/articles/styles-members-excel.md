# Styles Members (Excel)
A collection of all the  **[Style](style-object-excel.md)** objects in the specified or active workbook.

A collection of all the  **[Style](style-object-excel.md)** objects in the specified or active workbook.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](styles-add-method-excel.md)|Creates a new style and adds it to the list of styles that are available for the current workbook.|
|[Merge](styles-merge-method-excel.md)|Merges the styles from another workbook into the  **[Styles](styles-object-excel.md)** collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](styles-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Count](styles-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](styles-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](styles-item-property-excel.md)|Returns a single object from a collection.|
|[Parent](styles-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
