# Name Properties (Excel)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](name-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Category](name-category-property-excel.md)|Returns or sets the category for the specified name in the language of the macro. The name must refer to a custom function or command. Read/write  **String** .|
|[CategoryLocal](name-categorylocal-property-excel.md)|Returns or sets the category for the specified name, in the language of the user, if the name refers to a custom function or command. Read/write  **String** .|
|[Comment](name-comment-property-excel.md)|Returns or sets the comment associated with the name. Read/write  **String** .|
|[Creator](name-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Index](name-index-property-excel.md)|Returns a  **Long** value that represents the index number of the object within the collection of similar objects.|
|[MacroType](name-macrotype-property-excel.md)|Returns or sets what the name refers to. Read/write  **[XlXLMMacroType](xlxlmmacrotype-enumeration-excel.md)** .|
|[Name](name-name-property-excel.md)|Returns or sets a  **String** value representing the name of the object.|
|[NameLocal](name-namelocal-property-excel.md)|Returns or sets the name of the object, in the language of the user. Read/write  **String** .|
|[Parent](name-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
|[RefersTo](name-refersto-property-excel.md)|Returns or sets the formula that the name is defined to refer to, in the language of the macro and in A1-style notation, beginning with an equal sign. Read/write  **String** .|
|[RefersToLocal](name-referstolocal-property-excel.md)|Returns or sets the formula that the name refers to. The formula is in the language of the user, and it's in A1-style notation, beginning with an equal sign. Read/write  **String** .|
|[RefersToR1C1](name-referstor1c1-property-excel.md)|Returns or sets the formula that the name refers to. The formula is in the language of the macro, and it's in R1C1-style notation, beginning with an equal sign. Read/write  **String** .|
|[RefersToR1C1Local](name-referstor1c1local-property-excel.md)|Returns or sets the formula that the name refers to. This formula is in the language of the user, and it's in R1C1-style notation, beginning with an equal sign. Read/write  **String** .|
|[RefersToRange](name-referstorange-property-excel.md)|Returns the  **[Range](range-object-excel.md)** object referred to by a **Name** object. Read-only.|
|[ShortcutKey](name-shortcutkey-property-excel.md)|Returns or sets the shortcut key for a name defined as a custom Microsoft Excel 4.0 macro command. Read/write  **String** .|
|[ValidWorkbookParameter](name-validworkbookparameter-property-excel.md)|Returns  **True** if the specified **Name** object is a valid workbook parameter. Read-only **Boolen** .|
|[Value](name-value-property-excel.md)|Returns or sets a  **String** value that represents the formula that the name is defined to refer to.|
|[Visible](name-visible-property-excel.md)|Returns or sets a  **Boolean** value that determines whether the object is visible. Read/write.|
|[WorkbookParameter](name-workbookparameter-property-excel.md)|Returns or sets the specified  **Name** object as a workbook parameter. Read/write|
