# HPageBreaks Members (Excel)
The collection of horizontal page breaks within the print area.

The collection of horizontal page breaks within the print area.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](hpagebreaks-add-method-excel.md)|Adds a horizontal page break.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](hpagebreaks-application-property-excel.md)|When used without an object qualifier, this property returns an  **[Application](application-object-excel.md)** object that represents the Microsoft Excel application. When used with an object qualifier, this property returns an **Application** object that represents the creator of the specified object (you can use this property with an OLE Automation object to return the application of that object). Read-only.|
|[Count](hpagebreaks-count-property-excel.md)|Returns a  **Long** value that represents the number of objects in the collection.|
|[Creator](hpagebreaks-creator-property-excel.md)|Returns a 32-bit integer that indicates the application in which this object was created. Read-only  **Long** .|
|[Item](hpagebreaks-item-property-excel.md)|Returns a single object from a collection.|
|[Parent](hpagebreaks-parent-property-excel.md)|Returns the parent object for the specified object. Read-only.|
