# ChartFormat Members (Word)
Provides access to the OfficeArt formatting for chart elements. 

Provides access to the OfficeArt formatting for chart elements. 


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Adjustments](chartformat-adjustments-property-word.md)|Returns an [Adjustments](adjustments-object-word.md) object that contains a collection of adjustment values for the specified **Chart Format** object.|
|[Application](chartformat-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[AutoShapeType](chartformat-autoshapetype-property-word.md)|Returns or sets the auto shape type of the specified object. Read-write [MSOAutoShapeType](msoautoshapetype-enumeration-office.md).|
|[Creator](chartformat-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Fill](chartformat-fill-property-word.md)|Returns a  **[FillFormat](fillformat-object-word.md)** object for the parent chart element that contains fill formatting properties for the chart element. Read-only.|
|[Glow](chartformat-glow-property-word.md)|Returns the glow formatting properties for the chart element. Read-only  **[GlowFormat](glowformat-object-word.md)** .|
|[Line](chartformat-line-property-word.md)|Returns the line formatting properties for the specified chart element. Read-only  **[LineFormat](lineformat-object-word.md)** .|
|[Parent](chartformat-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[PictureFormat](chartformat-pictureformat-property-word.md)|Returns a  **[PictureFormat](pictureformat-object-word.md)** object for a specified chart that contains pictures. Read-only.|
|[Shadow](chartformat-shadow-property-word.md)|Returns shadow formatting properties for the chart element. Read-only  **[ShadowFormat](shadowformat-object-word.md)** .|
|[SoftEdge](chartformat-softedge-property-word.md)|Returns the soft edge formatting for a shape. Read-only  **[SoftEdgeFormat](softedgeformat-object-word.md)** .|
|[TextFrame2](chartformat-textframe2-property-word.md)|Returns the text formatting for the specified chart element. Read-only  **TextFrame2** .|
|[ThreeD](chartformat-threed-property-word.md)|Returns the 3-D-effect formatting properties for the specified chart. Read-only  **[ThreeDFormat](threedformat-object-word.md)** .|
