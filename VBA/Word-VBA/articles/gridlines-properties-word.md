# Gridlines Properties (Word)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](gridlines-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Border](gridlines-border-property-word.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-word.md)** .|
|[Creator](gridlines-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Format](gridlines-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[Name](gridlines-name-property-word.md)|Returns the name of the object. Read-only  **String** .|
|[Parent](gridlines-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
