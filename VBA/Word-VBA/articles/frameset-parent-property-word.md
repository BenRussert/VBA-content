# Frameset.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Frameset** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Frameset](frameset-object-word.md)** object.
