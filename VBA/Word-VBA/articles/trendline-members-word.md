# Trendline Members (Word)
Represents a trendline in a chart.

Represents a trendline in a chart.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[ClearFormats](trendline-clearformats-method-word.md)|Clears the formatting of the object.|
|[Delete](trendline-delete-method-word.md)|Deletes the object.|
|[Select](trendline-select-method-word.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](trendline-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Backward2](trendline-backward2-property-word.md)|Returns or sets the number of periods (or units on a scatter chart) that the trendline extends backward. Read/write  **Double** .|
|[Border](trendline-border-property-word.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-word.md)** .|
|[Creator](trendline-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[DataLabel](trendline-datalabel-property-word.md)|Returns the data label that is associated with the trendline. Read-only  **[DataLabel](datalabel-object-word.md)** .|
|[DisplayEquation](trendline-displayequation-property-word.md)| **True** if the equation for the trendline is displayed on the chart (in the same data label as the R-squared value). Read/write **Boolean** .|
|[DisplayRSquared](trendline-displayrsquared-property-word.md)| **True** if the R-squared value of the trendline is displayed on the chart (in the same data label as the equation). Read/write **Boolean** .|
|[Format](trendline-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[Forward2](trendline-forward2-property-word.md)|Returns or sets the number of periods (or units on a scatter chart) that the trendline extends forward. Read/write  **Double** .|
|[Index](trendline-index-property-word.md)|Returns the index number of the object within the collection of similar objects. Read-only  **Long** .|
|[Intercept](trendline-intercept-property-word.md)|Returns or sets the point where the trendline crosses the value axis. Read/write  **Double** .|
|[InterceptIsAuto](trendline-interceptisauto-property-word.md)| **True** if the point where the trendline crosses the value axis is automatically determined by the regression. Read/write **Boolean** .|
|[Name](trendline-name-property-word.md)|Returns or sets name of the object. Read/write  **String** .|
|[NameIsAuto](trendline-nameisauto-property-word.md)| **True** if Microsoft Word automatically determines the name of the trendline. Read/write **Boolean** .|
|[Order](trendline-order-property-word.md)|Returns or sets the trendline order (an integer greater than 1) when the trendline type is  **xlPolynomial** . Read/write **Long** .|
|[Parent](trendline-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[Period](trendline-period-property-word.md)|Returns or sets the period for the moving-average trendline. Read/write  **Long** .|
|[Type](trendline-type-property-word.md)|Returns or sets the trendline type. Read/write  **[XlTrendlineType](xltrendlinetype-enumeration-word.md)** .|
