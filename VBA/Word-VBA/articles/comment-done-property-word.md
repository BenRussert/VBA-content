# Comment.Done Property (Word)

Returns or sets a  **Boolean** whose value is **true** if the specified comment has been marked closed. Read-write.


## Syntax

 _expression_ . **Done**

 _expression_ A variable that represents a **Comment** object.


## Property value

 **BOOL**
