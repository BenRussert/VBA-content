# LegendEntries Members (Word)
A collection of all the  **[LegendEntry](legendentry-object-word.md)** objects in the specified chart legend.

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Item](legendentries-item-method-word.md)|Returns a single object from a collection.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](legendentries-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Count](legendentries-count-property-word.md)|Returns the number of objects in the collection. Read-only  **Long** .|
|[Creator](legendentries-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Parent](legendentries-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
