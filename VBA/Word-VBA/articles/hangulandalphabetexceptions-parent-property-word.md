# HangulAndAlphabetExceptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **HangulAndAlphabetExceptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[HangulAndAlphabetExceptions](hangulandalphabetexceptions-object-word.md)** collection.
