# Walls Members (Word)
Represents the walls of a 3-D chart. 

Represents the walls of a 3-D chart. 


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[ClearFormats](walls-clearformats-method-word.md)|Clears the formatting of the object.|
|[Paste](walls-paste-method-word.md)|Pastes a picture from the Clipboard on the walls of the specified chart. |
|[Select](walls-select-method-word.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](walls-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Creator](walls-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Format](walls-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[Name](walls-name-property-word.md)|Returns the name of the object. Read-only  **String** .|
|[Parent](walls-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[PictureType](walls-picturetype-property-word.md)|Returns or sets a value that specifies how pictures are displayed on the walls and faces of a 3-D chart. Read/write  **Variant** .|
|[PictureUnit](walls-pictureunit-property-word.md)|Returns or sets the unit for each picture on the chart if the  **[PictureType](series-picturetype-property-excel.md)** property is set to **xlStackScale** ; otherwise, this property is ignored. Read/write **Long** .|
|[Thickness](walls-thickness-property-word.md)|Returns or sets the thickness of the wall. Read/write  **Long** .|
