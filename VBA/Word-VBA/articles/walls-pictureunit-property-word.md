# Walls.PictureUnit Property (Word)

Returns or sets the unit for each picture on the chart if the  **[PictureType](http://msdn.microsoft.com/library/series-picturetype-property-excel%28Office.15%29.aspx)** property is set to **xlStackScale** ; otherwise, this property is ignored. Read/write **Long** .


## Syntax

 _expression_ . **PictureUnit**

 _expression_ A variable that represents a **[Walls](walls-object-word.md)** object.
