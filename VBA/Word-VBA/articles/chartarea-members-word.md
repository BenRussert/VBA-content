# ChartArea Members (Word)
Represents the chart area of a chart. 

Represents the chart area of a chart. 


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Clear](chartarea-clear-method-word.md)|Clears the entire object.|
|[ClearContents](chartarea-clearcontents-method-word.md)|Clears the data from a chart but leaves the formatting.|
|[ClearFormats](chartarea-clearformats-method-word.md)|Clears the formatting of the object.|
|[Copy](chartarea-copy-method-word.md)|Copies the object to the Clipboard.|
|[Select](chartarea-select-method-word.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chartarea-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Creator](chartarea-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Format](chartarea-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[Height](chartarea-height-property-word.md)|Returns or sets the height, in points, of the object. Read/write  **Double** .|
|[Left](chartarea-left-property-word.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double** .|
|[Name](chartarea-name-property-word.md)|Returns the name of the object. Read-only  **String** .|
|[Parent](chartarea-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[Shadow](chartarea-shadow-property-word.md)|Returns or sets a value that determines whether the object has a shadow. Read/write  **Boolean** .|
|[Top](chartarea-top-property-word.md)|Returns or sets the distance, in points, from the top edge of the object to the top of row 1 (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double** .|
|[Width](chartarea-width-property-word.md)|Returns or sets the width, in points, of the object. Read/write  **Double** .|
