# TextColumns.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TextColumns** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TextColumns](textcolumns-object-word.md)** collection.
