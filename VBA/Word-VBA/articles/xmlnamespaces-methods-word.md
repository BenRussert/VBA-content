# XMLNamespaces Methods (Word)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Add](xmlnamespaces-add-method-word.md)| Returns an **XMLNamespace** object that represents a schema that is added to the Schema Library and made available to users in Microsoft Word.|
|[InstallManifest](xmlnamespaces-installmanifest-method-word.md)|Installs the specified XML expansion pack on the user's computer, making an XML smart document solution available to one or more users.|
|[Item](xmlnamespaces-item-method-word.md)|Returns an individual  **XMLNamespace** object in a collection.|
