# ShapeRange.Count Property (Word)

Returns a  **Long** that represents the number of shapes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[ShapeRange](shaperange-object-word.md)** object.
