# DisplayUnitLabel Members (Word)
Represents a unit label on an axis in the specified chart.

Represents a unit label on an axis in the specified chart.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](displayunitlabel-delete-method-word.md)|Deletes the object.|
|[Select](displayunitlabel-select-method-word.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](displayunitlabel-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Caption](displayunitlabel-caption-property-word.md)|Returns or sets the display unit label text. Read/write  **String** .|
|[Characters](displayunitlabel-characters-property-word.md)|Returns a  **[ChartCharacters](chartcharacters-object-word.md)** object that represents a range of characters within the object text. You can use the **ChartCharacters** object to format characters within a text string.|
|[Creator](displayunitlabel-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Format](displayunitlabel-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[Formula](displayunitlabel-formula-property-word.md)|Gets or sets a  **String** value that represents the formula of the object using A1-style notation, in English. Read/write.|
|[FormulaLocal](displayunitlabel-formulalocal-property-word.md)|Returns or sets the formula for the object, using A1-style references in the language of the user. Read/write  **String** .|
|[FormulaR1C1](displayunitlabel-formular1c1-property-word.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the macro. Read/write  **String** .|
|[FormulaR1C1Local](displayunitlabel-formular1c1local-property-word.md)|Returns or sets the formula for the object, using R1C1-style notation in the language of the user. Read/write  **String** .|
|[Height](displayunitlabel-height-property-word.md)|Returns the height, in points, of the object. Read-only.|
|[HorizontalAlignment](displayunitlabel-horizontalalignment-property-word.md)|Returns or sets the horizontal alignment for the specified object. Read/write  **Variant** .|
|[IncludeInLayout](displayunitlabel-includeinlayout-property-word.md)| **True** if a display unit label will occupy the chart layout space when a chart layout is being determined. The default is **True** . Read/write **Boolean** .|
|[Left](displayunitlabel-left-property-word.md)|Returns or sets the distance, in points, from the left edge of the object to the left edge of the chart area. Read/write  **Double** .|
|[Name](displayunitlabel-name-property-word.md)|Returns the name of the object. Read-only  **String** .|
|[Orientation](displayunitlabel-orientation-property-word.md)|Returns or sets the text orientation. Read/write  **Long** .|
|[Parent](displayunitlabel-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[Position](displayunitlabel-position-property-word.md)|Returns or sets the position of the unit label on an axis in the chart. Read/write  **[XlChartElementPosition](xlchartelementposition-enumeration-word.md)** .|
|[ReadingOrder](displayunitlabel-readingorder-property-word.md)|Returns or sets an  **[XlReadingOrder](xlreadingorder-enumeration-word.md)** constant that represents the reading order for the specified object. Read/write **Long** .|
|[Shadow](displayunitlabel-shadow-property-word.md)|Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean** .|
|[Text](displayunitlabel-text-property-word.md)|Returns or sets the text for the specified object. Read/write  **String** .|
|[Top](displayunitlabel-top-property-word.md)|Returns or sets the distance, in points, from the top edge of the object to the top of the first row (on a worksheet) or the top of the chart area (on a chart). Read/write  **Double** .|
|[VerticalAlignment](displayunitlabel-verticalalignment-property-word.md)|Returns or sets the vertical alignment of the specified object. Read/write  **Variant** .|
|[Width](displayunitlabel-width-property-word.md)|Returns the width, in points, of the object. Read-only.|
