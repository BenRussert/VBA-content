# ChartFont Properties (Word)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chartfont-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Background](chartfont-background-property-word.md)|Returns or sets the type of background for text used in charts. Read/write  **Variant** that is set to one of the constants of **[XlBackground](xlbackground-enumeration-word.md)** .|
|[Bold](chartfont-bold-property-word.md)| **True** if the font is bold. Read/write **Variant** .|
|[Color](chartfont-color-property-word.md)|Returns or sets the primary color of the object. Read/write  **Variant** .|
|[ColorIndex](chartfont-colorindex-property-word.md)|Returns or sets the color of the font. Read/write  **Variant** .|
|[Creator](chartfont-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[FontStyle](chartfont-fontstyle-property-word.md)|Returns or sets the font style. Read/write  **String** .|
|[Italic](chartfont-italic-property-word.md)| **True** if the font style is italic. Read/write **Boolean** .|
|[Name](chartfont-name-property-word.md)|Returns or sets the name of the object. Read/write  **String** .|
|[Parent](chartfont-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[Size](chartfont-size-property-word.md)|Returns or sets the size of the font. Read/write  **Variant** .|
|[StrikeThrough](chartfont-strikethrough-property-word.md)| **True** if the font is struck through with a horizontal line. Read/write **Boolean** .|
|[Subscript](chartfont-subscript-property-word.md)| **True** if the font is formatted as subscript. The default is **False** . Read/write **Variant** .|
|[Superscript](chartfont-superscript-property-word.md)| **True** if the font is formatted as superscript. The default is **False** . Read/write **Variant** .|
|[Underline](chartfont-underline-property-word.md)|Returns or sets the type of underline applied to the font. Can be one of the  **[XlUnderlineStyle](xlunderlinestyle-enumeration-word.md)** constants. Read/write **Variant** .|
