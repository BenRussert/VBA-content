# ChartBorder Members (Word)
Represents the border of an object.

Represents the border of an object.


## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](chartborder-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Color](chartborder-color-property-word.md)|Returns or sets the primary color of the object. Read/write  **Variant** .|
|[ColorIndex](chartborder-colorindex-property-word.md)|Returns or sets the color of the border. Read/write  **Variant** .|
|[Creator](chartborder-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[LineStyle](chartborder-linestyle-property-word.md)|Returns or sets the line style for the border. Read/write  **[XlLineStyle](xllinestyle-enumeration-word.md)** , **xlGray25** , **xlGray50** , **xlGray75** , or **xlAutomatic** .|
|[Parent](chartborder-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[Weight](chartborder-weight-property-word.md)|Returns or sets the weight of the border. Read/write  **[XlBorderWeight](xlborderweight-enumeration-word.md)** .|
