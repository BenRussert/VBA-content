# UpBars Members (Word)
Represents the up bars in a chart group.

Represents the up bars in a chart group.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](upbars-delete-method-word.md)|Deletes the object.|
|[Select](upbars-select-method-word.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](upbars-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Border](upbars-border-property-word.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-word.md)** .|
|[Creator](upbars-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Fill](upbars-fill-property-word.md)|Returns a  **[FillFormat](fillformat-object-word.md)** object for the parent chart element that contains fill formatting properties for the chart element. Read-only.|
|[Format](upbars-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[Interior](upbars-interior-property-word.md)|Returns the interior of the object. Read-only  **[Interior](interior-object-word.md)** .|
|[Name](upbars-name-property-word.md)|Returns the name of the object. Read-only  **String** .|
|[Parent](upbars-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
