# Series Methods (Word)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[ApplyDataLabels](series-applydatalabels-method-word.md)|Applies data labels to a series.|
|[ClearFormats](series-clearformats-method-word.md)|Clears the formatting of the object.|
|[Copy](series-copy-method-word.md)|If the series has a picture fill, copies the picture to the Clipboard.|
|[DataLabels](series-datalabels-method-word.md)|Returns an object that represents either a single data label (a  **[DataLabel](datalabel-object-word.md)** object) or a collection of all the data labels for the series (a **[DataLabels](datalabels-object-word.md)** collection).|
|[Delete](series-delete-method-word.md)|Deletes the object.|
|[ErrorBar](series-errorbar-method-word.md)|Applies error bars to the series.|
|[Paste](series-paste-method-word.md)|Pastes a picture from the Clipboard as the marker on the selected series.|
|[Points](series-points-method-word.md)|Returns a collection of all the points in the series.|
|[Select](series-select-method-word.md)|Selects the object.|
|[Trendlines](series-trendlines-method-word.md)|Returns a collection of all the trendlines for the series.|
