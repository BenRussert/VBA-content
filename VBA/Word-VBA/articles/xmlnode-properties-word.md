# XMLNode Properties (Word)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](xmlnode-application-property-word.md)|Returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application.|
|[Attributes](xmlnode-attributes-property-word.md)|Returns an  **XMLNodes** collection that represents the attributes for the specified element.|
|[BaseName](xmlnode-basename-property-word.md)|Returns a  **String** that represents the name of the element without any prefix.|
|[ChildNodes](xmlnode-childnodes-property-word.md)|Returns an  **XMLNodes** collection that represents the child elements of a specified element.|
|[Creator](xmlnode-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[FirstChild](xmlnode-firstchild-property-word.md)|Returns a  **DiagramNode** object that represents the first child node of a parent node. Read-only.|
|[HasChildNodes](xmlnode-haschildnodes-property-word.md)|Returns a  **Boolean** that represents whether an XML node has child nodes. Read-only.|
|[LastChild](xmlnode-lastchild-property-word.md)|Returns an  **XMLNode** object that represents the last child node of an XML element.|
|[Level](xmlnode-level-property-word.md)|Returns a  **[WdXMLNodeLevel](77480b20-9400-c709-c321-d428d28d6799.md)** constant that represents whether an XML element is part of a paragraph, is a paragraph, or is contained within a table cell or contains a table row. Read-only.|
|[NamespaceURI](xmlnode-namespaceuri-property-word.md)|Returns a  **String** that represents the Uniform Resource Identifier (URI) of the schema namespace for the specified object. Read-only.|
|[NextSibling](xmlnode-nextsibling-property-word.md)|Returns an  **XMLNode** object that represents the next element in the document that is at the same level as the specified element.|
|[NodeType](xmlnode-nodetype-property-word.md)|Returns a  **[WdXMLNodeType](2f8824c0-d08e-56d2-9ec8-6fa478452316.md)** constant that represents the type of node.|
|[NodeValue](xmlnode-nodevalue-property-word.md)| Returns or sets a **String** that represents the value of an XML element. Read/write.|
|[OwnerDocument](xmlnode-ownerdocument-property-word.md)|Returns a  **Document** object that represents the parent document of the specified XML element.|
|[Parent](xmlnode-parent-property-word.md)|Returns an  **Object** that represents the parent object of the specified **XMLNode** object.|
|[ParentNode](xmlnode-parentnode-property-word.md)|Returns an  **XMLNode** object that represents the parent element of the specified element.|
|[PlaceholderText](xmlnode-placeholdertext-property-word.md)|Sets or returns a  **String** that represents the text displayed for an element that contains no text.|
|[PreviousSibling](xmlnode-previoussibling-property-word.md)|Returns an  **XMLNode** object that represents the previous element in the document that is at the same level as the specified element.|
|[Range](xmlnode-range-property-word.md)|Returns a  **[Range](range-object-word.md)** object that represents the portion of a document that is contained in the specified object. Read-only.|
|[Text](xmlnode-text-property-word.md)|Returns or sets the text contained within the XML element. Read/write  **String** .|
|[ValidationErrorText](xmlnode-validationerrortext-property-word.md)|Returns a  **String** that represents the description for a validation error on an **XMLNode** object.|
|[ValidationStatus](xmlnode-validationstatus-property-word.md)| Returns a **WdXMLValidationStatus** constant that represents whether an element or attribute is valid according to the attached schema.|
|[WordOpenXML](xmlnode-wordopenxml-property-word.md)|Returns a  **String** that represents the XML for the node in the Microsoft Word Open XML format. Read-only.|
|[XML](xmlnode-xml-property-word.md)|Returns a  **String** that represents the text, with or without XML markup, that is contained within an XML node. Read-only.|
