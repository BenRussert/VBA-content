# MailMessage Methods (Word)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[CheckName](mailmessage-checkname-method-word.md)|Validates the e-mail addresses that appear in the  **To**,  **Cc**, and  **Bcc** lines in the active e-mail message.|
|[Delete](mailmessage-delete-method-word.md)|Deletes the specified mail message.|
|[DisplayMoveDialog](mailmessage-displaymovedialog-method-word.md)|Displays the  **Move** dialog box, in which the user can specify a new location for the active e-mail message in an available message store.|
|[DisplayProperties](mailmessage-displayproperties-method-word.md)|Displays the  **Properties** dialog box for the active e-mail message.|
|[DisplaySelectNamesDialog](mailmessage-displayselectnamesdialog-method-word.md)|Displays the  **Select Names** dialog box, in which the user can add addresses to the **To**,  **Cc**, and  **Bcc** lines in the active, unsent e-mail message.|
|[Forward](mailmessage-forward-method-word.md)|Opens a new e-mail message with an empty  **To** line for forwarding the active message.|
|[GoToNext](mailmessage-gotonext-method-word.md)|Displays the next mail message if you are using Word as your e-mail editor.|
|[GoToPrevious](mailmessage-gotoprevious-method-word.md)|Displays the previous mail message if you are using Word as your e-mail editor.|
|[Reply](mailmessage-reply-method-word.md)|Opens a new e-mail message — with the sender's address on the  **To** line — for replying to the active message.|
|[ReplyAll](mailmessage-replyall-method-word.md)|Opens a new e-mail message — with the sender's and all other recipients' addresses on the  **To** and **Cc** lines, as appropriate — for replying to the active message.|
|[ToggleHeader](mailmessage-toggleheader-method-word.md)|Toggles the display of the header in the active e-mail message.|
