# ShapeRange.BackgroundStyle Property (Word)

Sets or returns the background style of the shapes in the specified shape range. Read/write [MsoBackgroundStyleIndex](http://msdn.microsoft.com/library/msobackgroundstyleindex-enumeration-office%28Office.15%29.aspx).


## Syntax

 _expression_ . **BackgroundStyle**

 _expression_ A variable that represents a **[ShapeRange](shaperange-object-word.md)** object.
