# XMLNode Methods (Word)

## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Copy](xmlnode-copy-method-word.md)|Copies the specified XML element, excluding XML markup, to the Clipboard.|
|[Cut](xmlnode-cut-method-word.md)|Removes the specified XML element from the document and places it on the Clipboard.|
|[Delete](xmlnode-delete-method-word.md)|Deletes the specified XML element from an XML document.|
|[RemoveChild](xmlnode-removechild-method-word.md)|Removes a child element from the specified element.|
|[SelectNodes](xmlnode-selectnodes-method-word.md)|Returns an  **[XMLNodes](xmlnodes-object-word.md)** collection that represents all the child elements that match the XPath parameter, in the order in which they appear within the specified XML element.|
|[SelectSingleNode](xmlnode-selectsinglenode-method-word.md)|Returns an  **XMLNode** object that represents the first child element that matches the XPath parameter within the specified XML element. .|
|[SetValidationError](xmlnode-setvalidationerror-method-word.md)|Changes the validation error text displayed to a user for a specified node and forces Word to report a node as invalid.|
|[Validate](xmlnode-validate-method-word.md)|Validates an individual XML element against the XML schemas that are attached to a document.|
