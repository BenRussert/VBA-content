# System Properties (Word)

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](system-application-property-word.md)|Returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application.|
|[ComputerType](system-computertype-property-word.md)|You have requested Help for a Visual Basic keyword used only on the Macintosh. For information about the  **ComputerType** property of the **System** object, consult the language reference Help included with Microsoft Office Macintosh Edition. Read-only.|
|[CountryRegion](system-countryregion-property-word.md)|Returns the country/region designation of the system. Read-only  **WdCountry** .|
|[Creator](system-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Cursor](system-cursor-property-word.md)|Returns or sets the state (shape) of the pointer. Can be one of the following  **WdCursorType** constants: **wdCursorIBeam** , **wdCursorNormal** , **wdCursorNorthwestArrow** , or **wdCursorWait** . Read/write **Long** .|
|[FreeDiskSpace](system-freediskspace-property-word.md)|Returns the available disk space for the current drive, in bytes. Use the ChDrive statement to change the current drive. Read-only  **Long** .|
|[HorizontalResolution](system-horizontalresolution-property-word.md)|Returns the horizontal display resolution, in pixels. Read-only  **Long** .|
|[LanguageDesignation](system-languagedesignation-property-word.md)|Returns the designated language of the system software. Read-only  **String** .|
|[MacintoshName](system-macintoshname-property-word.md)|You have requested Help for a Visual Basic keyword used only on the Macintosh. For information about the  **MacintoshName** property for the **System** object, consult the language reference Help included with Microsoft Office Macintosh Edition.|
|[MathCoprocessorInstalled](system-mathcoprocessorinstalled-property-word.md)| **True** if a math coprocessor is installed on the system. Read-only **Boolean** .|
|[OperatingSystem](system-operatingsystem-property-word.md)|Returns the name of the current operating system (for example, "Windows" or "Windows NT"). Read-only  **String** .|
|[Parent](system-parent-property-word.md)|Returns an  **Object** that represents the parent object of the specified **System** object.|
|[PrivateProfileString](system-privateprofilestring-property-word.md)|Returns or sets a string in a settings file or the Microsoft Windows registry. Read/write  **String** .|
|[ProfileString](system-profilestring-property-word.md)|Returns or sets a value for an entry in the Windows registry under the following subkey: `HKEY_CURRENT_USER\Software\Microsoft\Office\version\Word`. Read/write  **String** .|
|[QuickDrawInstalled](system-quickdrawinstalled-property-word.md)|You have requested Help for a Visual Basic keyword used only on the Macintosh. For information about the  **QuickDrawInstalled** property for the **System** object, consult the language reference Help included with Microsoft Office Macintosh Edition.|
|[Version](system-version-property-word.md)|Returns the version number of the operating system. Read-only  **String** .|
|[VerticalResolution](system-verticalresolution-property-word.md)|Returns the vertical screen resolution in pixels. Read-only  **Long** .|
