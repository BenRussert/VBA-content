# DataTable Members (Word)
Represents a chart data table.

Represents a chart data table.


## Methods



|**Name**|**Description**|
|:-----|:-----|
|[Delete](datatable-delete-method-word.md)|Deletes the object.|
|[Select](datatable-select-method-word.md)|Selects the object.|

## Properties



|**Name**|**Description**|
|:-----|:-----|
|[Application](datatable-application-property-word.md)|When used without an object qualifier, returns an  **[Application](application-object-word.md)** object that represents the Microsoft Word application. When used with an object qualifier, returns an **Application** object that represents the creator of the specified object (you can use this property with an Automation object to return the application of that object). Read-only.|
|[Border](datatable-border-property-word.md)|Returns the border of the object. Read-only  **[ChartBorder](chartborder-object-word.md)** .|
|[Creator](datatable-creator-property-word.md)|Returns a 32-bit integer that indicates the application in which the specified object was created. Read-only  **Long** .|
|[Font](datatable-font-property-word.md)|Returns the font of the specified object. Read-only  **[ChartFont](chartfont-object-word.md)** .|
|[Format](datatable-format-property-word.md)|Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](chartformat-object-word.md)** .|
|[HasBorderHorizontal](datatable-hasborderhorizontal-property-word.md)| **True** if the chart data table has horizontal cell borders. Read/write **Boolean** .|
|[HasBorderOutline](datatable-hasborderoutline-property-word.md)| **True** if the chart data table has outline borders. Read/write **Boolean** .|
|[HasBorderVertical](datatable-hasbordervertical-property-word.md)| **True** if the chart data table has vertical cell borders. Read/write **Boolean** .|
|[Parent](datatable-parent-property-word.md)|Returns the parent for the specified object. Read-only  **Object** .|
|[ShowLegendKey](datatable-showlegendkey-property-word.md)| **True** if the data label legend key is visible. Read/write **Boolean** .|
