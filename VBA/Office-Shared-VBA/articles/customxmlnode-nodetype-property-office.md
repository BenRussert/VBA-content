# CustomXMLNode.NodeType Property (Office)

Gets the type of the current node. Read-only.


## Syntax

 _expression_. **NodeType**

 _expression_ An expression that returns a **CustomXMLNode** object.


### Return Value

MsoCustomXMLNodeType
